# 메이크렌즈(MakeLens) Studio Pro 완벽 사용 가이드

## 개요
메이크렌즈(MakeLens)는 AI 기반 유튜브 영상 자동 제작 도구입니다.
대본 작성부터 음성 생성, 이미지 생성, 자막 편집, 최종 영상 출력까지
유튜브 영상 제작의 전 과정을 하나의 프로그램에서 자동으로 처리할 수 있습니다.

쇼츠(Shorts) 영상과 롱폼(Long-form) 영상 모두 제작할 수 있으며,
AI가 벤치마킹 분석, 주제 추천, 대본 생성, 이미지 생성, 음성 합성까지 도와줍니다.

메이크렌즈는 자동차의 역할을 하고, Gemini API와 그록은 연료의 역할을 합니다.
처음에 API 세팅만 잘 해두면 이후에는 편하게 사용하실 수 있습니다.

## 프로그램 전체 구조

메이크렌즈는 크게 3개의 메인 메뉴와 4단계 작업 흐름으로 구성되어 있습니다.

### 메인 메뉴 (좌측 사이드바)
- 작업실: 영상 제작의 핵심 공간 (4단계 작업 흐름)
- 프로젝트: 저장된 프로젝트 관리 (불러오기/삭제)
- 설정: 스타일 프리셋 및 API 키 설정
- 버전 정보: 사이드바 중앙의 버전 번호를 클릭하면 현재 버전 확인 및 최신 버전 업데이트가 가능합니다.
- 딸깍 버튼: 사이드바에 있는 특별한 버튼으로, 모든 작업을 원클릭으로 자동화합니다.

### 4단계 작업 흐름 (작업실 내부)
1. 대본 - 벤치마킹 분석 → 주제 추천 → AI 대본 생성
2. 장면/이미지 - 장면 분석 → AI 이미지 생성 → 동영상 생성(그록 연동)
3. 음성 - AI 음성(TTS) 생성 및 미리듣기
4. 최종 영상 - 편집 준비 → 자막/효과 편집 → 영상 출력

## 다운로드 및 설치

### 다운로드
1. 튜브렌즈 홈페이지의 '프로그램 다운' 메뉴에서 '메이크렌즈'를 다운로드합니다.
2. 다운로드가 안 되는 경우, 본인이 사용하는 백신 프로그램(V3, 알약, 윈도우 디펜더 등)과 바이러스 및 위협방지의 '실시간 보호 기능'을 잠시 끄고 다운로드해 보세요.

### 설치
1. 다운로드한 설치 파일을 더블 클릭하여 설치합니다.
2. 컴퓨터에 1.2GB 이상의 여유 공간이 필요합니다.
3. 기본적으로 바탕 화면에 설치되게 되어 있습니다.
4. 나중에 생성된 이미지나 영상을 빠르게 사용하시려면 바탕 화면에 설치하는 것을 추천드립니다.
5. 설치가 완료되면 바탕 화면에 바로가기 아이콘과 프로그램 폴더가 생성됩니다.

### 설치 폴더 구성
메이크렌즈 폴더를 열면 다음과 같은 구성입니다:
- _internal 폴더: 프로그램 내부 파일 (수정하지 마세요)
- 그록오토 폴더: 메이크렌즈와 연동되는 크롬 확장 프로그램 (동영상 생성용)
- 프로젝트 폴더: 생성된 프로젝트들이 저장되는 폴더
- 효과음 폴더: 효과음 파일을 넣는 폴더 (사용자가 직접 추가 가능)
- 배경음악 폴더: 배경 음악 파일을 넣는 폴더 (사용자가 직접 추가 가능)
- MakeLens.exe: 메이크렌즈 실행 파일
- uninstall.exe: 프로그램 삭제용 파일

## 처음 시작하기: 인증 및 API 설정

### 1단계: 인증키 입력
1. 메이크렌즈를 처음 실행하면 인증키 입력창이 나타납니다.
2. 무료 체험 인증키 또는 유료 인증키를 입력하고 확인을 누릅니다.
3. 인증이 성공적으로 완료되면 확인 메시지가 표시됩니다.

### 2단계: Gemini API 키 설정 (필수)
1. 인증 후 프로그램에 들어가면 "Gemini API 키가 설정되지 않았습니다"라는 안내 메시지가 표시됩니다.
2. 확인을 누르면 자동으로 설정 탭으로 이동합니다.
3. Gemini API 키 입력란에 본인의 '유료 API 키'를 입력합니다.
4. API 설정 저장 버튼을 클릭하면 알림음과 함께 저장이 완료됩니다.

> 중요: 반드시 '결제 활성화'가 된 유료 API 키를 사용해야 합니다. '결제 설정'으로만 되어 있는 무료 등급 API 키로는 이미지 생성이 되지 않습니다.

### Gemini API 키 발급 방법
- API키는 Google AI Studio (https://aistudio.google.com/) 에서 발급받을 수 있습니다.
- 300달러(약 43만원) 무료 크레딧은 구글 클라우드 콘솔(https://cloud.google.com/cloud-console?hl=ko)에서 받을 수 있습니다. (정책 변경 가능)
- API 키 발급 후 결제 활성화까지 완료해야 메이크렌즈에서 사용 가능합니다.
- 자세한 발급 방법은 별도의 안내 영상을 참고해 주세요.
영상 링크 : https://youtu.be/ExzGaUyixqg

### 인증 확인 및 관리
- 설정 탭 우측에서 본인의 인증 상태와 남은 기간을 확인할 수 있습니다.
- 인증키 재설정: 무료 체험 기간 만료 전에 유료 인증키를 구매했다면, 인증키 재설정 버튼을 눌러 새 키를 입력할 수 있습니다.

### API 크레딧 및 사용량 확인
- 남은 크레딧 보기: 클릭하면 300달러 중 얼마나 남았는지 확인할 수 있습니다.
  - 크레딧이 안 보이면 우측 상단에서 해당 구글 계정과 프로젝트가 맞는지 확인하세요.
  - 또는 구글 콘솔의 '결제' 탭에서 확인하실 수 있습니다.
  - "유료 1등급" 또는 "티어 1"이라고 써 있으면 유료 등급입니다.(무료 체험 포함)
  - "무료 등급"이라고 써 있으면 무료 API 키를 사용 중인 것입니다.
  - 기간을 28일로 바꾸면 28일간 사용량을 상세히 확인할 수 있습니다.

## 프로젝트 생성 및 관리

### 새 프로젝트 만들기
1. 화면 상단의 '프로젝트 이름' 입력란에 원하는 프로젝트 이름을 입력합니다.
2. 저장 버튼을 클릭하거나 Enter 키를 눌러 프로젝트를 저장합니다.
3. 프로젝트 이름을 먼저 저장해야 모든 기능을 사용할 수 있습니다.

### 새 프로젝트 초기화 (+ 버튼)
- 프로젝트 이름 옆의 + 버튼을 클릭하면 새 프로젝트를 시작할 수 있습니다.
- "새 프로젝트를 생성하시겠습니까?" 확인 팝업이 뜹니다.

### 프로젝트 저장
- 저장 버튼을 클릭하면 현재 작업 상태가 저장됩니다.
- 프로그램 종료 시에도 자동으로 마지막 상태가 저장됩니다.
- 일부 작업(이미지 생성, 음성 생성 등) 완료 후에는 자동 저장됩니다.

### 프로젝트 폴더 구조
각 프로젝트 폴더 안에는 다음과 같은 하위 폴더가 자동으로 생성됩니다:
- 이미지_및_오디오 폴더
  - 이미지: 생성된 이미지가 자동 저장됩니다. 이미지를 교체하면 여기도 자동으로 변경됩니다.
  - 오디오: TTS 음성 파일이 자동 저장됩니다.
  - 비디오: 그록에서 생성한 동영상이 저장됩니다.
- 최종영상 폴더: 최종 내보내기한 영상이 저장됩니다.
- 캐릭터 폴더: 캐릭터 일관성에 사용된 이미지가 저장됩니다.

### 프로젝트 탭 (사이드바 > 프로젝트)
- 저장된 프로젝트 목록을 확인할 수 있습니다.
- 프로젝트를 클릭하면 해당 프로젝트를 불러옵니다.
- 현재 프로젝트가 있는 경우 저장 확인 팝업이 표시됩니다:
  - 저장 후 전환: 현재 프로젝트를 저장한 후 전환
  - 저장 안함: 저장하지 않고 전환
  - 취소: 전환 취소
- 프로젝트 카드 우측의 삭제 버튼을 클릭하면 삭제 확인 후 프로젝트가 삭제됩니다.

### 프로젝트에 저장되는 정보
- 대본 텍스트 (벤치마킹 대본 포함)
- 벤치마킹 분석 결과 및 추천 주제 목록
- 장면 데이터 (나레이션, 프롬프트)
- 생성된 이미지
- 생성된 음성 파일
- 자막 데이터 및 자막 스타일 설정
- 이미지 효과, 트랜지션, 효과음 설정
- 스타일 프리셋 설정
- 캐릭터 이미지
- 음성 설정 (목소리, 말투, 속도 등)

## 1단계: 대본 (Script)

대본 탭에서는 유튜브 영상의 대본을 작성합니다.
두 가지 모드를 선택할 수 있습니다.

### 모드 선택

#### 벤치대본 모드 (기본)
조회수 높은 기존 대본들을 분석하여, 그 스타일을 벤치마킹한 새로운 대본을 생성합니다.
이 기능은 메이크렌즈의 핵심 기능 중 하나입니다.

#### 일반대본 모드
뉴스 기사, 블로그 포스트 등 원문을 입력하면 유튜브용 대본으로 변환합니다.

### 영상 형식 설정

- 쇼츠: 10초 이상의 세로 영상 (초 단위로 설정 가능)
- 롱폼: 1분~60분 분량의 가로 영상 (1분 단위로 설정 가능) / 퀄리티를 위해 30분 이내로 추천
- 형식에 따라 예상 글자 수가 자동 계산됩니다.

---

### 벤치대본 모드 사용법

#### 1. 벤치마킹 대본 입력
- 좌측 '벤치마킹 대본' 영역에 조회수 높은 대본들을 붙여넣습니다.
- 대본 3개 이상을 넣는 것을 권장합니다.
- 대본과 대본 사이에는 --- (하이픈 3개 이상)을 넣어 구분합니다.
- 대본파일 불러오기 버튼으로 txt 또는 xlsx 파일을 불러올 수도 있습니다.
  - txt 파일: ---로 구분된 대본 텍스트
  - xlsx 파일: '자막' 헤더가 있는 열의 데이터를 자동으로 읽어옵니다.
  - 튜브렌즈에서 자막 수집 후 엑셀 저장한 파일도 그대로 사용 가능합니다.

#### 2. 벤치 분석 및 주제 추천
- 벤치 분석 및 주제 추천 버튼을 클릭합니다.
- AI가 입력한 대본들의 후킹, 말투, 기승전결 등을 분석하고 지침으로 저장합니다.
- 분석이 끝나면 우측에 추천 주제 10개가 표시됩니다.
- 각 주제 옆에 "높음", "중간" 등 바이럴 가능성이 표시됩니다. 이는 AI가 추측하는 바이럴 정도입니다.

#### 3. 주제 방향 설정 (선택 사항)
- 우측 상단의 입력란에 원하는 방향을 입력할 수 있습니다.
- 예: "물건", "음식", "여행지", "IT기기", "회사" 등
- 벤치마킹 채널이 인물에 대해 다뤘더라도, 여기에 "음식"이라고 입력하면 같은 결(스타일)로 음식 관련 주제를 추천해 줍니다.
- 이를 통해 떡상 채널의 결을 파악해서 다른 주제로 파생시켜 나갈 수 있습니다.

#### 4. 추천 주제 활용
- 추천된 주제를 클릭하면 상세 정보 팝업이 표시됩니다:
  - 주제명
  - 메인 소재 (누구/무엇에 대한 내용인지)
  - 벤치 대본과의 유사점
  - 대본 작성 흐름 (어떤 흐름으로 대본을 만들 것인지)
  - 바이럴 가능성
- 대본 생성 버튼을 클릭하면 해당 주제로 AI 대본이 자동 생성됩니다.
- 주제를 우클릭하면 제목이 클립보드에 복사됩니다.
- 주제 10개 재추천 버튼으로 새로운 주제를 받을 수 있습니다.

#### 5. 대본 수정
- 생성된 대본이 마음에 들면 바로 진행해도 됩니다.
- 마음에 안 드는 부분이 있다면 대본 영역에서 바로 수정할 수 있습니다.
- 대본이 너무 짧으면 더 길게 쓰고, 너무 길면 삭제하여 편집합니다.

#### 6. 원본 대본으로 가공 (벤치대본 모드의 추가 기능)
- 벤치대본 모드에서 '원본 대본' 버튼을 클릭하면 빈 입력 공간이 나타납니다.
- 이곳에 가공하고 싶은 원본 대본을 넣으면, 벤치마킹 분석한 결(스타일)에 맞춰 유튜브형 대본으로 변경해 줍니다.
- 예시: 다른 주제의 대본을 넣으면, 벤치마킹 채널의 스타일로 해당 내용을 재구성해 줍니다.

> 팁: 같은 벤치마킹 프로젝트를 계속 운영하면서, 다양한 주제로 매번 영상을 생성하여 한 채널을 운영하시면 됩니다.

### 일반대본 모드 사용법

#### 1. 원문 입력
- '원문 입력' 영역에 뉴스 기사, 블로그 포스트, 또는 원하는 내용을 입력합니다.
- 기사나 블로그 글들을 가져와서 빠르게 요약해서 영상으로 만들 때 유용합니다.

#### 2. AI 대본 생성
- AI 대본 생성 버튼을 클릭합니다.
- AI가 입력한 원문을 기반으로 설정한 시간 길이에 맞게 유튜브 대본을 생성합니다.

### 유튜브용 대본 (공통)

- 생성된 대본은 하단 '유튜브용 대본' 영역에 표시됩니다.
- 이 영역에서 대본을 자유롭게 수정할 수 있습니다.
- 수정 내용은 대본 탭과 음성 탭에 실시간으로 반영(미러링)됩니다.
- 우측 상단에 공백포함/공백제외 글자 수가 표시됩니다.

> 중요: 딸깍 버튼을 포함해서 장면 이미지, 음성, 최종 영상의 모든 기능은 기본적으로 '유튜브용 대본' 섹션 내용이 꼭 있어야만 실행이 가능합니다.

## 2단계: 장면/이미지 (Scenes/Images)

대본을 기반으로 장면을 분석하고, 각 장면에 맞는 AI 이미지를 생성합니다.

### 장면 분석

#### 장면 분석 실행
1. 먼저 스타일과 이미지 수를 설정합니다.
2. 장면 분석 실행 버튼을 클릭합니다.
3. AI가 대본을 토대로 배경, 스타일 등을 분석해서 장면을 분할하고, 각 장면에 맞는 이미지 프롬프트와 동영상 프롬프트를 자동 생성합니다.
4. 분석이 완료되면 장면 카드 목록이 표시됩니다.

> 중요: 스타일과 이미지 수를 먼저 정한 후에 장면 분석 실행을 해야 합니다.

#### 장면 분할 옵션
장면을 나누는 방식을 선택할 수 있습니다:
- 기본 (복합 분할): AI가 자동으로 최적의 장면 수를 결정합니다. 쇼츠 선택시만 선택 가능한 옵션이며, 거의 0.5문장~1문장마다 분할됩니다. 쇼츠는 빠른 변화가 있어야 이탈률이 적기 때문입니다.
- 전체 이미지 수 지정: 전체 이미지 개수를 직접 지정합니다 (예: 5장, 6장, 10장 내외). 대본을 지정한 수만큼의 문장으로 분할해서 이미지를 생성합니다.
- 문장 당 이미지 수 지정: 문장 몇 개마다 이미지 1장을 생성할지 지정합니다 (예: 1문장당, 2문장당).

#### 장면 분석 재실행
이미 결과가 있는 상태에서 장면 분석 실행을 다시 누르면:
- 전체 새로 생성: 분할부터 전부 다시 만듭니다.
- 이미지/동영상 프롬프트 재생성: 분할과 나레이션은 유지하고 프롬프트만 새로 생성합니다. 스타일을 바꿨을 때 유용합니다.

### 장면 카드

각 장면 카드에는 다음 정보가 표시됩니다:

- 나레이션: 해당 장면의 대본 텍스트
  - 나레이션 텍스트 안에서 Enter 키를 누르면 장면이 자동으로 분할됩니다.
- 이미지 프롬프트: AI가 이미지를 생성할 때 사용하는 설명
- 동영상 프롬프트: 그록에서 동영상 생성 시 사용하는 설명
- 이미지 미리보기: 생성된 이미지가 표시됩니다 (클릭하면 크게 볼 수 있음)
- 캐릭터 표시: 장면 번호 옆에 캐릭터 1, 2, 3이 같은 색상으로 표시되어 어떤 장면에 어떤 캐릭터가 들어가는지 바로 확인할 수 있습니다.
- 초록색 테두리: 이미지에 초록색 테두리가 있으면 해당 이미지의 동영상이 존재한다는 뜻입니다. 어디까지 영상이 생성되었는지 쉽게 확인할 수 있습니다.

#### 장면 편집 버튼
각 장면 카드에는 다음 버튼들이 있습니다:
- 합치기 (화살표): 아래 장면과 현재 장면을 합칩니다. 장면 2의 대본이 장면 1로 끌려오고 장면 2가 사라집니다.
- 삭제 (휴지통): 현재 장면을 삭제합니다.
- 프롬프트 재생성 (별표): 해당 장면 하나의 이미지 프롬프트와 동영상 프롬프트를 다시 생성합니다.
- 이미지 재생성 (회전 버튼): 현재 프롬프트로 해당 장면의 이미지를 다시 생성합니다.
- 영상 재생성 (플레이 버튼): 해당 장면 하나의 영상만 그록에서 새로 생성합니다.
- 이미지 불러오기 (마우스 우클릭): 이미지 영역에서 우클릭하면 외부 이미지를 불러와 교체할 수 있습니다.

### 스타일 설정

장면/이미지 탭 상단에서 이미지 생성 스타일을 선택할 수 있습니다.
드롭다운을 클릭하면 스타일 이미지를 바로 확인할 수 있습니다.

#### 스타일 프리셋 (18종)
- 커스텀: 직접 스타일을 입력하거나, 이미지를 업로드하여 AI가 스타일을 분석합니다.
- 졸라맨 실사 합성: 2D 졸라맨 캐릭터 + 실사 배경
- 감성 수채화: 부드러운 수채화풍 일러스트
- K웹툰: 한국 웹툰 스타일
- 실사: 4K 초고해상도 실사 사진
- 3D 애니메이션: 픽사/디즈니 스타일
- 수채화: 부드러운 수채화
- 사이버펑크: 네온 조명, 미래 도시
- 클레이 애니메이션: 점토 스톱모션 스타일
- 미국 코믹북: 마블/DC 코믹스 스타일
- 연필 스케치: 흑백 연필 드로잉
- 팝아트: 앤디 워홀 스타일
- 일본 애니메이션: 신카이 마코토 스타일
- 픽셀아트: 레트로 게임 스타일
- 파스텔 몽환: 몽환적 파스텔 색감
- 종이공예: 페이퍼크래프트 스타일
- 유화 명화: 클래식 유화 스타일
- 인포그래픽: 미니멀 플랫 디자인

> 팁: 설정의 스타일 클릭 시 하단에서 스타일 프롬프트를 확인할 수 있고, 텍스트를 수정해서 사용하는 것도 가능합니다.

### 모델 선택
- 나노바나나 프로: 고품질 이미지 생성 (추천). 이미지당 약 150~190원 정도 소요.
  - 1:1: 정사각형 비율
  - 자동: 쇼츠일 경우 9:16(세로), 롱폼일 경우 16:9(가로)로 자동 설정
  - 캐릭터 반영이 가능합니다.
  - 미국에서 사용량이 많아 과부하 에러(503)가 발생할 수 있습니다. 그럴 때는 나노바나나 일반으로 전환하세요.
- 나노바나나 일반: 빠른 이미지 생성, API 사용량 절약. 이미지당 약 50~70원.
  - 자동: 쇼츠일 경우 9:16(세로), 롱폼일 경우 16:9(가로)로 자동 설정
  - 1:1: 정사각형 비율
  - 캐릭터 반영이 거의 되지 않습니다.

> 300달러 무료 크레딧을 사용하므로 프로로 사용하시는 것을 권장합니다.

### 캐릭터 일관성

캐릭터 버튼 (char_1, char_2, char_3)을 사용하여 일관된 캐릭터 이미지를 유지할 수 있습니다:
- 캐릭터 버튼을 클릭하면 캐릭터 이미지를 업로드할 수 있습니다.
- 업로드한 캐릭터의 얼굴을 인식해서 해당 캐릭터가 등장하는 모든 장면에 일관되게 적용됩니다.
- 캐릭터 버튼을 우클릭하면 해당 캐릭터 이미지가 삭제됩니다.
- 캐릭터 반영은 나노바나나 프로에서만 가능합니다.

> 이미지 생성 시: 캐릭터는 "누구인지"(얼굴), 스타일 프롬프트는 "어떤 분위기/그림체로", 대본은 "무엇을 하고 있는지"를 결정합니다.

### 이미지 생성

#### 이미지/영상 생성 버튼
클릭하면 다음 옵션이 표시됩니다:
- 전체 이미지 생성: 모든 장면의 이미지를 한 번에 생성합니다.
- 전체 이미지 재생성: 기존 이미지를 모두 삭제하고 처음부터 다시 생성합니다.
- 이어서 이미지 생성: 이미 나온 이미지를 제외하고 나머지만 이어서 생성합니다. 생성 도중 중단했을 때 유용합니다.
- 전체 영상 생성: 모든 장면의 이미지를 그록(Grok)과 연동하여 동영상으로 변환합니다.

### 이미지 저장/백업

#### 저장 열기 버튼
- 이미지 저장: 현재 이미지들을 원하는 폴더에 백업합니다. 저장할 위치를 선택하고 폴더 이름을 입력하면 모든 이미지가 복사됩니다.
- 이미지 폴더 열기: 해당 프로젝트의 이미지 폴더를 바로 엽니다.
- 비디오 폴더 열기: 해당 프로젝트의 비디오 폴더를 바로 엽니다.

> 참고: 이미지를 교체하면 프로젝트 폴더 내의 이미지도 자동으로 변경됩니다. 원본을 보존하고 싶으면 미리 이미지 저장으로 백업해 두세요.

## 그록(Grok) 연동 동영상 생성

메이크렌즈는 그록(Grok) 사이트와 연동하여 이미지를 무료로 동영상으로 변환할 수 있습니다.

> 중요: 그록을 연동한 무료 영상 생성 기능은 그록의 정책 변경이나 차단에 의해 언제든지 사라질 수 있습니다. 메이크렌즈의 메인 기능은 이미지를 생성하고 합쳐서 영상을 만드는 것이며, 그록 연동은 추가 기능입니다.

### 그록 연동 설정 방법
사용법 영상 참조 : 링크 https://youtu.be/z5x_SbaaPi0?si=a5EaOvidARjH_Ha2&t=1339

#### 1단계: 크롬 브라우저에서 그록 로그인
1. 반드시 크롬 브라우저를 사용해야 합니다 (웨일, 엣지 등 다른 브라우저 불가).
2. 크롬에서 그록(grok.com)에 접속합니다.
3. 로그인 버튼을 클릭하고 구글 계정으로 로그인합니다.

#### 2단계: 그록 설정 변경
1. 로그인 후 왼쪽 하단의 프로필 버튼을 클릭합니다.
2. 설정을 클릭합니다.
3. 세 번째에 있는 '동작' 버튼을 클릭합니다.
4. 하단에 있는 "자동 비디오 생성 활성화"를 반드시 꺼야 합니다.
   - 이 설정을 끄지 않으면 메이크렌즈에서 동영상 프롬프트를 자동으로 입력할 수 없습니다.

#### 3단계: 크롬 확장 프로그램 설치
1. 크롬 우측 상단의 점 세 개(더보기) 버튼을 클릭합니다.
2. '확장 프로그램' > '확장 프로그램 관리'를 클릭합니다.
3. 우측 상단의 개발자 모드 토글을 켭니다.
4. 왼쪽의 '압축 해제된 확장 프로그램을 로드합니다' 버튼을 클릭합니다.
5. 메이크렌즈 설치 폴더에서 '그록오토' 폴더를 선택합니다 (폴더 자체를 선택, 파일이 아님).
6. '폴더 선택'을 클릭하면 "메이크렌즈 그록 비디오 제너레이터"가 설치됩니다.

#### 4단계: 확장 프로그램 고정
1. 크롬 우측 상단의 퍼즐 모양 아이콘을 클릭합니다.
2. '메이크렌즈 그록'을 찾아 핀 아이콘을 클릭하여 고정합니다.
3. 고정된 아이콘을 클릭하면 사이드바가 열립니다.
4. 메이크렌즈가 켜져 있으면 "연결됨"으로 표시됩니다. 꺼져 있으면 연결이 끊깁니다.

### 동영상 생성 사용법

#### 전체 영상 생성
1. 장면/이미지 탭에서 이미지가 모두 생성된 상태에서 이미지/영상 생성 > 전체 영상 생성을 클릭합니다.
2. 크롬의 사이드패널에서 이미지가 영상으로 생성되는 과정을 확인할 수 있습니다.
3. 상단에는 작업 대기열, 하단에는 완료된 장면이 표시됩니다.
4. 영상 생성이 완료되면 자동으로 업스케일링 후 저장됩니다.
5. 하나의 영상을 생성하는 데 보통 40~50초 정도 소요됩니다.

#### 장면별 영상 재생성
- 장면 카드의 플레이 버튼을 클릭하면 해당 장면 하나의 영상만 새로 생성할 수 있습니다.

#### 영상 존재 확인
- 이미지에 초록색 테두리가 있으면 해당 이미지의 동영상이 비디오 폴더에 존재한다는 뜻입니다.

> 주의: 그록은 하루 영상 생성 한도가 있습니다. 한도가 초과되면 새 구글 아이디로 로그인하거나 다음 날 다시 시도하세요.(또는 슈퍼그록 유료가입)

## 3단계: 음성 (Voice/TTS)

AI를 활용하여 대본의 음성을 자동으로 생성합니다.

### 대본 확인 및 수정 (미러링)
- 상단에 대본이 미러링되어 표시됩니다.
- 여기서 대본을 수정하면 1단계 대본 탭에도 실시간으로 반영됩니다.
- 반대로 대본 탭에서 수정해도 여기에 자동으로 반영됩니다.

> 주의: 이곳에서 수정한 대본이 장면/이미지 탭의 나레이션과 호환되지는 않습니다. 나레이션 대본과 여기 대본이 일치하지 않으면 나중에 최종 영상에서 TTS 음성과 자막 타이밍이 안 맞을 수 있으니 주의하세요.

### 500자 분할 기능
- 대본 옆에 500자 분할 토글이 있습니다.
- 켜면 대본이 500자를 넘어갈 때 음성 생성을 분할해서 만듭니다.
- 구글TTS는 기능상 글자 수가 많을수록 음성이 깨지거나 갑자기 속삭이는 등 퀄리티가 낮아질 수 있기 때문에 500자씩 분할 생성 가능하도록 설계되어 있습니다.
- 분할 생성된 음성들은 알아서 자동으로 합쳐주니 걱정하지 않아도 됩니다.
- 단, 분할 생성 시 목소리가 조금씩 다를 수 있습니다.
- 목소리 퀄리티를 위해 항상 켜두고 사용하는 것을 권장합니다.

### 목소리 설정

#### 남성 목소리 (9종)
- 태성 (신뢰감 있는 목소리)
- 민수 (활기찬 목소리)
- 승우 (부드러운 목소리)
- 지한 (유머러스한 목소리)
- 성진 (따뜻한 목소리)
- 민혁 (차분한 목소리)
- 준호 (신비로운 목소리)
- 동혁 (부드러운 목소리)
- 서준 (단단한 목소리)

#### 여성 목소리 (9종)
- 유나 (차분한 목소리)
- 서연 (단정한 목소리)
- 민지 (명랑한 목소리)
- 은지 (친근한 목소리)
- 소희 (앳된 호기심 있는 목소리)
- 지수 (자연스러운 목소리)
- 소은 (중후한 목소리)
- 지선 (친근한 목소리)
- 하윤 (발랄한 목소리)

### 말투 스타일 (5종)
- 기본: 일반적인 말투
- 명랑하게: 밝고 경쾌한 말투
- 차분하게: 잔잔하고 침착한 말투
- 강하고 단호하게: 힘 있고 확신에 찬 말투
- 뉴스 리포터처럼: 전문적인 뉴스 앵커 말투

### 음성 속도
- 슬라이더로 0.5x ~ 2.0x 속도를 조절할 수 있습니다.
- 기본값: 1.1x (약간 빠르게)
- 구글 TTS에서는 원래 음성 속도 조절이 되지 않지만, 메이크렌즈에서 이를 가능하게 해두었습니다.
- 쇼츠의 경우 1.3~1.4배속을 추천합니다. 너무 루즈하면 이탈률이 높아질 수 있습니다.

### 말투 세부 지시 (디렉팅, 선택 사항)
- AI 음성의 발화 스타일을 더 세밀하게 지시할 수 있습니다.
- 위의 말투 스타일을 선택해도 좋고, 원하는 말투가 있다면 이곳에 직접 작성합니다.
- 예: "억양이 센 부산 사투리", "뉴스 앵커처럼 깔끔하고 단호하게, 3초마다 명확하게 끊어서 읽어줘" 등
- 작성한 내용대로 TTS에 말투 지시가 반영됩니다.

### AI 음성 생성
1. 목소리, 말투 스타일, 속도를 설정합니다.
2. AI 음성 생성 버튼을 클릭합니다.
3. 생성이 완료되면 하단에 오디오 플레이어가 나타납니다.

### 오디오 미리듣기
- 재생/일시정지, 탐색(시크바), 속도 조절(0.5x~2.0x), 볼륨 조절 기능이 있습니다.
- 현재/전체 재생 시간이 표시됩니다.

## 4단계: 최종 영상 (Editor)

생성된 대본, 이미지(또는 동영상), 음성을 합쳐 최종 영상을 만듭니다.

> 중요: 이미지를 수정했거나 대본을 수정해서 음성을 다시 만든 경우, '편집 준비'를 '반드시' 다시 눌러야 변경된 내용이 적용됩니다.

### 편집 준비

#### 편집 준비 버튼
클릭하면 설정 팝업이 표시됩니다:

##### 소스 선택
- 이미지 모드: 이미지를 기반으로 영상을 만듭니다. 트랜지션과 이미지 효과를 적용할 수 있습니다.
- 자동(영상우선) 모드: 그록에서 생성한 동영상과 이미지 조합으로 기반으로 영상을 만듭니다. 모든 장면에 영상이 있으면 영상기반으로, 영상이 없는 장면은 이미지를 조합하여 영상을 만듭니다.


##### 트랜지션
- 장면이 넘어갈 때 툭툭 끊기지 않고 부드러운 효과를 넣어 자연스럽게 전환됩니다.

##### 무음 자동 삭제
- TTS로 만든 음성의 중간중간 무음 구간을 자동으로 삭제해 줍니다.
- 쇼츠처럼 빠른 템포가 필요한 영상에 유용합니다.

##### 영상 효과음 보존
- 그록으로 생성한 영상의 소리가 편집된 영상에 포함시킬지 정할 수 있습니다.
- 비디오 볼륨의 50%로 설정되어 있습니다.

##### 이미지 효과
- 이미지에 모션 효과를 넣어 단순히 찍어낸 영상이 아닌 느낌을 줍니다.
- 유튜브 정책에 좀 더 안정적으로 영상을 만들 수 있습니다.
- 이미지 모드에서 자동 모드로 사용하는 것을 추천합니다.

#### 편집 준비 과정 (자동)
1. 음성 분석 (Whisper): 음성 파일을 위스퍼AI가 자막을 추출합니다.
2. 장면 매칭 (Gemini AI): 추출된 자막과 음성 타이밍 그리고 장면을 매칭합니다.
3. 자막 생성 (AI 분할): 장면별 자막을 자동으로 생성합니다.
(위스퍼AI가 TTS음성을 듣고 자막을 만들기 때문에, 오타가 있을 수 있습니다)

### 이미지 효과 (6종)

영상의 이미지에 적용할 수 있는 모션 효과입니다:

- 자동 모드 (Auto Mode): AI가 최적의 효과를 자동으로 배치합니다. 장면마다 다른 효과가 번갈아 적용됩니다. (추천)
- 줌 인/아웃 : 이미지가 확대/축소되는 효과로, 몰입감을 높여줍니다.
- 화면 이동 : 이미지가 가로 또는 세로로 이동하는 효과입니다.
- 유영 효과 : 이미지가 몽환적으로 부유하듯 움직이는 효과입니다.
- 복합 연출 : 줌과 패닝을 조합한 역동적인 연출입니다.
- 효과 없음 : 이미지를 정지 상태로 표시합니다.

### 비디오 캔버스 (자막 스타일 설정)

편집 준비가 완료되면 화면이 변하며 다양한 자막 스타일을 설정할 수 있습니다:

#### 텍스트 스타일
- B (굵게): 글씨를 굵게 합니다. 폰트에 따라 적용 여부가 다릅니다. 설정되면 청록색으로 표시됩니다.
- I (기울임): 글씨를 기울입니다.
- 폰트 선택: 무료 폰트들로, 마우스 휠로 빠르게 탐색할 수 있습니다.
  - 자주 쓰는 폰트는 별표를 눌러 즐겨찾기 해두면 편리합니다.
- 글씨 크기: 원하는 크기로 조절합니다.
  - 쇼츠 기본값: 90, 롱폼 기본값: 75

#### 색상 설정
- A (글자 색상): 자막 텍스트 색상을 변경합니다.
- 배경: 자막 배경 박스를 만들 수 있습니다. 투명도와 범위(패딩)도 조절 가능합니다.
- 테두리 (외곽선): 글자 테두리의 두께와 색상을 조절합니다.
- 그림자: 글자 그림자의 색상, 번짐, 흐림 등을 조절하여 예쁜 자막을 만들 수 있습니다.

#### 자막 위치
- 슬라이더로 자막의 세로 위치(%)를 조절합니다.
  - 쇼츠 기본값: 45%, 롱폼 기본값: 10%
  - 원하는 곳에 두고 설정하면 내보내기 시 해당 위치로 적용됩니다.

#### 한 줄 글자 수 (줄바꿈)
- 자막 한 줄에 몇 글자까지 놓이게 할지 정하는 설정입니다.
  - 쇼츠 기본값: 10자, 롱폼 기본값: 20자
- 슬라이더를 조절하면 글자가 늘어나거나 줄어듭니다.
- 슬라이더로 조정하면 어느 정도 단어별로 분할되지만, 옆에 있는 AI 분할 (반짝이 버튼)을 누르면 AI가 의미 단위로 자막을 더 자연스럽게 분할해 줍니다.

#### 자막 토글 (켜기/끄기)
- 자막 토글을 끄면 자막이 나오지 않는 영상을 만들 수 있습니다.
- 자막 없이 내보내기 하고 싶을 때 사용합니다.

### 효과음 설정

#### 효과음 선택
- 편집 화면에서 효과음 버튼을 클릭하면 효과음 목록이 표시됩니다.
- 음표 아이콘이 있는 것은 개발자가 넣어둔 기본 효과음입니다.
- 반짝이 표시가 있는 것은 사용자가 직접 넣은 커스텀 효과음입니다.

#### 커스텀 효과음 추가
- 메이크렌즈 폴더 안의 효과음 폴더에 원하는 효과음 파일을 넣으면 자동으로 메이크렌즈에서 연동됩니다.

#### 효과음 볼륨
- 효과음 볼륨을 조절할 수 있습니다 (보통 30% 정도 추천).
- 장면이 전환되는 타이밍에 효과음이 재생됩니다.

### 배경 음악 설정

#### 배경 음악 추가
- 메이크렌즈 폴더 안의 배경음악 폴더에 본인이 사용하려는 배경 음악 파일을 넣습니다.
- 메이크렌즈에서 배경음 버튼을 클릭하면 연동된 음악 목록이 표시됩니다.

#### 배경 음악 설정
- 목록에서 원하는 배경 음악을 선택합니다.
- 미리듣기가 가능합니다.
- 볼륨을 조절합니다 (보통 10% 정도 추천).
- 적용 버튼을 누르면 영상에 배경 음악이 반영됩니다.

### 자막 편집 (텍스트 에디터)

편집 준비가 완료되면 우측에 자막 목록이 표시됩니다:

#### 개별 자막 색상 변경
- 각 문장 우측의 팔레트 버튼을 누르면 해당 문장만 색상을 바꿀 수 있습니다.
- 다른 문장은 원래 색상 유지, 선택한 문장만 다른 색으로 표시됩니다.
- 원래대로 돌리려면 팔레트를 눌러 '기본값 사용'을 클릭하거나 Ctrl+Z로 되돌립니다.

#### 자막 타이밍 수정
- 각 자막의 시작/종료 시간(초)이 표시됩니다.
- 타이밍이 안 맞을 때 여기서 초를 직접 변경하면 됩니다.

#### 자막 분할/합치기
- 문장을 더 쪼개고 싶으면 원하는 곳에 커서를 놓고 Enter를 누르면 문장이 쪼개집니다.
- 다시 합치고 싶으면 문장 끝에 커서를 놓고 Delete 키를 누르거나 Ctrl+Z로 되돌립니다.

#### 자막 클릭 시 이동
- 자막을 클릭하면 해당 시점으로 미리보기가 이동합니다.

### 미리보기

- 좌측 프리뷰 영역에서 이미지와 자막이 실시간으로 미리보기됩니다.
- 화면을 마우스로 클릭해도 재생됩니다.
- Space 키로 재생/일시정지를 전환할 수 있습니다.
- 하단의 재생 버튼(▶)으로도 재생할 수 있습니다.
- 우측에서 배속 조절이 가능합니다 (빠르게/느리게 보기).
- 볼륨 조절도 할 수 있습니다.

### 영상 출력 (내보내기)

#### 내보내기 버튼
내보내기를 클릭하면 두 가지 옵션이 있습니다:

##### 전체 영상 내보내기
- 수정한 자막 세팅값으로 최종 영상을 MP4로 출력합니다.
- 쇼츠: 1080x1920 (세로) / 롱폼: 1920x1080 (가로) 해상도로 출력됩니다.
- 영상은 프로젝트 폴더 내 '완성영상' 폴더에 저장됩니다.
- 내보내기 완료 시 팝업이 표시됩니다.

##### SRT 자막 내보내기
- 다른 편집 프로그램(프리미어 프로, 캡컷 등)에서 자막을 수정하거나 다른 폰트를 사용하고 싶을 때 활용합니다.
- SRT 형식의 자막 파일로 내보냅니다.

#### QR코드 핸드폰 전송 기능
- 내보내기 완료 팝업에 QR코드가 표시됩니다.
- 핸드폰으로 QR을 찍으면 영상을 바로 다운로드할 수 있습니다.
- 중요: 핸드폰이 컴퓨터와 같은 Wi-Fi에 연결되어 있어야 전송이 가능합니다 (데이터로는 불가).
- 쇼츠의 경우 핸드폰에서 직접 업로드하면 첫 부분의 썸네일을 선택하기 편리합니다.
- 내보내기 완료 팝업의 폴더 열기 버튼으로 영상 저장 위치를 바로 확인할 수 있습니다.

#### QR 바로가기 버튼
- 내보내기 버튼 옆의 QR 모양 버튼을 누르면 마지막으로 내보내기한 영상의 QR코드와 폴더 열기가 다시 표시됩니다.
- 이전에 완성했던 영상을 찾을 때 유용합니다.

## 딸깍 버튼 (원클릭 자동화)

사이드바에 있는 딸깍 버튼은 모든 작업을 한 번에 자동화하는 기능입니다.

### 사용 조건
- 유튜브용 대본이 반드시 있어야 합니다.
- 대본이 없으면 딸깍 기능이 동작하지 않습니다.

### 딸깍 설정 화면

딸깍 버튼을 클릭하면 한 번에 모든 설정을 할 수 있는 화면이 나타납니다:

#### Step 1. 대본 및 영상 기초
- 영상 형식: 쇼츠 / 롱폼 선택 (영상 길이는 대본 길이에 맞게 자동 생성되므로 무시해도 됩니다)

#### Step 2. 비주얼 소스 연출
- 생성 모델: 나노바나나 프로 / 나노바나나 일반 / 이매진 4 울트라
- 이미지 스타일: 18종 스타일 프리셋 중 선택
- 이미지 수: 기본(복합 분할), 전체 이미지 수 지정 또는 문장당 이미지 수 지정
- 캐릭터 이미지: 캐릭터 사진 설정 (나노바나나 프로에서만 가능)
- 동영상 생성 토글: 켜면 그록과 연동하여 이미지를 동영상으로 변환합니다. (크롬을 열고 확장앱을 열어두는 것을 권장드립니다.)

#### Step 3. AI 음성 및 나레이션
- 목소리 선택 : 남자/여자 목소리 선택
- 말투 스타일 선택
- 속도 설정 및 500자 분할 토글
- 말투 세부 지시

#### Step 4. 효과 및 자동화 범위
- 트랜지션 / 자막 / 무음삭제 / 영상효과음 사용여부
- 효과음 선택
- 배경 음악 선택
- 이미지 효과 및 영상 효과
- 자동화 범위
1) 1단계: 이미지 생성까지
2) 2단계: 음성 생성까지
3) 3단계: 미리보기까지
4) 4단계: 최종 MP4 내보내기까지

### 딸깍 실행
- 모든 설정을 마친 후 딸깍 시작 버튼을 누르면 자동화가 시작됩니다.
- 로딩 창이 뜨며 전체 경과 시간과 각 탭별 소요 시간을 확인할 수 있습니다.
- 중간에 중단하고 싶으면 중지 버튼을 누릅니다.
- 닫기 버튼을 누르면 로딩 창이 닫히고 각 탭에서 진행 상황을 직접 볼 수 있습니다.
- 다시 딸깍을 누르면 로딩 창을 다시 확인할 수 있습니다.

> 참고: 컴퓨터 사양에 따라 속도 차이가 있습니다. 특히 미리보기 영상과 최종 내보내기 영상은 CPU를 많이 사용하므로 사양이 낮으면 시간이 오래 걸릴 수 있습니다.

> 예시: 나노바나나 프로로 6장 이미지 + 그록 6개 동영상 + TTS + 최종 MP4 내보내기까지 약 10분 소요.

## 설정

### 스타일 탭

#### 스타일 프리셋 선택
- 18가지 스타일 프리셋이 이미지 썸네일과 함께 표시됩니다.
- 클릭하면 하단에서 스타일 프롬프트를 확인할 수 있고 텍스트를 수정하여 사용하는 것도 가능합니다.
- 선택한 스타일은 이미지 생성 시 적용됩니다.

#### 커스텀 스타일
- '커스텀' 카드를 클릭하여 직접 스타일을 정의할 수 있습니다.
- 이미지로 분석: 이미지를 업로드하면 AI가 해당 이미지의 스타일을 분석하여 프롬프트를 자동 생성합니다.

> 예시: 유행하는 해골 콘텐츠를 만들고 싶다면, 해당 스타일의 영상을 캡처하여 커스텀에 넣으면 AI가 스타일을 분석하여 비슷한 이미지를 생성할 수 있습니다.

> 참고: 커스텀 스타일 이미지 분석은 프로젝트가 먼저 생성되어야 사용할 수 있습니다.

### API 설정 탭

#### Gemini API 키
- Google의 Gemini AI를 사용하기 위한 API 키를 입력합니다.
- API 키가 없으면 대본 생성, 이미지 생성, 음성 생성 등의 AI 기능을 사용할 수 없습니다.
- API 키는 Google AI Studio에서 발급받을 수 있습니다.
- 메이크렌즈는 반드시 유료API를 이용하시기 바랍니다.

#### 인증 관리
- 우측에서 인증 상태와 남은 기간을 확인할 수 있습니다.
- 인증키 재설정으로 새 키를 입력할 수 있습니다.

## 외부 콘텐츠 호환 (이미지/음성/영상 교체)

메이크렌즈는 다른 곳에서 만든 이미지, 영상, TTS 음성을 호환할 수 있습니다.

### 이미지 교체
- 프로젝트 폴더 > 이미지_및_오디오 > 이미지 폴더에서 파일명을 동일하게 맞춰 교체합니다.
예시) scene_1.png (scene_01.png X)
- 또는 장면 카드에서 마우스 우클릭으로 외부 이미지를 불러올 수 있습니다.

### 음성 교체
- 다른 TTS 서비스(일레븐랩스, 타입캐스트 등)에서 생성한 음성을 사용하고 싶다면:
- 프로젝트 폴더 > 이미지_및_오디오 > 오디오 폴더에 파일을 넣습니다.
- 파일명을 narration.wav 또는 narration.mp3로 저장하여 추가하거나 교체하면 됩니다.

### 영상 교체
- 그록 이외의 영상 생성 서비스에서 만든 영상을 사용하고 싶다면:
- 프로젝트 폴더 > 이미지_및_오디오 > 비디오 폴더에서 기존 영상 파일명과 동일하게 교체합니다.
예시) scene_01_grok.mp4 (scene_1_grok.mp4 X)

> 중요: 이미지, 음성, 영상을 교체한 후에는 반드시 편집 준비를 다시 실행해야 변경 사항이 미리보기 및 내보내기에 적용됩니다.

## 에러 메시지 안내

### 400대 에러

#### 401 에러
- 원인: API 키가 올바르지 않습니다.
- 해결: 본인의 API 키가 올바르게 입력되었는지 재확인하세요.

#### 403 에러
- 원인: 권한 부족 또는 결제 미등록 상태입니다.
- 해결: 반드시 '결제 활성화'가 된 유료 API 키를 사용해야 합니다. '결제 설정'으로만 되어 있는 무료 등급 API 키로는 이미지 생성이 되지 않습니다.

#### 404 에러
- 거의 발생하지 않습니다.

#### 429 에러
- 원인: 사용량 한도 초과입니다.
- 해결 (이미지 생성 시): 오늘의 API 할당량이 소진되었습니다. 24시간 후에 다시 시도하세요.
- 해결 (그록 영상 생성 시): 오늘의 영상 생성 한도가 초과되었습니다. 새 구글 아이디로 로그인하거나 다음 날 다시 시도하세요. (또는 슈퍼그록 유료가입)

### 500대 에러

#### 500 에러, 503 에러, 타임아웃 에러
- 원인: 구글 서버 과부하 또는 오류입니다.
- 해결: 프로그램의 문제가 아니라 나노바나나 프로를 사용하는 사용자가 많아 구글 서버에 과부하가 일어난 것입니다.
  - 나노바나나 일반 또는 이매진 4 울트라로 전환하면 문제없이 생성 가능합니다.
  - 또는 시간을 두고 나중에 재시도하세요. 세계적으로 나노바나나 프로 사용자가 몰려서 그렇습니다.

### Safety 에러 (콘텐츠 정책 위반)
- 원인: 이미지 또는 영상 생성 시 콘텐츠 정책에 위반되는 내용이 감지되었습니다.
- 해결: 프롬프트를 수정하여 다시 시도하면 문제없이 생성 가능합니다.

> 참고: 위의 에러들은 메이크렌즈 자체의 문제가 아니므로, 개발자에게 별도로 연락할 필요가 없습니다.

## 자주 묻는 질문 (FAQ)

### Q: 프로그램을 처음 실행했는데 아무것도 안 됩니다. 어떻게 해야 하나요?
A: 먼저 다음 세 가지를 확인해주세요:
   1. 인증키를 입력했는지 확인
   2. 프로젝트 이름을 입력하고 저장 버튼을 눌러 프로젝트를 생성했는지 확인
   3. 설정 > API 설정 탭에서 Gemini API 키를 입력했는지 확인
   이 세 가지가 완료되어야 모든 기능을 사용할 수 있습니다.
(주의 : 간혹 '메이크렌즈 인증키'와 '튜브렌즈 인증키'를 혼동하거나 'API키'와 혼동하는 경우가 있습니다. 튜브렌즈와 별도의 프로그램이오니 재확인 바랍니다)

### Q: Gemini API 키는 어디서 받나요?
A: Google AI Studio (https://aistudio.google.com/) 에서 발급받을 수 있습니다. 
- 300달러 무료 크레딧은 구글 클라우드 콘솔(https://cloud.google.com/cloud-console?hl=ko)에서 받을 수 있습니다. (정책 변경 가능).

### Q: 무료 API 키와 유료 API 키의 차이는 뭔가요?
A: 무료 등급(결제 설정)의 API 키로는 이미지 생성이 되지 않습니다. 반드시 '결제 활성화'가 된 유료 API 키를 사용해야 합니다. 

### Q: 남은 API 크레딧은 어떻게 확인하나요?
A: 설정 > API 설정 탭에서 '남은 크레딧 보기' 버튼을 클릭하면 300달러 중 얼마나 남았는지 확인할 수 있습니다. 크레딧이 보이지 않으면 구글 계정과 프로젝트가 맞는지 확인하세요.
구글 콘솔의 '결제' 탭에서도 확인하실 수 있습니다.

### Q: 503 에러가 자주 발생합니다.
A: 나노바나나 프로 사용 시 구글 서버 과부하로 발생하는 문제입니다. 시간을 두고 나중에 시도하거나, 나노바나나 일반으로 전환하면 수월하게 생성할 수 있습니다.

### Q: "대본 생성에 실패했습니다" 오류가 나옵니다.
A: 다음 사항을 확인해보세요:
   - Gemini API 키가 올바르게 입력되어 있는지 확인
   - 인터넷 연결이 정상적인지 확인
   - API 할당량이 초과되지 않았는지 확인 (24시간 후 자동 복구)

### Q: 벤치마킹 대본은 어떻게 구분하나요?
A: 대본과 대본 사이에 --- (하이픈 3개 이상)을 넣어 구분합니다. 수동 입력 시에도 동일합니다.
   예시:
   ```
   첫 번째 대본 내용...
   ---
   두 번째 대본 내용...
   ---
   세 번째 대본 내용...
   ```

### Q: 쇼츠와 롱폼의 차이가 뭔가요?
A:
   - 쇼츠: 유튜브 쇼츠용 세로 영상 (1080x1920), 10초 이상
   - 롱폼: 일반 유튜브 영상용 가로 영상 (1920x1080), 1~60분
   이미지 생성시 프로 자동 모드에서 쇼츠는 9:16, 롱폼은 16:9 비율로 자동 생성됩니다.

### Q: 이미지 스타일은 중간에 바꿀 수 있나요?
A: 네, 설정 탭이나 장면/이미지 탭 상단의 스타일 드롭다운에서 언제든지 변경할 수 있습니다. 단, 변경 후에는 장면 분석 실행에서 '이미지/동영상 프롬프트 재생성'을 하거나 이미지를 다시 생성해야 새 스타일이 적용됩니다.

### Q: 이미지가 마음에 안 들어요. 특정 장면만 다시 만들 수 있나요?
A: 네, 해당 장면 카드에서 이미지 재생성 (회전 버튼)을 클릭하면 현재 프롬프트로 해당 장면의 이미지만 다시 생성됩니다. 프롬프트를 수정한 후 재생성하면 다른 느낌의 이미지를 얻을 수 있습니다.

### Q: 프롬프트를 직접 수정해도 되나요?
A: 네, 각 장면 카드의 '이미지 프롬프트'를 직접 수정할 수 있습니다. 수정 후 이미지 재생성 버튼으로 새 이미지를 생성하세요. 또는 프롬프트 재생성 버튼으로 대본에 맞게 프롬프트를 다시 자동 생성할 수도 있습니다.

### Q: 캐릭터 일관성은 어떻게 유지하나요?
A: 장면/이미지 탭 상단의 캐릭터 버튼(char_1, char_2, char_3)에 캐릭터 이미지를 업로드하세요. 해당 캐릭터가 등장하는 모든 장면에서 일관된 외모로 이미지가 생성됩니다. 단, 나노바나나 프로에서만 가능합니다.

### Q: 음성 생성 후 목소리가 마음에 안 들면 어떻게 하나요?
A: 목소리, 말투 스타일, 속도를 변경한 후 AI 음성 생성 버튼을 다시 클릭하면 새로운 음성이 생성됩니다.

### Q: TTS 음성이 중간에 속삭이거나 깨집니다.
A: 글자 수가 많으면 TTS 품질이 떨어질 수 있습니다. 500자 분할 토글을 켜면 500자씩 나누어 생성한 후 자동으로 합쳐주므로 품질이 향상됩니다.

### Q: 다른 TTS 서비스의 음성을 사용할 수 있나요?
A: 네, 프로젝트 폴더 > 이미지_및_오디오 > 오디오 폴더에 narration.wav 또는 narration.mp3 파일을 넣으면 됩니다. 교체 후 반드시 편집 준비를 다시 실행하세요.

### Q: 자막이 자연스럽게 끊기지 않아요.
A: 4단계 '최종 영상' 탭에서 AI 분할 (반짝이 버튼)을 클릭하면 AI가 의미 단위로 자막을 다시 분할해 줍니다. '한 줄 글자 수' 슬라이더를 조절한 후 재분할하면 더 나은 결과를 얻을 수 있습니다. 또는 '편집준비'를 재실행해보세요.

### Q: 자막 위치를 조절하고 싶어요.
A: 4단계 '최종 영상' 탭에서 '자막 위치' 슬라이더를 조절하면 됩니다. 값이 작을수록 화면 아래쪽, 클수록 화면 위쪽에 자막이 표시됩니다.

### Q: 자막 없이 영상을 만들 수 있나요?
A: 네, 최종 영상 탭에서 자막 토글을 끄면 자막이 없는 영상으로 내보내기됩니다.

### Q: 특정 자막만 색상을 바꿀 수 있나요?
A: 네, 자막 에디터에서 해당 문장 우측의 팔레트 버튼을 클릭하면 그 문장만 다른 색상으로 변경할 수 있습니다.

### Q: 그록 연동이 안 됩니다.
A: 다음 사항을 확인해주세요:
   1. 크롬 브라우저를 사용하고 있는지 확인 (다른 브라우저 불가)
   2. 그록 사이트에 로그인이 되어 있는지 확인
   3. 그록 설정 > 동작 > 자동 비디오 생성 활성화가 꺼져 있는지 확인
   4. 크롬 확장 프로그램에 그록오토가 설치되어 있는지 확인
   5. 메이크렌즈가 켜져 있고 사이드패널에 "연결됨"으로 표시되는지 확인

### Q: 그록 영상 생성이 갑자기 안 됩니다.
A: 그록의 하루 영상 생성 한도가 초과되었을 수 있습니다. 새 구글 아이디로 로그인하거나 다음 날 다시 시도하세요. 또한 그록의 정책 변경으로 기능이 제한될 수도 있습니다.

### Q: 이미지 모드와 자동 모드의 차이가 뭔가요?
A: 편집 준비 시 선택할 수 있습니다.
   - 이미지 모드: 이미지를 기반으로 영상을 만들며, 이미지 효과(줌, 패닝 등)를 적용할 수 있습니다.
   - 자동(영상우선) 모드: 그록에서 생성한 동영상과 이미지 조합으로 기반으로 영상을 만듭니다. 모든 장면에 영상이 있으면 영상기반으로, 영상이 없는 장면은 이미지를 조합하여 영상을 만듭니다.

### Q: 효과음이나 배경 음악을 직접 추가할 수 있나요?
A: 네, 메이크렌즈 설치 폴더의 효과음 폴더에 효과음 파일을, 배경음악 폴더에 배경 음악 파일을 넣으면 자동으로 프로그램에 연동됩니다.

### Q: 편집 준비는 왜 해야 하나요?
A: '편집 준비'는 다음을 자동으로 처리합니다:
   - 생성 된 이미지와 영상을 랜더링
   - 음성 파일을 위스퍼AI가 자막생성
   - 대본과 음성 타이밍 그리고 장면의 매칭 (Gemini AI)
이미지나 음성을 수정한 후에는 반드시 편집 준비를 다시 실행해야 합니다.

### Q: 프로젝트 파일은 어디에 저장되나요?
A: 프로젝트 파일은 메이크렌즈 설치 폴더 내 '프로젝트' 폴더에 저장됩니다. 각 프로젝트별로 하위 폴더가 생성되며, 이미지, 오디오, 비디오, 완성 영상 등이 체계적으로 정리되어 있습니다.

### Q: 장면을 합치거나 분할할 수 있나요?
A: 네, 가능합니다:
   - 분할: 나레이션 텍스트 안에서 원하는 위치에 커서를 놓고 Enter를 누르면 장면이 자동으로 분할됩니다.
   - 합치기: 장면 카드의 합치기 버튼(화살표)을 클릭하면 아래 장면과 합쳐집니다.

### Q: 완성된 영상을 핸드폰으로 보내고 싶어요.
A: 내보내기 완료 후 팝업에 표시되는 QR코드를 핸드폰으로 스캔하면 바로 다운로드됩니다. 단, 컴퓨터와 같은 Wi-Fi에 연결되어 있어야 합니다.

### Q: 딸깍 버튼은 뭔가요?
A: 대본부터 최종 영상까지 모든 과정을 한 번에 자동으로 처리하는 원클릭 자동화 기능입니다. 설정만 해두고 딸깍 시작을 누르면 됩니다.

## 프로그램 업데이트 및 삭제

### 업데이트
- 프로그램 시작 시 자동으로 업데이트를 확인합니다.
- 사이드바 중앙의 버전 번호를 클릭하면 수동으로 최신 버전을 확인하고 업데이트할 수 있습니다.
- 업데이트 시에도 배경음악, 프로젝트, 효과음 데이터는 삭제되지 않으니 걱정하지 않으셔도 됩니다.
- 업데이트 후 프로그램 실행 시 공지 팝업에 업데이트 내역을 확인할 수 있습니다.

### 삭제
- 설치 폴더의 uninstall.exe 파일을 실행하여 프로그램을 삭제합니다.
- 삭제하더라도 배경음악, 프로젝트, 효과음 데이터는 삭제되지 않습니다.

## 문제 해결

### 다운로드가 되지 않는 경우
- 본인이 사용하는 백신 프로그램(V3, 알약, 윈도우 디펜더 등)과 바이러스 및 위협방지의 '실시간 보호 기능'을 잠시 끄고 다운로드해 보세요.

### 모든 기능이 비활성화되어 있는 경우
- 프로젝트 이름을 입력하고 저장 버튼을 클릭하여 프로젝트를 먼저 생성하세요.
- 프로젝트가 생성되어야 모든 입력 필드와 버튼이 활성화됩니다.

### API 관련 오류가 발생하는 경우
- 설정 > API 설정 탭에서 Gemini API 키가 올바르게 입력되어 있는지 확인하세요.
- 결제 활성화된 유료 API 키인지 확인하세요.
- API 할당량이 초과된 경우 24시간 후에 자동으로 복구됩니다.
- 네트워크 연결이 불안정한 경우 다시 시도하세요.

### 이미지 생성이 실패하는 경우
- API 키 확인 (유료 등급인지 확인)
- 인터넷 연결 확인
- 프롬프트에 부적절한 내용(Safety 위반)이 포함되어 있지 않은지 확인
- 503 에러 시: 구글 서버 과부하. 시간을 두고 재시도하거나 나노바나나 일반 또는 이매진 4 울트라로 전환
- 잠시 후 다시 시도 (API 서버 일시적 장애 가능)

### 음성 생성이 실패하는 경우
- 대본이 비어있지 않은지 확인 (1단계 대본 탭에서 대본 작성 필요)
- API 키 확인
- 대본이 너무 긴 경우 500자 분할 토글을 켜고 시도

### MP4 영상 생성이 실패하는 경우
- 자막이 생성되어 있는지 확인 ('편집 준비' 먼저 실행 필요)
- 이미지/영상, 음성, 장면 데이터가 모두 준비되어 있는지 확인
- 디스크 공간이 충분한지 확인

### 프로그램이 느리게 동작하는 경우
- 많은 수의 장면을 동시에 처리하면 느려질 수 있습니다.
- 미리보기 및 내보내기는 CPU를 많이 사용하므로 컴퓨터 사양에 따라 시간이 달라집니다.
- 이미지와 비디오의 양이 많을수록 오래걸립니다.
- 불필요한 백그라운드 프로그램을 종료하세요.
- 컴퓨터를 재시작하여 메모리를 정리하세요.

## 작업 흐름 요약 (Quick Reference)

### 빠른 시작 가이드

1. 인증키 입력: 프로그램 첫 실행 시 인증키 입력
2. API 키 설정: 설정 > API 설정 > Gemini API 키 입력
3. 프로젝트 생성: 프로젝트 이름 입력 → 저장
4. 대본 작성: 벤치 대본 입력 → 분석 및 추천 → 주제 선택 → 대본 생성
5. 장면/이미지: 장면 분석 실행 → 이미지 생성 (→ 선택: 그록 영상 생성)
6. 음성 생성: 목소리/스타일/속도 선택 → AI 음성 생성
7. 영상 편집: 편집 준비 → 자막/효과 편집 → 영상 내보내기

### 딸깍 원클릭 가이드
1. 대본 작성 (1단계에서 유튜브 대본 준비)
2. 딸깍 버튼 클릭
3. 설정 (스타일, 모델, 음성, 효과 등)
4. 딸깍 시작 → 자동으로 완성

## 롱폼 대본 생성 방식 안내

10분 이상의 긴 대본은 AI가 한 번에 쓰지 않습니다. 사람이 글을 쓰듯, 구조를 먼저 잡고 파트별로 나눠서 씁니다.

생성 과정:
1. AI가 먼저 전체 대본의 뼈대를 설계합니다 (후킹 → 전개 → 클라이맥스 → 결론)
2. 분량에 따라 자동으로 파트 수가 정해집니다
3. 각 파트를 순서대로 작성하며, 이전에 쓴 내용을 보면서 다음 파트를 이어 씁니다
4. 모든 파트가 완성되면 하나로 합쳐집니다

분량별 파트 수:
- 10분: 5파트
- 20분: 5파트
- 30분: 5파트
- 40분: 7파트
- 50분: 8파트
- 60분: 10파트

파트가 많을수록 생성 시간이 길어지지만, 
최대한 촘촘하고 밀도있는 대본이 만들어질 수 있도록 노력했습니다.

미리보기와 내보내기mp4는
1) 컴퓨터 cpu/gpu 사양에따라
2) 이미지로만 만들었는지, 비디오로 만들었는지에 따라
3) 제작 분량(자막 줄수)에 따라 시간차이가 많이 납니다.
너무 오래 걸린다고 생각이 들면,
만들어진 이미지/동영상, tts를 사용중인 편집프로그램에서 조합 및 랜더링 하는것도 한 방법이니
참고 부탁드립니다.

[롱폼 영상 테스트 v1.4.8 기준]
영상길이 : 18분
자막 줄수 :  총 422줄 (중요)
장면 : 73개

1) 저사양 컴퓨터 
Intel N100 / RAM 16GB
- 편집준비 : 완료 약 20분

- 편집준비 : 완료 약 6~7분
- mp4 고화질 출력 : 약 12분

3) Intel Ultra 7 / RAM 64GB / 3070ti
- 편집준비 : 완료 약 5분
- mp4 고화질 출력 : 8분

미리보기와 내보내기시, 속도 영향

# 튜브렌즈 실시간 검색어 기능 상세 가이드

## 챗봇 질문 방법
- 튜브렌즈에 대해 질문이 있으시면 ? 물음표를 먼저 쓰고 질문해주세요 
- 만약 사용자가 메이크렌즈에 대해 물어보면 '메이크렌즈에 대해 질문이 있으시면 / 슬러시를 먼저쓰고 질문해주세요' 라고 답변.

## 개요
튜브렌즈의 실시간 검색어 기능은 사용자에게 여러 플랫폼(구글트렌드, 줌, 네이트, 네이버)의 인기 검색어를 한 눈에 보여주는 기능입니다. 
이를 통해 최신 트렌드를 파악하고 콘텐츠 아이디어를 얻을 수 있습니다.

## 튜브렌즈 전체 사용법 영상 (37분)
https://youtu.be/_U0ZlWsQC2w

## 주요 기능

### 1. 다중 플랫폼 검색어 통합
- **구글트렌드**: 구글에서 현재 인기 있는 검색어 표시
- **ZUM AI 이슈 트렌드**: 줌에서 제공하는 AI 기반 이슈 트렌드 검색어
- **NATE Biz.인기 검색어**: 네이트 비즈니스 관련 인기 검색어
- **네이버 실시간 검색어**: 네이버의 실시간 인기 검색어

### 2. 사용자 인터페이스
- 깔끔하고 직관적인 카드 형태의 UI
- 각 플랫폼별 고유 색상과 아이콘으로 시각적 구분
- 각 검색어 항목에 순위 표시(1~10위)
- 현재 날짜와 시간 표시로 정보의 최신성 확인 가능

### 3. 상호작용 기능
- 검색어 클릭 시 해당 플랫폼에서 바로 검색 결과 열기
- 새로고침 버튼으로 최신 데이터 업데이트
- 닫기 버튼으로 창 닫기

## 사용 방법

### 실시간 검색어 창 열기
1. 튜브렌즈 메인 화면에서 '실시간 검색어' 버튼 클릭
2. 실시간 검색어 창이 표시됨

### 검색어 확인 및 활용
1. 각 플랫폼 카드에서 인기 검색어 목록 확인
2. 관심 있는 검색어 클릭 시 해당 플랫폼의 검색 결과 페이지가 브라우저에서 열림
3. 이를 통해 콘텐츠 제작 아이디어나 최신 트렌드 파악 가능

### 데이터 새로고침
1. 창 하단의 '새로고침' 버튼 클릭
2. 모든 플랫폼의 검색어 데이터가 최신 정보로 업데이트됨
3. 업데이트된 날짜와 시간이 상단에 표시됨

## 자주 묻는 질문(FAQ)

### Q: 실시간 검색어 데이터는 얼마나 자주 업데이트되나요?
A: 새로고침 버튼을 클릭할 때마다 모든 플랫폼의 최신 데이터를 가져옵니다. .

### Q: 특정 플랫폼의 검색어만 볼 수 있나요?
A: 현재 버전에서는 모든 플랫폼의 검색어가 항상 표시됩니다. 특정 플랫폼만 선택하여 보는 기능은 제공되지 않습니다.

### Q: 검색어를 클릭하면 어떤 브라우저에서 열리나요?
A: 시스템 기본 브라우저에서 열립니다. 별도의 브라우저 설정은 필요하지 않습니다.

### Q: "데이터를 가져올 수 없습니다" 메시지가 표시됩니다. 어떻게 해야 하나요?
A: 다음 사항을 확인해보세요:
   - 인터넷 연결 상태가 정상인지 확인
   - 새로고침 버튼을 다시 클릭해보기
   - 방화벽이나 보안 프로그램이 API 접근을 차단하고 있지 않은지 확인
   - 잠시 후 다시 시도 (일시적인 서버 문제일 수 있음)

### Q: 검색어 데이터가 일부 플랫폼에서만 표시되지 않아요.
A: 각 플랫폼별로 API 접근이 독립적으로 이루어지기 때문에, 
특정 플랫폼만 일시적으로 서비스 장애가 있을 수 있습니다. 
새로고침을 시도하거나, 잠시 후 다시 확인해보세요.

### Q: 검색어 결과를 저장할 수 있나요?
A: 현재 버전에서는 검색어 결과를 직접 저장하는 기능은 제공되지 않습니다. 
필요하시다면 화면 캡처를 이용하시기 바랍니다.

## 문제 해결

### 모든 데이터가 표시되지 않는 경우
- 인터넷 연결 확인
- 새로고침 버튼 클릭
- 튜브렌즈 애플리케이션 재시작

### 특정 플랫폼 데이터만 표시되지 않는 경우
- 해당 플랫폼 API 서버에 일시적인 장애가 있을 수 있음
- 잠시 후 새로고침 시도
- 문제가 지속되면 튜브렌즈 개발자에게 문의

### 검색어 클릭 시 브라우저가 열리지 않는 경우
- 시스템 기본 브라우저 설정 확인
- 브라우저 실행 권한 확인
- 튜브렌즈 애플리케이션 재시작

# 튜브렌즈 메인 기능 상세 가이드
## 개요
튜브렌즈(TubeLens)는 유튜브 영상 분석 도구로, 
동영상 데이터를 다양한 관점에서 분석하고 인사이트를 제공합니다.

`TubeLens`는 영상 정보 검색, 채널 분석, 성과 측정 등 다양한 기능을 제공합니다.

## 주요 기능
### 1. 기본 영상 정보 조회

튜브렌즈를 통해 다음과 같은 기본 영상 정보를 조회할 수 있습니다:

- **영상 제목**: 선택한 영상의 제목을 확인할 수 있습니다.
- **썸네일 URL**: 영상의 썸네일 이미지 URL을 확인할 수 있습니다.
- **영상 길이**: 영상의 재생 시간을 확인할 수 있습니다.
- **게시일**: 영상이 언제 게시되었는지 확인할 수 있습니다.
- **조회수**: 영상의 현재 조회수를 확인할 수 있습니다.
- **좋아요 수**: 영상의 좋아요 수를 확인할 수 있습니다.
- **댓글 수**: 영상에 달린 댓글 수를 확인할 수 있습니다.

### 2. 채널 정보 조회
선택한 영상의 채널에 대한 정보를 확인할 수 있습니다:
- **채널명**: 영상을 게시한 채널의 이름을 확인할 수 있습니다.
- **구독자 수**: 채널의 구독자 수를 확인할 수 있습니다.
- **총 영상 수**: 채널에서 게시한 총 영상 수를 확인할 수 있습니다.

### 3. 분석 데이터
튜브렌즈는 다음과 같은 고급 분석 정보를 제공합니다:
- **기여도(Contribution)**: 해당 영상이 채널 성장에 얼마나 기여했는지를 나타내는 지표입니다.
- **성과(Performance)**: 영상의 전반적인 성과를 평가한 지표입니다.
- **CII(Content Impact Index)**: 컨텐츠 영향력 지수로, 영상의 영향력을 수치화합니다.
- **참여도(Engagement)**: 시청자 참여율을 나타내는 지표입니다.

### 4. 댓글 데이터
영상에 달린 댓글을 분석할 수 있습니다:
- **인기 댓글 보기**: 좋아요 순으로 정렬된 인기 댓글을 확인할 수 있습니다.
- **전체 댓글 조회**: 영상의 모든 댓글을 확인할 수 있습니다.

### 5. 자막 데이터
영상의 자막 정보를 확인할 수 있습니다:
- **자막 조회**: 영상에 포함된 자막을 텍스트로 확인할 수 있습니다.

### 6. 채널 상세 정보
채널에 대한 더 자세한 정보를 확인할 수 있습니다:
- **채널 설명**: 채널 소개 내용을 확인할 수 있습니다.
- **채널 총 조회수**: 채널의 모든 영상 누적 조회수를 확인할 수 있습니다.
- **채널 개설일**: 채널이 언제 개설되었는지 확인할 수 있습니다.
- **평균 조회수**: 영상당 평균 조회수를 확인할 수 있습니다.
- **인기 영상 목록**: 채널의 인기 영상 리스트를 확인할 수 있습니다.
- **채널 배너 및 프로필**: 채널의 배너와 프로필 이미지를 확인할 수 있습니다.

### 7. 보관 기능
- **영상 보관**: 원하는 영상을 저정/보관 할수 있습니다.

## 사용 방법
### 프로그램 시작 및 인증
1. 튜브렌즈를 처음 실행하면 인증키 입력 창이 나타납니다.
2. 제공받은 인증키를 입력하여 프로그램 사용 권한을 얻습니다.
3. 인증이 완료되면 메인 화면이 표시됩니다.
4. 인증키 만료일이 7일 이내인 경우 만료 알림이 표시됩니다.

### 영상 분석하기
1. 검색 기능을 사용하여 영상 수집을 합니다.
2. 영상의 다양한 정보가 화면에 표시됩니다.

### 채널 정보 보기
- 채널 정보 탭을 클릭하여 자세한 채널 데이터를 확인할 수 있습니다.

### 댓글 및 자막 분석
1. 댓글 탭을 클릭하여 영상의 댓글을 분석할 수 있습니다.
2. 자막 탭을 클릭하여 영상의 자막 내용을 확인할 수 있습니다.

## 자주 묻는 질문(FAQ)

### Q: 인증키는 어디서 받을 수 있나요?
A: 튜브렌즈 공식 웹사이트를 통해 인증키를 발급받을 수 있습니다.
https://tubelens.kr/

### Q: 인증키 만료일이 임박했다는 메시지가 나타납니다. 어떻게 해야 하나요?
A: 인증키 갱신이 필요합니다. 공식 웹사이트에서 인증키 구매하여 갱신하세요.
https://tubelens.kr/

### Q: YouTube API 할당량 초과 오류가 발생합니다.
A: Google 계정의 YouTube API 또는 등록하신 Youtube API 할당량이 소진되었습니다. 추가 Youtube API를 등록하거나, 24시간 후에 다시 시도해보세요.

### Q: 일부 영상 정보가 로드되지 않습니다.
A: 네트워크 연결을 확인하세요. 
또한 일부 비공개 영상이나 제한된 영상은 정보를 가져올 수 없을 수 있습니다.

### Q: 분석 데이터(기여도, 성과, CII)는 어떻게 계산되나요?
A: 이 지표들은 조회수, 좋아요 수, 댓글 수, 영상 길이 등 여러 요소를 고려하여 튜브렌즈의 독자적인 알고리즘으로 계산됩니다.

## 문제 해결
### YouTube API 연결 실패
- 네트워크 연결 확인
- API 할당량 초과 여부 확인
- 다른 Youtube API 등록

### 프로그램이 느리게 동작할 경우
- 많은 수의 영상을 동시에 분석하는 것은 피하세요
- 컴퓨터를 재시작하여 메모리를 정리하세요
- 불필요한 백그라운드 프로그램을 종료하세요

### 데이터 로딩 오류
- 프로그램을 재시작하세요
- 최신 버전의 튜브렌즈를 사용하고 있는지 확인하세요

## 기술 참고사항

튜브렌즈는 다음 기술을 활용합니다:
- YouTube Data API v3
- 데이터 분석 알고리즘

각 API의 일일 할당량 제한으로 인해 과도한 사용 시 일부 기능이 제한될 수 있습니다.

# 튜브렌즈 타이틀 메이커 상세 가이드
## 개요
타이틀 메이커는 튜브렌즈의 핵심 기능 중 하나로, 
AI 기술을 활용하여 유튜브 영상의 최적화된 제목을 추천해주는 기능입니다.

사용자가 선택한 영상들을 분석하여 핵심 키워드와 태그를 추출하고, 
이를 바탕으로 클릭율과 노출을 극대화할 수 있는 다양한 유형의 제목을 생성합니다.

## 주요 기능
### 1. 영상 데이터 분석
타이틀 메이커는 선택된 영상들에서 다음과 같은 데이터를 분석합니다:
- **제목 핵심 키워드**: 영상 제목에서 자주 등장하는 중요 키워드 TOP 10을 추출합니다.
- **영상 내 태그 순위**: 분석된 영상들에서 사용된 태그의 빈도를 계산하여 순위를 매깁니다.
- **연관 태그**: 유튜브 검색 API를 활용하여 관련 검색어와 태그를 추천합니다.

### 2. AI 제목 생성
사용자가 입력한 영상 주제와 분석된 데이터를 바탕으로 
다음 10가지 유형의 제목을 생성합니다:

- **일반형 제목**: 직관적이고 명확한 내용 전달, 핵심 정보를 간결하게 제시
- **극단적 반전 유형**: 시청자의 예상을 뒤엎는 충격적 반전 요소를 담은 제목
- **자극적&도발 유형**: 강력한 호기심과 클릭을 유도하는 자극적인 제목
- **궁금증 폭발 유형**: 질문형 제목으로 시청자의 궁금증을 최대한 자극
- **강력추천형 제목**: 강한 추천 메시지를 담은 설득력 있는 제목
- **인생팁형 제목**: 유용한 인생 조언과 팁을 암시하는 제목
- **시간효율형 제목**: 빠른 시간 내 효과를 강조하는 제목
- **검증실험형 제목**: 직접 검증한 실험 결과를 강조하는 제목
- **비교분석형 제목**: 명확한 비교와 차이점을 제시하는 제목
- **꿀팁&정보 제목**: 유용한 정보와 꿀팁을 제공하는 제목

각 제목 유형별로 추천 이유도 함께 제공하여 
사용자가 제목의 효과를 이해할 수 있도록 돕습니다.

### 3. 고급 설정
사용자가 원하는 스타일과 목적에 맞게 
제목 생성을 조정할 수 있는 고급 설정을 제공합니다:

- **톤앤매너**: 제목의 전반적인 어조를 설정 (신뢰감 있는, 전문적인, 친근한, 충격적인, 호기심 자극)
- **타겟 시청자**: 목표 시청자층의 연령대 설정 (10대, 20대, 30대, 40대 이상, 전연령)
- **제목 구조**: 제목의 기본 구조 패턴 설정 (이모티콘+키워드+호기심유발, HOW TO+혜택+키워드 등)
- **금지어 설정**: 제목에 포함하지 않을 단어 지정

## 사용 방법
### 1. 타이틀 메이커 창 열기
튜브렌즈 메인 화면에서 '타이틀 메이커' 기능을 클릭하여 창을 엽니다.

### 2. 데이터 분석 확인
타이틀 메이커가 자동으로 선택된 영상들을 분석하여 다음 정보를 보여줍니다:
- 분석된 영상 수
- 평균 조회수
- 제목 핵심 키워드 TOP 10
- 영상 내 태그 순위
- 연관 태그

### 3. 영상 주제 입력
'영상 주제를 입력해주세요' 필드에 제목을 생성할 영상의 주제를 자세히 입력합니다.
- 주제가 구체적일수록 더 관련성 높은 제목이 생성됩니다.
- 예: "초보자도 쉽게 따라할 수 있는 홈트레이닝 루틴"

### 4. 고급 설정 조정 (선택사항)
'고급 설정' 버튼을 클릭하여 제목 생성 설정을 세부적으로 조정할 수 있습니다:
- 톤앤매너 선택
- 타겟 시청자 연령대 설정
- 제목 구조 패턴 선택
- 금지어 설정

### 5. 제목 생성하기
'제목 생성하기' 버튼을 클릭하면 AI가 10가지 유형의 제목을 생성합니다.
- 생성 중에는 진행 상태가 표시됩니다.
- 생성이 완료되면 결과 창이 나타납니다.

### 6. 결과 활용하기
결과 창에서 다음과 같은 작업을 할 수 있습니다:
- 각 제목 유형별 생성 결과 확인
- 마음에 드는 제목 복사하기 (개별 복사 가능)
- 특정 유형의 제목만 재생성하기
- 모든 제목 한 번에 재생성하기
- 전체 제목 목록 복사하기

### 7. 태그 활용하기
'전체 태그 복사' 버튼을 클릭하여 분석된 모든 태그를 클립보드에 복사할 수 있습니다.
- 중복 제거된 태그 목록이 콤마로 구분되어 복사됩니다.
- 이 태그들을 영상 업로드 시 바로 붙여넣기 해서 사용할 수 있습니다.

## 자주 묻는 질문(FAQ)

### Q: 분석에 사용되는 영상은 어떻게 선택되나요?
A: 기본적으로 튜브렌즈 메인 화면에 표시된 모든 영상이 분석에 사용됩니다. 
특정 영상만 선택하려면 메인 화면에서 해당 영상을 미리 선택해주세요.

### Q: 영상 주제를 입력하지 않으면 어떻게 되나요?
A: 주제를 입력하지 않으면 분석된 영상의 키워드를 바탕으로 주제를 자동 유추합니다. 
하지만 최상의 결과를 위해 구체적인 주제 입력을 권장합니다.

### Q: 제목 생성에 시간이 오래 걸리는 이유는 무엇인가요?
A: AI 모델이 고품질의 제목을 생성하기 위해 복잡한 분석과 처리 과정을 거치기 때문입니다. 
보통 10-15초 정도 소요됩니다.

### Q: "요청 속도 제한" 오류가 발생했습니다. 어떻게 해야 하나요?
A: Gemini API의 요청 속도 제한에 도달했을 때 발생합니다. 
10-15초 후에 다시 시도하거나, 설정에서 다른 API 키를 선택해보세요.

### Q: 생성된 제목이 제 주제와 맞지 않습니다.
A: 더 구체적인 영상 주제를 입력하고, 고급 설정에서 톤과 구조를 조정해보세요. 
필요하다면 '전체 재생성' 버튼을 눌러 다시 시도할 수 있습니다.

### Q: 태그는 어떻게 활용하는 것이 좋나요?
A: '전체 태그 복사' 버튼으로 복사한 태그를 유튜브 영상 업로드 시 태그 필드에 붙여넣으면 검색 노출에 도움이 됩니다.

## 문제 해결
### 제목 생성이 되지 않는 경우
- 인터넷 연결 상태 확인
- Gemini API 키가 올바르게 설정되었는지 확인
- 다른 API 키로 변경 시도
- 프로그램 재시작 후 다시 시도

### "연결 오류" 메시지가 나타나는 경우
- 인터넷 연결 상태 확인
- 방화벽이 API 접속을 차단하고 있지 않은지 확인
- 프록시 설정 확인 (사용 중인 경우)

### 특정 유형의 제목만 재생성되지 않는 경우
- 개별 재생성 버튼 대신 전체 재생성 시도
- 고급 설정에서 금지어 목록 확인 (사용 중인 키워드가 금지어로 등록되어 있을 수 있음)
- 주제를 더 명확하게 재설정

# 튜브렌즈 인증 및 라이센스 관리 가이드
## 개요
튜브렌즈는 안전한 사용 및 서비스 품질 유지를 위해 라이센스 인증 시스템을 사용합니다.
이 문서는 튜브렌즈의 인증 시스템과 라이센스 관리 방법에 대한 상세 정보를 제공합니다.

## 인증 시스템 소개
튜브렌즈의 인증 시스템은 두 가지 주요 기능을 관리합니다:

1. **라이센스 관리**: 프로그램 사용 권한 확인 및 기간 관리
2. **YouTube API 인증**: 유튜브 데이터 액세스를 위한 Google 계정 연동

이 두 인증 시스템은 별개로 작동하며, 각각 튜브렌즈 사용과 유튜브 데이터 접근을 위해 필요합니다.

## 라이센스 인증 프로세스
### 초기 라이센스 등록

1. 튜브렌즈를 처음 실행하면 인증키 입력 창이 나타납니다.
2. 구매 시 제공받은 라이센스 키를 입력합니다.
3. 시스템은 라이센스 키의 유효성을 확인하고, 현재 컴퓨터에 등록합니다.
4. 인증에 성공하면 프로그램이 정상적으로 실행됩니다.

### 라이센스 특징
- **하드웨어 바인딩**: 라이센스는 최초 인증한 컴퓨터에 고유하게 바인딩됩니다.
- **만료일 확인**: 프로그램 실행 시 라이센스 만료일을 확인합니다.
- **자동 인증**: 한 번 인증된 컴퓨터에서는 인증키를 다시 입력할 필요가 없습니다.
- **만료 알림**: 라이센스 만료 7일 전부터 알림이 표시됩니다.

### 만료일 및 상태 확인
- 프로그램 타이틀 바에 만료일과 남은 기간이 표시됩니다.
- 예: "Tube Lens - 만료일: 2025-12-31 (남은 기간: 30일)"

## Google 계정 인증 (YouTube API)
튜브렌즈는 YouTube 데이터에 접근하기 위해 사용자의 Google 계정 인증이 필요합니다.
또는 Youtube API키 등록이 필요합니다.

### Google 인증 특징
- **토큰 관리**: 인증 토큰이 안전하게 저장 및 관리됩니다.
- **자동 갱신**: 인증 토큰이 만료되면 자동으로 갱신됩니다.
- **데이터 접근 범위**: YouTube 데이터 읽기 및 일부 작업에 대한 권한만 요청합니다.
- **로그아웃 옵션**: 필요 시 Google 계정을 로그아웃할 수 있습니다.

# 튜브렌즈 자막 처리 기능 가이드
## 개요
튜브렌즈의 자막 처리 기능은 YouTube 영상의 자막을 자동으로 추출하고 분석할 수 있게 해주는 기능입니다. 이 기능을 통해 영상 내용을 텍스트로 확인하고, 키워드 분석, 내용 요약 등 다양한 분석 작업을 수행할 수 있습니다.

## 주요 기능
### 자막 추출
튜브렌즈는 YouTube 영상에서 **선택한 국가 설정에 따라** 자막을 추출합니다:

1. **국가별 언어 우선순위 자동 적용**: 선택한 국가에 맞는 언어로 자막을 우선적으로 검색합니다.
2. **대체 언어 검색**: 주 언어 자막이 없는 경우 대체 언어(주로 영어)를 검색합니다.
3. **자동 생성 자막 활용**: 수동 자막이 없는 경우 자동 생성된 자막을 검색합니다.

자막이 전혀 없는 경우 "자막 없음/재시도"로 표시됩니다.

### 지원 언어
현재 버전에서는 다음 언어의 자막을 **국가별 우선순위**로 지원합니다:

- 한국어 (한국 지역 선택 시)
- 일본어 (일본 지역 선택 시)
- 영어 (미국, 영국, 호주, 캐나다 지역 선택 시)
- 중국어 (대만 지역 선택 시)
- 독일어 (독일 지역 선택 시)
- 프랑스어 (프랑스 지역 선택 시)
- 스페인어 (스페인 지역 선택 시)
- 포르투갈어 (브라질 지역 선택 시)
- 러시아어 (러시아 지역 선택 시)
- 힌디어 (인도 지역 선택 시)

### 병렬 처리
최신 버전에서는 4개 영상을 동시에 처리하는 병렬 처리 기능을 통해 자막 수집 속도가 크게 향상되었습니다.

### 수집 중단 및 재개
자막 수집 도중 사용자가 필요에 따라 수집을 중단하고 나중에 다시 시작할 수 있습니다.

## 사용 방법

### 단일 영상 자막 확인
1. 튜브렌즈 메인 화면에서 자막을 확인할 영상의 "자막" 컬럼을 클릭합니다.
2. "자막수집" 상태라면 자동으로 자막 수집이 시작됩니다.
3. 자막이 수집되면 해당 내용이 팝업 창에 표시됩니다.
4. 복사 버튼을 통해 자막 내용을 클립보드에 복사할 수 있습니다.

### 여러 영상 자막 일괄 처리
1. 튜브렌즈 메인 화면에서 자막을 수집할 영상들을 N열에서 선택합니다(선택 안하면 전체 처리).
2. 사이드바의 "필터 설정" 영역에서 "자막 수집" 버튼을 클릭합니다.
3. 진행 상황이 하단 상태 표시줄에 표시됩니다.
4. 수집 중에는 "자막 수집" 버튼이 "수집 중단"으로 변경되며, 클릭 시 수집이 중단됩니다.
5. 수집이 완료되면 각 영상의 자막 컬럼에 자막 내용이 표시됩니다.

## 새로운 기능

### 국가별 언어 자동 설정
이제 국가 선택 시 해당 국가에 맞는 언어로 자막이 자동 수집됩니다:
- 한국(KR) 선택 시: 한국어 → 영어 순으로 검색
- 일본(JP) 선택 시: 일본어 → 영어 순으로 검색
- 대만(TW) 선택 시: 중국어(번체) → 중국어(간체) → 영어 순으로 검색
- 기타 국가: 해당 국가 공용어 → 영어 순으로 검색

### 병렬 처리 속도 향상
최대 4개 영상을 동시에 처리하는 멀티스레딩 기능을 통해 자막 수집 속도가 최대 4배 향상되었습니다.

### 자막 컬럼 색상 표시
- 자막 수집 성공: 연한 파란색 배경(#E3F2FD)
- 자막 없음/오류: 연한 빨간색 배경(#FFEBEE)과 빨간색 글씨(#FF5252)

## 자막 분석 활용법
추출된 자막은 다음과 같은 용도로 활용할 수 있습니다:

### 키워드 빈도 분석
자막에서 자주 등장하는 키워드를 추출하여 영상의 주요 주제를 파악할 수 있습니다.

### 내용 요약
자막 텍스트를 기반으로 AI 요약 기능을 활용해 영상 내용을 간략하게 요약할 수 있습니다.

### SEO 최적화
자막에서 추출한 키워드를 활용하여 영상 제목, 설명, 태그 등의 SEO를 최적화할 수 있습니다.

### 경쟁사 분석
경쟁 채널의 영상 자막을 분석하여 콘텐츠 전략과 주요 키워드를 파악할 수 있습니다.

## 자주 묻는 질문(FAQ)

### Q: 모든 YouTube 영상에서 자막을 추출할 수 있나요?
A: 영상 업로더가 자막을 추가했거나, YouTube가 자동 생성 자막을 제공하는 경우에만 자막을 추출할 수 있습니다. 자막이 전혀 없는 영상에서는 "자막 없음/재시도"로 표시됩니다.

### Q: 다양한 언어 자막은 어떻게 지원되나요?
A: 사용자가 선택한 국가에 따라 자동으로 해당 언어 자막을 우선적으로 검색하며, 없는 경우 영어 등 대체 언어 자막을 검색합니다. 현재 10개 이상의 언어 자막을 지원합니다.

### Q: 자막 수집 속도가 느린 경우 어떻게 해야 하나요?
A: 최신 버전에서는 4개 영상을 동시에 처리하는 기능이 추가되어 속도가 크게 향상되었습니다. 그러나 네트워크 상태에 따라 여전히 시간이 소요될 수 있습니다.

### Q: 추출된 자막을 저장할 수 있나요?
A: 네, 추출된 자막은 다음과 같은 방법으로 저장할 수 있습니다:
- 자막 컬럼 클릭 후 팝업 창에서 "텍스트 복사" 버튼 클릭
- "Excel로 저장" 기능을 통해 모든 데이터와 함께 저장
- 마우스 우클릭 시 컨텍스트 메뉴에서 텍스트 복사 옵션 선택

## 문제 해결
### 자막이 추출되지 않는 경우
- 해당 영상에 자막이 있는지 YouTube에서 직접 확인해보세요.
- 인터넷 연결 상태를 확인하세요.
- "자막 없음/재시도" 항목을 다시 클릭하여 재시도해보세요.

### 잘못된 언어의 자막이 추출되는 경우
- 국가 선택 설정을 확인하고 원하는 언어에 해당하는 국가로 변경해보세요.
- 해당 영상에 원하는 언어의 자막이 없을 수 있습니다.

### 자막 수집이 중단된 경우
- "자막 수집" 버튼을 다시 클릭하여 수집을 재개할 수 있습니다.
- 네트워크 연결을 확인하세요.

# 튜브렌즈 API 키 설정 화면 사용 가이드
## 개요
튜브렌즈의 '설정' 화면은 프로그램 인증 및 API 키 관리를 위한 중앙 허브입니다. 
이 화면에서 인증키(라이센스) 확인, YouTube API 키 관리, Gemini AI API 키 관리 등 다양한 설정을 수행할 수 있습니다.

## 설정 화면 접근 방법
튜브렌즈 메인 화면에서 '설정' 버튼을 클릭하면 설정 화면에 접근할 수 있습니다.

## 주요 섹션
설정 화면은 세 가지 주요 섹션으로 구성되어 있습니다:
1. **인증키 관리**
2. **YouTube API 키 관리**
3. **Google AI Studio API 관리**

## 1. 인증키 관리
이 섹션에서는 현재 등록된 튜브렌즈 라이센스 정보를 확인하고 관리할 수 있습니다.

### 표시되는 정보
- **인증키**: 현재 등록된 인증키
- **만료일**: 인증키의 만료 날짜와 남은 기간
  - 30일 이상 남은 경우: 초록색 배경
  - 7-30일 남은 경우: 파란색 배경
  - 7일 이하 남은 경우: 노란색 배경으로 강조 표시

### 가능한 작업
- **인증키 재설정**: 다른 인증키로 변경하려면 '인증키 재설정' 버튼을 클릭하세요.

### 인증키 재설정 방법
1. '인증키 재설정' 버튼 클릭
2. 새 인증키 입력
3. 확인을 클릭하면 유효성 검사 후 인증키가 변경됩니다
4. 성공적으로 변경되면 새 만료일과 남은 기간이 표시됩니다

## 2. YouTube API 키 관리
이 섹션에서는 YouTube 데이터에 접근하기 위한 API 키를 관리할 수 있습니다.

### 기능 설명
- YouTube API 키 추가, 변경, 삭제
- 키 상태 확인(활성/할당량 초과)
- 할당량 초과 시 남은 시간 표시
- Google 계정 직접 사용 옵션

### API 키 추가 방법
1. 입력 필드에 YouTube API 키 입력
2. '추가' 버튼 클릭
3. 키 유효성 검사 후 테이블에 추가됩니다
4. 키 발급이 필요한 경우 '키 발급받기' 버튼을 클릭하여 Google Cloud Console로 이동

### API 키 관리
테이블에는 다음 정보가 표시됩니다:
- **API 키**: 보안을 위해 마지막 5자리만 표시
- **상태**: 활성(초록색) 또는 할당량 초과(빨간색)
- **남은 시간**: 할당량 초과 상태인 경우 리셋까지 남은 시간
- **작업**: '사용' 및 '삭제' 버튼

#### 키 사용하기
- 테이블에서 사용하려는 키 옆의 '사용' 버튼 클릭
- 현재 사용 중인 키는 파란색 배경으로 표시됨

#### 키 삭제하기
- 테이블에서 삭제하려는 키 옆의 '삭제' 버튼 클릭
- 확인 메시지에 동의하면 키가 삭제됨

### Google 계정 사용
테이블 상단에 "Google 계정" 항목이 있으며:
- YouTube API 키 대신 Google 계정을 사용하도록 선택 가능
- '사용' 버튼을 클릭하여 Google 계정으로 전환

## 3. Google AI Studio API 관리
이 섹션에서는 AI 기능(예: 타이틀 메이커)에 사용되는 Gemini AI API 키를 관리합니다.

### 기능 설명
- Gemini API 키 추가, 변경, 삭제
- 키 상태 확인
- 요청 속도 제한 상태 및 리셋 시간 확인

### API 키 추가 방법

2. '추가' 버튼 클릭
3. 키가 테이블에 추가됨
4. 키 발급이 필요한 경우 '키 발급받기' 버튼을 클릭하여 Google AI Studio로 이동

### API 키 관리
테이블에는 다음 정보가 표시됩니다:
- **API 키**: 보안을 위해 마지막 5자리만 표시
- **상태**: 활성(초록색) 또는 할당량 초과(빨간색)
- **남은 시간**: 할당량 초과 상태인 경우 리셋까지 남은 시간

#### 키 사용하기

# 튜브렌즈 API 키 관리 시스템 가이드
## 개요
튜브렌즈는 YouTube 데이터 분석과 AI 기반 기능을 제공하기 위해 두 가지 유형의 API 키를 사용합니다:

1. **YouTube Data API 키** - YouTube 영상 정보, 채널 데이터, 댓글 등을 가져오는 데 사용
2. **Gemini AI API 키** - 제목 추천, 내용 분석 등 AI 기반 기능에 사용

이 가이드는 두 API 키 유형의 관리 방법과 일반적인 문제 해결에 대한 정보를 제공합니다.

## API 키의 중요성
API 키는 튜브렌즈의 핵심 기능을 사용하기 위해 필수적입니다:

- **YouTube Data API 키** 없이는 유튜브 영상 검색, 채널 정보 수집, 댓글 분석 등이 불가능합니다.
- **Gemini AI API 키** 없이는 타이틀 메이커, 콘텐츠 분석, AI 기반 추천 등의 기능을 사용할 수 없습니다.

## API 키 관리 기능
### 1. API 키 추가
YouTube와 Gemini API 키를 시스템에 추가하고 관리할 수 있습니다:

- **YouTube API 키 추가**
- **Gemini API 키 추가**

### 2. 현재 사용 중인 키 설정
여러 API 키 중 현재 사용할 키를 선택할 수 있습니다:
- 한 번에 하나의 키만 활성화 가능
- 키 목록에서 원하는 키를 선택하여 현재 키로 설정

### 3. 할당량 관리
API 키의 사용량과 할당량 상태를 추적합니다:
- **할당량 초과 감지** - 키가 할당량을 초과하면 자동으로 상태 업데이트
- **할당량 리셋 시간 표시** - 초과된 키의 할당량이 언제 리셋되는지 확인 가능

### 4. API 키 제거
더 이상 사용하지 않는 API 키를 시스템에서 제거할 수 있습니다.

## API 키 할당량 이해하기
### YouTube Data API 할당량
YouTube Data API는 일일 할당량 제한이 있습니다:
- 기본 할당량은 **하루 10,000 유닛**
- 작업에 따라 다른 유닛이 소비됨 (예: 검색은 50개당 100유닛, 영상 정보 조회는 1유닛)
- 할당량은 **한국 시간 오후 5시(17:00)**에 매일 리셋됨

### Gemini AI API 할당량
Gemini AI API도 사용량 제한이 있습니다:
- **분당 요청 수 제한** 있고, 분당 최대 토큰 처리량: 1,000,000토큰
- 요청 간격이 너무 짧으면 "속도 제한" 오류가 발생할 수 있음

현재 튜브렌즈에서는
gemini api를 flash와 pro 모델 둘다 사용하기 때문에
명확한 수치를 말하기는 어렵습니다.
대략적으로 gemini api를 하루 50~100회 사용 가능합니다.

###Gemini API를 사용하여 5분 길이의 유튜브 대본을 작성하려면, 
먼저, 5분 대본은 대략 750단어 정도이며, 
토큰으로 환산하면 약 1,000토큰 정도로 추정할 수 있습니다. 
입력 프롬프트까지 합치면 한 번의 요청당 약 1,200토큰이 필요합니다.

### Gemini 2.5 Flash 사용 한도
Gemini 2.0 Flash 모델은 다음과 같은 한도가 있습니다:
- 일일 최대 요청 횟수: 250회
- 분당 최대 요청 횟수: 10회
- 분당 최대 토큰 처리량: 250,000토큰

위 수치는 이론적인 최대값이며, 
실제 사용 상황에서는 다양한 요인에 따라 달라질 수 있습니다.

0. 기본적으로 '키 발급받기' 버튼을 눌러서 해당 사이트에 접속 가능합니다.
1. Google AI Studio(https://aistudio.google.com)에 접속
2. 로그인 후 상단 우측 "Get API key" 클릭

3. YouTube 또는 Gemini API 섹션에서 발급받은 API 키 입력 후 "추가" 버튼 클릭

### 현재 사용할 API 키 설정하기

2. 선택한 키가 현재 사용 중인 키로 표시됨

### API 키 제거하기
1. "API 키 관리" 탭에서 제거할 키 선택
2. "삭제" 버튼 클릭

## 자주 묻는 질문(FAQ)
### Q: API 키를 여러 개 추가하는 이유는 무엇인가요?
A: API 키는 일일 사용량 제한이 있습니다. 
여러 키를 추가하면 하나의 키가 할당량을 초과했을 때 다른 키를 자동으로 사용하여 중단 없이 작업을 계속할 수 있습니다.

### Q: "API 키의 할당량이 초과되었습니다"라는 오류가 발생합니다.
A: YouTube API 키의 일일 할당량(10,000 유닛)을 모두 사용했습니다. 
다른 API 키를 추가하여 사용하거나, 할당량이 리셋되는 한국 시간 오후 5시 이후에 다시 시도해보세요.

### Q: "속도 제한에 도달했습니다"라는 오류가 발생합니다.
A: Gemini API는 짧은 시간 내에 너무 많은 요청을 보내면 속도 제한이 적용됩니다. 
10-15초 정도 기다린 후 다시 시도하거나, 다른 Gemini API 키를 사용해보세요.

### Q: API 키가 보안상 안전한가요?
A: 예, 튜브렌즈는 API 키를 로컬 컴퓨터에만 저장하며 외부로 전송하지 않습니다. 
키는 %LOCALAPPDATA%\TubeLens 폴더에 저장됩니다.

### Q: API 키가 여러 대의 컴퓨터에서 사용 가능한가요?
A: 예, 같은 API 키를 여러 컴퓨터에서 사용할 수 있습니다. 
단, 모든 컴퓨터에서의 사용량이 합산되어 일일 할당량에 적용됩니다.

### Q: API 키 할당량은 어떻게 늘릴 수 있나요?
A: YouTube Data API의 경우, 구글 아이디 1개당 1개의 API키만 사용 가능합니다.
1개의 아이디에서 여러개의 프로젝트를 생성해서 여러개의 API를 생성하는게 가능하나,
구글 정책상, 실제 1개의 프로그램에서는 '최초의 API' 1개만 사용 가능합니다.
여러개의 API를 생성시에는 여러개의 구글 아이디를 사용해야합니다.
속도 제한에 유의하시기 바랍니다.

## 문제 해결
### API 키 추가 실패
#### "유효하지 않은 API 키입니다" 오류
- API 키를 정확히 복사했는지 확인
- 키 앞뒤의 공백 제거
- Google Cloud Console에서 키가 활성화되어 있는지 확인
- 1개의 구글 아이디로 여러개의 API를 생성한것은 아닌지 확인(최초의 API만 사용가능)

#### "API 키의 할당량이 초과되었습니다" 오류
- 이미 할당량이 소진된 키입니다.
- 다른 키를 추가하거나 할당량이 리셋될 때까지 기다려야 합니다.

### 할당량 초과 문제
#### YouTube API 할당량 초과
- 여러 API 키를 추가하여 번갈아 사용
- 한국 시간 오후 5시 이후에 다시 시도

#### Gemini API 속도 제한
- 요청 사이에 10-15초 간격을 두고 시도
- 다른 Gemini API 키 사용
- 짧은 시간 동안 여러 요청을 보내지 않도록 주의

### API 키 상태 초기화
가끔 API 키의 상태가 잘못 표시될 수 있습니다. 이 경우:
1. 튜브렌즈 종료
2. %LOCALAPPDATA%\TubeLens 폴더에 있는 api_keys.json 또는 gemini_api_keys.json 파일 백업
3. 해당 파일 삭제
4. 튜브렌즈 재시작
5. API 키 다시 추가

# 튜브렌즈 유튜브 영상 검색 및 분석 기능 상세 가이드
## 개요
튜브렌즈의 유튜브 영상 검색 및 분석 기능은 유튜브 동영상 데이터를 다양한 관점에서 검색, 수집하고 심층적으로 분석할 수 있는 핵심 기능입니다. 
이 기능을 통해 콘텐츠 제작자는 트렌드를 파악하고, 경쟁 채널을 분석하며, 자신의 콘텐츠 전략을 세울 수 있습니다.

## 주요 기능
### 1. 검색 설정
- **키워드 검색**: 다양한 검색어를 입력하여 관련 영상을 검색할 수 있습니다.
- **정렬 옵션**: 최신순 또는 조회수순으로 검색 결과를 정렬할 수 있습니다.
- **영상 수집 수**: 50개, 100개, 200개, 500개, 1000개, 2000개, 5000개 중 원하는 수만큼 영상을 수집할 수 있습니다.
- **기간 선택**: 모든 기간, 1년 이내, 6개월 이내, 3개월 이내, 1개월 이내, 7일 이내 등 다양한 기간 옵션을 선택하거나 직접 날짜 범위를 지정할 수 있습니다.

### 2. 필터 기능
- **비디오 타입 필터**: 쇼츠(60초 이하) 또는 롱폼 영상만 선택적으로 볼 수 있습니다.
- **CII 필터**: Great!!, Good, Soso 등 콘텐츠 영향력 지수(CII)를 기준으로 필터링할 수 있습니다.
- **조회수 필터**: 1만 이상, 5만 이상, 10만 이상 등 조회수 기준으로 필터링할 수 있습니다.
- **구독자 수 필터**: 1천명 이하, 5천명 이하, 1만명 이하 등 채널의 구독자 수를 기준으로 필터링할 수 있습니다.

### 3. 분석 지표
- **채널 기여도(Contribution)**: 해당 영상이 채널의 전체 조회수에서 차지하는 비중(%)을 나타냅니다.
- **성과도 배율(Performance)**: 구독자 수 대비 조회수의 비율로, 영상이 얼마나 많은 시청자를 끌어들였는지를 보여줍니다.
- **CII(Content Impact Index)**: 채널 기여도(30%)와 성과도 배율(70%)을 종합한 점수로, 영상의 전반적인 영향력을 나타냅니다.
  - Great!!: 70점 이상
  - Good: 50-69점
  - Soso: 30-49점
  - Not bad: 10-29점
  - Bad: 9점 이하
- **참여율(Engagement)**: 조회수 대비 좋아요와 댓글 수의 비율로, 시청자의 참여도를 나타냅니다.

### 4. 부가 기능
- **자막 수집**: 검색된 영상의 자막을 수집하여 분석할 수 있습니다.
- **Excel로 저장**: 검색 결과를 Excel 파일로 내보내고 저장할 수 있습니다.
- **MP4/MP3 다운로드**: 선택한 영상을 MP4(영상) 또는 MP3(오디오) 형식으로, Youtube API와는 상관이 없기때문에, 특별한 횟수 제한없이 다운로드 할 수 있습니다.
- **보관 기능**: 현재 검색 결과를 보관하고 나중에 다시 볼 수 있습니다.
- **AI 추천 아이디어**: AI를 활용하여 콘텐츠 아이디어를 추천받을 수 있습니다.
- **타이틀 메이커**: 효과적인 영상 제목을 만들 수 있는 도구를 제공합니다.
- **분석 및 대본 생성**: 선택한 영상을 분석하고 관련 대본을 생성할 수 있습니다.
- **실시간 검색어**: 다양한 플랫폼의 실시간 검색어 트렌드를 확인할 수 있습니다.

## 사용 방법
### 영상 검색하기
1. 검색 설정 섹션에서 정렬 방식(최신순 또는 조회수순)을 선택합니다.
2. 영상 수집 수(50개, 100개, 200개, 500개, 1000개, 2000개, 5000개 중 선택)를 지정합니다.
3. 기간을 선택합니다(모든 기간, 1년 이내, 직접 선택 등).
4. 검색어를 입력란에 입력합니다.
5. '검색' 버튼을 클릭하여 검색을 시작합니다.
6. 검색 결과가 테이블에 표시됩니다.

### 필터 적용하기
1. 필터 설정 섹션에서 원하는 필터 옵션을 선택합니다:
   - 비디오 타입: 쇼츠 또는 롱폼
   - CII: Great!!, Good, Soso 등
   - 조회수: 1만 이상, 5만 이상 등
   - 구독자 수: 1천명 이하, 5천명 이하 등
2. '필터 적용' 버튼을 클릭하여 필터를 적용합니다.
3. '필터 해제' 버튼을 클릭하여 모든 필터를 초기화할 수 있습니다.

### 자막 수집하기
1. 자막을 수집할 영상을 선택합니다(N열 클릭).
2. 영상을 선택하지 않으면 모든 영상의 자막이 수집됩니다.
3. '자막 수집' 버튼을 클릭합니다.
4. 수집 진행 상황이 프로그레스 바에 표시됩니다.
5. 수집이 완료되면 자막 열에 자막 내용이 표시됩니다.

### 검색 결과 저장 및 관리
1. 영상 선택: 테이블에서 N열을 클릭하여 영상을 선택합니다.
2. 보관하기: '보관' 버튼을 클릭하여 현재 검색 결과를 저장합니다.
3. 보관함 열기: '보관함' 버튼을 클릭하여 이전에 저장한 검색 결과를 확인합니다.
4. Excel로 저장: 'Excel로 저장' 버튼을 클릭하여 검색 결과를 Excel 파일로 내보냅니다.

### 영상 다운로드하기
1. 다운로드할 영상을 선택합니다(N열 클릭).
2. 영상을 선택하지 않으면 현재 표시된 모든 영상이 다운로드됩니다.
3. 'MP4 다운' 또는 'MP3 다운' 버튼을 클릭합니다.
4. 저장 위치를 선택합니다.
5. 다운로드가 시작되고 진행 상황이 표시됩니다.
6. Youtube API와는 상관이 없기때문에, 특별한 횟수 제한없이 다운로드 할 수 있습니다.

### 부가 기능 활용하기
1. AI 추천 아이디어: 'AI 추천 아이디어' 버튼을 클릭하여 콘텐츠 아이디어를 제안받습니다.
2. 타이틀 메이커: '타이틀 메이커' 버튼을 클릭하여 효과적인 제목을 생성합니다.
3. 분석 및 대본 생성: '분석 및 대본 생성' 버튼을 클릭하여 선택한 영상을 분석하고 관련 대본을 생성합니다.
4. 실시간 검색어: '실시간 검색어' 버튼을 클릭하여 다양한 플랫폼의 트렌드를 확인합니다.

## 자주 묻는 질문(FAQ)
### Q: CII(콘텐츠 영향력 지수)는 어떻게 계산되나요?
A: CII는 채널 기여도(30%)와 성과도 배율(70%)을 종합한 점수입니다. 
채널 기여도는 해당 영상이 채널 전체 조회수에서 차지하는 비중이며,
성과도 배율은 구독자 수 대비 조회수의 비율입니다. 
70점 이상은 "Great!!", 50-69점은 "Good", 30-49점은 "Soso", 10-29점은 "Not bad", 9점 이하는 "Bad"로 등급이 매겨집니다.

### Q: 쇼츠와 롱폼 영상은 어떻게 구분되나요?
A: 튜브렌즈에서는 영상 길이가 60초(1분) 이하인 영상을 쇼츠로, 그 이상인 영상을 롱폼으로 분류합니다.

### Q: 자막 수집이 되지 않는 영상이 있어요.
A: 모든 유튜브 영상에 자막이 있는 것은 아닙니다. 
자막이 비활성화되어 있거나, 자막이 없는 영상의 경우 "자막 없음"으로 표시됩니다. 
튜브렌즈는 한국어와 영어 자막을 우선적으로 수집하며, 자동 생성된 자막도 수집합니다.

### Q: YouTube API 할당량 초과 오류가 발생합니다.
A: YouTube API는 하루 10,000 유닛의 할당량 제한이 있습니다. 할당량이 초과되면 다음 방법을 시도해보세요:
   1. 다른 API 키로 전환
   2. 다음 날까지 기다리기 (할당량은 매일 오후 5시에 재설정됩니다)

### Q: Excel로 저장할 때 어떤 정보가 포함되나요?
A: 검색 결과의 모든 정보(썸네일, 채널명, 제목, 게시일, 조회수, 좋아요 수, 댓글 수 등)와 함께 베스트 댓글 10개, 자막 내용, 영상 설명이 포함됩니다. 
또한 썸네일 이미지도 Excel 파일에 삽입됩니다.

### Q: 영상 다운로드 시 "FFmpeg 설치가 필요합니다" 메시지가 나옵니다.
A: 영상 다운로드를 위해서는 FFmpeg가 필요합니다. 
메시지가 나타나면 '예'를 클릭하여 자동으로 설치를 진행하세요. 
설치가 완료되면 다운로드가 자동으로 시작됩니다.

## 문제 해결
### 검색이 되지 않는 경우
- API 키가 올바르게 설정되어 있는지 확인합니다.
- API 할당량이 초과되었을 수 있으므로, 다른 API 키를 추가하거나 다음 날 다시 시도합니다.
- 인터넷 연결 상태를 확인합니다.

### 필터가 작동하지 않는 경우
- '필터 해제' 버튼을 클릭한 후 다시 필터를 설정해보세요.
- 검색 결과가 충분한지 확인하세요. 검색 결과가 적으면 필터링 후 표시될 결과가 없을 수 있습니다.

### 자막 수집이 안되는 경우
- 해당 영상에 자막이 없거나 비활성화되어 있을 수 있습니다.
- 인터넷 연결 상태를 확인합니다.
- 자막 수집 중 오류가 발생하면 다시 시도해보세요.

### 다운로드가 되지 않는 경우
- FFmpeg가 올바르게 설치되어 있는지 확인합니다.
- 저장 경로가 유효하고 쓰기 권한이 있는지 확인합니다.
- 다운로드 중에는 시스템 성능에 따라 시간이 걸릴 수 있으므로 기다려주세요.
- 영상이 비공개이거나 접근이 제한된 경우 다운로드가 불가능할 수 있습니다.

### 프로그램이 느려지는 경우
- 한 번에 너무 많은 영상을 검색하거나 다운로드하지 마세요.
- 컴퓨터를 재시작하여 메모리를 정리하세요.

### API 키 관련 문제
- API 키의 할당량 상태를 확인합니다.
- 새로운 API 키를 추가합니다.

# AI 콘텐츠 분석 리포트 기능 가이드

## 개요
AI 콘텐츠 분석 리포트는 선택한 유튜브 영상들을 AI가 심층 분석하여 성공적인 콘텐츠 제작 전략을 제시하는 고급 분석 기능입니다. 
AI를 활용하여 영상의 제목 패턴, 조회수 비결, 차별화 요소를 자동으로 분석하고, 실전에 바로 적용할 수 있는 구체적인 콘텐츠 전략을 제공합니다.

## 주요 기능

### 1. 5단계 AI 분석 프로세스
튜브렌즈의 AI 분석은 다음과 같은 5단계로 진행됩니다:

**1단계: 선택한 영상 분석**
- 사용자가 선택한 영상 중 조회수가 높은 상위 30개를 분석합니다.
- 제목 패턴, 조회수 비결, 차별화 요소를 AI가 자동으로 파악합니다.

**2단계: 핵심 키워드 추출**
- 1단계 분석 결과를 바탕으로 6개의 핵심 키워드를 추출합니다.
  - 메인 키워드: 가장 핵심적인 단어
  - 서브 키워드: 메인을 보조하는 단어
  - 롱테일 키워드: 구체적인 검색어
  - 추천 키워드 1-3: 관련 추천 키워드

**3단계: 키워드 기반 영상 수집**
- 추출된 메인, 서브, 롱테일 키워드로 유튜브에서 관련 영상을 자동 수집합니다.
- 각 키워드당 50개씩, 총 최대 150개의 영상을 수집합니다.
- 어제(한국시간 기준) 업로드된 인기 영상만 수집합니다.

**4단계: 수집된 영상 재분석**
- 수집한 영상 중 조회수가 높은 상위 30개를 다시 분석합니다.
- 키워드를 어떻게 활용했는지, 어떤 전략이 효과적이었는지 파악합니다.
- 상위 6개 영상은 썸네일과 함께 시각적으로 표시됩니다.

**5단계: 최종 전략 제시**
- 1단계와 4단계의 분석 결과를 종합합니다.
- 콘텐츠 제작 전략, 제목 작성 팁, 차별화 방법을 구체적으로 제시합니다.
- 즉시 실전에 적용 가능한 실용적인 조언을 제공합니다.

### 2. 분석 리포트 구성

**📺 선택한 영상 분석**
- 사용자가 선택한 영상들의 TOP 30 분석 결과
- 제목 패턴, 조회수 비결, 차별화 요소를 간결하게 정리

**🔑 추출된 키워드**
- 6개의 핵심 키워드를 색상으로 구분하여 표시
- 메인/서브/롱테일: 파란색 테두리
- 추천1/2/3: 초록색 테두리

**🔍 키워드 검색 어제의 인기영상 TOP 6**
- 썸네일 이미지와 함께 시각적으로 표시
- 각 영상의 제목, 채널명, 구독자수, 조회수 정보 제공
- 썸네일 클릭 시 해당 영상으로 바로 이동

**💡 AI 분석 결과**
- 키워드 검색으로 수집한 영상들의 TOP 30 분석
- 키워드 활용 방법과 성공 요인 분석

**🎯 최종 정리**
- 전체 분석을 종합한 실전 콘텐츠 전략
- 구체적인 제목 작성 팁
- 경쟁 영상과의 차별화 방법
- 핵심 키워드와 강조 내용을 볼드체로 표시

### 3. 부가 기능

**재분석 기능**
- 분석 결과가 마음에 들지 않으면 "🔄 다시 분석하기" 버튼으로 재분석할 수 있습니다.
- 같은 영상 데이터로 새로운 관점의 분석을 받을 수 있습니다.
- 재분석 중에는 버튼이 빨간색으로 변하고 "⏳ 분석중..."으로 표시됩니다.

**전체 복사 기능**
- "📋 전체 복사" 버튼으로 분석 결과를 텍스트로 복사할 수 있습니다.
- TOP 6 영상의 링크도 함께 복사되어 별도로 정리하기 편리합니다.

## 사용 방법

### AI 분석 시작하기
1. 유튜브 영상 검색 기능에서 분석할 영상들을 검색합니다.
2. 필터를 설정하여 원하는 영상만 테이블에 남깁니다.
3. "AI 콘텐츠 분석" 버튼을 클릭합니다.
4. 분석 진행 상황이 단계별로 표시됩니다:
   - 1단계: 선택한 영상 분석 중... (10%)
   - 2단계: 핵심 키워드 추출 중... (35%)
   - 3단계: 키워드로 영상 수집 중... (45%)
   - 4단계: 수집된 영상 재분석 중... (65%)
   - 5단계: 최종 분석 중... (90%)
   - 분석 완료! (100%)
5. 분석이 완료되면 "AI 콘텐츠 분석 리포트" 창이 자동으로 열립니다.

### 분석 리포트 활용하기
1. **선택한 영상 분석 섹션**: 선택한 영상들의 공통 패턴을 확인합니다.
2. **추출된 키워드 섹션**: 6개의 핵심 키워드를 재검색에 사용합니다.
3. **TOP 6 영상 섹션**: 
   - 썸네일을 클릭하여 관심 있는 영상을 바로 확인할 수 있습니다.
   - 조회수 색상으로 인기도를 직관적으로 파악합니다:
     - 회색: 1만 미만
     - 주황색: 1만~5만
     - 파란색: 5만~10만
     - 초록색: 10만 이상
4. **AI 분석 결과 섹션**: 키워드 활용 전략을 자세히 읽어봅니다.
5. **최종 정리 섹션**: 실전에 적용할 콘텐츠 전략을 확인합니다.

### 재분석하기
1. 분석 결과 창 하단의 "🔄 다시 분석하기" 버튼을 클릭합니다.
2. 버튼이 빨간색으로 변하고 "⏳ 분석중..."으로 표시됩니다.
3. 5단계 분석이 다시 진행됩니다 (약 30초~1분 소요).
4. 새로운 분석 결과가 기존 내용을 대체하여 표시됩니다.
5. 완료 메시지가 나타나면 새로운 관점의 분석을 확인할 수 있습니다.

### 분석 결과 저장하기
1. "📋 전체 복사" 버튼을 클릭합니다.
2. 시스템 소리(beep)가 나면 복사가 완료된 것입니다.
3. 메모장, 워드, 구글 문서 등 원하는 곳에 붙여넣기(Ctrl+V)합니다.
4. 복사된 내용에는 다음이 포함됩니다:
   - 날짜 정보
   - 선택한 영상 분석
   - 추출된 6개 키워드
   - 키워드 기반 영상 분석
   - TOP 6 영상 정보 (제목, 채널명, 구독자수, 조회수, 링크)
   - 최종 전략 정리

## 자주 묻는 질문(FAQ)

### Q: AI 분석은 얼마나 걸리나요?
A: 선택한 영상 수와 네트워크 상태에 따라 다르지만, 보통 30초~1분 정도 소요됩니다. 
5단계 분석 과정이 순차적으로 진행되며, 각 단계마다 진행 상황이 표시됩니다.

### Q: 최소 몇 개의 영상을 선택해야 하나요?
A: 최소 5개 이상의 영상을 선택하는 것을 권장합니다. 
영상이 많을수록 더 정확한 패턴 분석이 가능하지만, 1단계에서는 조회수 기준 상위 30개만 분석에 사용됩니다.

### Q: 어떤 영상을 선택하면 좋은 분석 결과를 얻을 수 있나요?
A: 자신이 만들고 싶은 콘텐츠 주제와 관련된 영상 중 조회수가 높은 영상들을 선택하세요. 
결이 같은 영상일수록 분석 및 추천영상이 내 주제와 관련도가 깊어집니다.

### Q: 추출된 키워드가 적절하지 않아요.
A: "🔄 다시 분석하기" 버튼을 눌러 재분석을 시도해보세요. 
AI가 다른 관점에서 키워드를 추출할 수 있습니다. 
여러 번 재분석하여 가장 적합한 키워드를 찾으실 수 있습니다.

### Q: Gemini API 키는 어디서 설정하나요?
A: 튜브렌즈 설정 메뉴에서 Gemini API 키를 설정할 수 있습니다. 

### Q: 재분석할 때마다 비용이 발생하나요?
A: Gemini API의 무료 할당량 내에서는 비용이 발생하지 않습니다. 
무료 할당량 내에서 재분석을 여러 번 시도할 수 있습니다.

## 문제 해결

### "🔑 API 키 인증 오류" 메시지가 나타나는 경우
**원인**: Gemini API 키가 올바르지 않거나 만료되었습니다.
**해결 방법**:
1. 튜브렌즈 설정에서 Gemini API 키를 다시 확인합니다.
2. Google AI Studio에서 새로운 API 키를 발급받습니다.
3. 설정에서 새 API 키를 입력하고 저장합니다.
4. 분석을 다시 시도합니다.

### "📊 Gemini API 하루 할당량 초과" 메시지가 나타나는 경우
**원인**: 오늘의 Gemini API 무료 할당량을 모두 사용했습니다.
**해결 방법**:
1. 다음 날 00시(한국시간)에 할당량이 초기화될 때까지 기다립니다.
2. 또는 Google AI Studio에서 새로운 API 키를 생성하여 교체합니다.

### "⏰ AI 요청 속도 초과" 메시지가 나타나는 경우
**원인**: Gemini API의 분당 요청 제한(15회)을 초과했습니다.
**해결 방법**:
1. 1분 정도 기다린 후 다시 시도합니다.
2. 너무 자주 재분석을 하지 않도록 주의합니다.

### "🔧 Gemini 서버 오류" 메시지가 나타나는 경우
**원인**: Gemini AI 서버에 일시적인 문제가 발생했습니다.
**해결 방법**:
1. 프로그램이 자동으로 3번 재시도하므로 잠시 기다립니다.
3번 재시도 후에도 실패하면:
2. 5~10분 후 다시 시도합니다.
3. Google AI Studio 상태를 확인합니다.
4. 계속 문제가 발생하면 나중에 다시 시도합니다.

### 분석 결과가 너무 짧거나 부실한 경우
**원인**: AI가 분석할 데이터가 부족하거나 일시적인 API 응답 문제일 수 있습니다.
**해결 방법**:
1. "🔄 다시 분석하기"를 눌러 재분석을 시도합니다.
2. 더 많은 영상을 선택하고 처음부터 다시 분석합니다.
3. 조회수가 더 높은 영상들을 선택해봅니다.

### YouTube API 키도 필요한가요?
A: 네, YouTube API 키는 3단계(키워드로 영상 수집)와 4단계(영상 상세 정보 가져오기)에서 사용됩니다. 
튜브렌즈 설정에서 YouTube API 키가 올바르게 설정되어 있는지 확인하세요.
한번 분석시 유튜브API 할당량은 약 326 units이 소모됩니다.


**참고사항**:
- AI 분석 기능은 Gemini API와 YouTube API를 모두 사용합니다.
- 두 API 모두 무료 할당량이 있으므로, 과도한 사용은 피해주세요.
- 분석 결과는 AI의 제안일 뿐이며, 최종 콘텐츠 전략은 사용자의 판단이 중요합니다.

## 🎤 Basic TTS (무료 무제한)

### 주요 특징
- **완전 무료**: 비용 부담 없이 무제한 사용
- **빠른 속도**: 대본 변환이 빠름
- **간편함**

### 목소리 옵션 (6가지)

**남자 목소리 (3가지)**
- **준서**
- **우진**
- **태훈**

**여자 목소리 (3가지)**
- **지아**
- **소연**
- **민서**

### 음성 조절 기능

**속도 조절 (-5 ~ +5)**
- -5: 매우 느리게 (70% 속도)
- 0: 표준 속도 (100%)
- +5: 매우 빠르게 (130% 속도)

**높낮이 조절 (-5 ~ +5)**
- -5: 매우 낮은 톤 (중후한 느낌)
- 0: 표준 톤
- +5: 매우 높은 톤 (밝은 느낌)

### 사용 방법

1. **🎧 튜브 TTS 버튼 클릭**
   - 메인 화면에서 "🎧 튜브 TTS" 버튼 클릭
   - TTS 창이 열림
   - **Basic TTS 탭**이 기본으로 선택됨

2. **텍스트 입력**
   - 상단 입력창에 대본 입력
   - 글자 수 실시간 표시
   - 5000자 이상은 빨간색으로 표시 (입력은 가능)

3. **목소리 선택**
   - 원하는 목소리 버튼 클릭 (파란색으로 강조)
   - 🔊 버튼으로 샘플 미리듣기 가능
     - "안녕하세요 튜브렌즈입니다" 샘플 재생

4. **음성 조절**
   - 속도 슬라이더 좌우 이동 (느리게 ↔ 빠르게)
   - 높낮이 슬라이더 좌우 이동 (낮게 ↔ 높게)
   - 현재 값 실시간 표시

5. **🎙️ 음성생성** 버튼 클릭
   - 음성 생성 시작
   - 진행 상황 표시 (경과 시간 표시)
   - **생성 완료 시 자동 재생**
   - 생성 중 "⏹ 생성중지" 버튼으로 언제든 취소 가능

6. ** 미리듣기** 버튼 클릭
   - 이미 생성한 음성 다시 재생
   - 재생 중 "⏹ 듣기중지" 버튼으로 중지

7. **💾 파일저장** 버튼 클릭
   - 저장 위치 선택창 열림
   - 파일명 지정 (기본: 목소리이름_날짜시간.mp3)
   - **MP3 형식**으로 저장
   - 저장 완료 후 "✅ 저장완료" 메시지 클릭 → 폴더 열기

## 🎙️ Plus TTS (AI 고품질)

### 주요 특징
- **자연스러운 음성**: Google Gemini TTS
- **감정 표현**: 9가지 말투 스타일 선택 가능
- **높은 품질**: 사람처럼 자연스러운 억양

### ⚠️ 사용 전 필수 준비

### 말투 스타일 (9가지)

- **차분하게**: 안정적이고 편안한 톤 (명상, 힐링)
- **화나게**: 강한 감정 표현 (드라마, 연기)
- **속삭이듯**: 부드럽고 조용한 톤 (ASMR, 수면 유도)
- **극적으로**: 드라마틱한 연출 (스토리텔링)
- **전문적으로**: 뉴스/다큐멘터리 스타일

## 🚨 에러 해결 가이드

### Plus TTS 에러

**🔑 API 키가 설정되지 않았습니다**
- **원인**: API 키 미등록
- **해결**: 
  1. 설정 메뉴 열기
  2. 스크롤 내려서 "Google AI Studio API(Gemini) 키 관리" 섹션
  3. Gemini API 키 추가하기 (유튜브 API X)

**📊 API 할당량이 초과되었습니다 (429 에러)**
- **원인**: 하루 10회 제한 초과
- **자동 처리**: 다른 등록된 API 키로 자동 전환
- **해결**: 
  1. "다시 생성" 버튼 클릭 (자동으로 다음 키 사용)
  2. 모든 키 소진 시:
     - 내일 오후 5시까지 대기, 또는
     - 새 구글 계정으로 API 키 추가 발급

**😓 서버가 계속 혼잡합니다 (503 에러)**
- **원인**: Gemini 서버 과부하 (피크 시간대)
- **자동 처리**: 프로그램이 자동으로 3번 재시도
- **해결**: 
  1. 1~2분 후 다시 시도
  2. 보통 1~2분이면 정상화됨

**🔑 API 키가 유효하지 않습니다**
- **원인**: 잘못 복사했거나 만료된 키
- **해결**: 
  1. Google AI Studio 다시 접속
  2. API 키 재확인 (전체 복사했는지)
  3. 새 키 발급 후 재등록

**❌ Gemini API 응답이 없습니다**
- **원인**: 네트워크 문제 또는 일시적 API 오류
- **해결**: 
  1. 인터넷 연결 확인
  2. 잠시 후 재시도
  3. 다른 API 키로 시도

**❌ Gemini가 음성을 생성하지 못했습니다**
- **원인**: 텍스트가 Google 안전 필터에 걸림
- **해결**: 
  1. 텍스트 내용 확인 (욕설, 폭력, 선정성 제거)
  2. 텍스트 수정 후 재시도

**🌐 네트워크 연결에 실패했습니다**
- **원인**: 인터넷 연결 끊김
- **자동 처리**: 3번 재시도
- **해결**: 
  1. 인터넷 연결 확인
  2. Wi-Fi 재연결
  3. 방화벽/백신 프로그램 확인

### Basic TTS 에러

**❌ 음성 생성 오류**
- **원인**: 일시적인 서버 문제
- **해결**: 
  1. 텍스트 길이 줄여보기 (5000자 이하)
  2. 특수문자 제거
  3. 잠시 후 다시 시도
  4. 다른 목소리로 시도

### 공통 에러

**⚠️ 텍스트가 비어있습니다**
- **해결**: 텍스트 입력 후 재시도

**❌ 파일 저장 실패**
- **원인**: 쓰기 권한 없음 또는 용량 부족
- **해결**: 
  1. 다른 폴더 선택 (예: 바탕화면)
  2. 디스크 공간 확인
  3. 프로그램 관리자 권한으로 실행

**발음이 이상해요**
- **해결**: 
  1. 띄어쓰기 정확히 하기
  2. 숫자는 한글로 (예: 10 → 십)
  3. 특수문자 제거
  4. 다른 목소리로 시도

## 자주 묻는 질문 (FAQ)

### 기본 사용

**Q: 처음 실행했는데 바로 사용 가능한가요?**
A: Basic TTS는 바로 사용 가능합니다! Plus TTS는 API 키 등록이 필요해요.

**Q: Basic과 Plus 중 뭘 써야 하나요?**
A: 
- 정보 전달 목적 → Basic TTS (무료 무제한)
- 감정 표현 중요 → Plus TTS (고품질)

**Q: 생성된 음성에 워터마크가 들어가나요?**
A: 절대 없습니다! 입력한 텍스트만 순수하게 변환됩니다.

**Q: 한 번에 몇 글자까지 가능한가요?**
A: 제한은 없지만 **2500자 이하 권장**합니다.

**Q: MP3와 WAV 차이는 뭔가요?**
A: 
- Basic TTS (MP3): 파일 작음, 대부분 편집 프로그램 호환
- Plus TTS (WAV): 음질 최상, 전문 편집용
- 두 형식 모두 영상 제작에 문제없어요!

### Plus TTS 관련

**Q: API 키 발급이 어려운데 도와주세요**
A: 
1. https://aistudio.google.com 접속
2. 구글 로그인
3. 우측 상단 "Get API key" 클릭
4. "Create API key" 클릭
5. 나온 키 전체 복사 (AIza~로 시작)
6. 튜브렌즈 설정에서 붙여넣기

**Q: API 키는 유료인가요?**
A: **완전 무료**입니다! 구글이 무료로 제공합니다. 하지만 유료 API 사용도 가능합니다.

**Q: 하루 10회면 너무 적은데요?**
A: 
- 구글 계정 3개 = API 키 3개 = 하루 30회
- 구글 계정 5개 = API 키 5개 = 하루 50회
- 필요한 만큼 계정 만들어서 등록하세요!

**Q: 할당량은 언제 초기화되나요?**
A: 한국 시간 기준 **오후 5시**에 초기화됩니다.

**Q: 503 에러가 자주 나는데요?**
A: Gemini 서버 혼잡할 때 발생합니다. **1~2분 후 재시도**하면 대부분 해결돼요.

**Q: 여러 API 키가 자동으로 전환되나요?**
A: 네! 첫 번째 키 소진 시 자동으로 두 번째 키 사용합니다.

### 기술적 문제

**Q: 생성 중 멈춰요**
A: "⏹ 생성중지" 버튼 눌러 취소 후 다시 시도하세요.

**Q: 저장한 파일이 재생 안 돼요**
A: 
1. 다른 미디어 플레이어로 열어보기
2. 영상 편집 프로그램에 직접 불러오기
3. 문제 지속 시 다시 생성

**Q: 발음이 이상해요**
A: 
- 띄어쓰기 정확히 하기
- 숫자는 한글로 쓰기 (10 → 십)
- 특수문자 제거하기
- 다른 목소리로 시도해보기

**Q: 인터넷 연결이 필요한가요?**
A: **Plus TTS만 필요**합니다. Basic TTS는 오프라인에서도 작동합니다!

# 튜브렌즈 대본 생성기 기능 상세 가이드
## 개요
튜브렌즈의 대본 생성기 기능은 유튜브 영상의 자막을 AI가 분석하여 새로운 콘텐츠 대본을 자동으로 생성해주는 강력한 기능입니다. 
원본 영상의 자막을 분석하고 그 구조와 특성을 파악한 후, 
완전히 새로운 표현과 구성으로 대본을 제작해 어문저작권 침해 없이 콘텐츠를 만들 수 있도록 도와줍니다.

## 주요 기능
### 1. 자막 분석
- **요약 분석**: 영상의 핵심 내용을 3-4문장으로 압축하여 정리합니다.
- **구성 방식 분석**: 영상의 구성을 도입-본론-결론 구조를 중심으로 단계별로 정리합니다.
- **초반 후킹 방법 분석**: 시청자의 관심을 끌기 위해 영상에서 사용한 방법들을 파악합니다.
- **핵심 키워드 추출**: 영상에서 자주 언급되는 주요 키워드와 그 의미를 정리합니다.

### 2. 대본 생성
- **AI 원본 재구성 모드**: 원본 자막의 주제와 구성을 유지하면서 표현만 완전히 바꾸어 어문저작권 침해 없이 대본을 생성합니다. 쇼츠 제작에 적합합니다.
- **AI 3단계 구조화 모드**: 분석된 내용을 바탕으로 인트로-본론-아웃트로 구조의 새로운 대본을 생성합니다. 롱폼 콘텐츠 제작에 적합합니다.
- **맞춤형 제목 생성**: 생성된 대본에는 클릭을 유도하는 매력적인 제목이 포함됩니다.
- **추가 프롬프트 지원**: 사용자가 원하는 특정 요구사항을 추가 프롬프트로 입력하여 대본 생성을 조정할 수 있습니다.

### 3. 대본 포맷팅
- **섹션별 구조화**: 생성된 대본은 섹션별로 구분되어 가독성이 높습니다.
- **강조 표시**: 중요한 부분, 질문, 구독 유도 문구 등이 자동으로 강조됩니다.
- **액션/감정 표현**: 필요한 부분에 액션이나 감정 표현이 적절히 포함됩니다.
- **복사 기능**: 생성된 대본을 원클릭으로 클립보드에 복사할 수 있습니다.

## 사용 방법
### 1. 대본 생성기 실행하기
1. 메인 화면에서 분석할 영상을 선택합니다.
2. 화면 상단의 '📝 분석 및 대본생성' 버튼을 클릭합니다.
3. 선택한 영상의 자막을 수집하고 분석을 시작합니다.

### 2. 분석 결과 확인하기
1. '분석 결과' 탭에서 영상의 자막 분석 정보를 확인합니다:
   - **요약**: 영상의 핵심 내용에 대한 간결한 요약
   - **구성 방식**: 영상의 단계별 구성 설명
   - **초반 후킹 방법**: 시청자 관심을 끌기 위한 영상의 방법
   - **주요 키워드**: 영상에서 중요하게 사용된 키워드와 의미

### 3. 대본 생성하기
1. '대본 생성' 탭으로 이동합니다.
2. 대본 스타일을 선택합니다:
   - **AI 원본 재구성**: 원본 자막의 주제와 구성을 유지하면서 새로운 대본 생성(쇼츠 방향성)
   - **AI 3단계 구조화**: 원본 자막을 참고하여 새로운 인트로-본론-아웃트로 구조의 대본 생성(롱폼 방향성)
3. 추가 프롬프트를 입력합니다(선택사항):
   - 대본 생성 시 특별히 반영되길 원하는 요구사항이나 방향성 입력
4. '대본 생성하기' 버튼을 클릭합니다.
5. 생성이 완료되면 대본 내용이 표시됩니다.
6. '대본 복사' 버튼을 클릭하여 생성된 대본을 클립보드에 복사할 수 있습니다.

## 대본 스타일 선택 가이드
### AI 원본 재구성 (쇼츠 방향성)
- **목적**: 어문저작권 침해 없이 원본 자막의 내용을 새롭게 표현
- **특징**: 
  - 원본 자막의 주제와 구성을 그대로 유지
  - 초반 후킹 방식을 유지하면서 다양한 표현으로 변경
  - 불필요한 반복이나 군더더기 표현 정리
- **적합한 경우**: 
  - 쇼츠 영상 제작 시
  - 인기 있는 영상의 내용을 참고하여 비슷한 주제로 빠르게 대본을 작성하고 싶을 때
  - 원본 영상과 유사한 구성을 유지하고 싶을 때

### AI 3단계 구조화 (롱폼 방향성)
- **목적**: 원본 자막을 참고하여 롱폼 구조의 대본 생성
- **특징**: 
  - 인트로-본론-아웃트로의 명확한 3단계 구조
  - 강력한 시청자 후킹 요소 포함
  - 시청 유지를 위한 전략적 구성
  - 시청자 참여 유도 요소 추가
- **적합한 경우**: 
  - 롱폼 영상 제작 시
  - 원본 영상을 참고하되 자신만의 독창적인 콘텐츠를 만들고 싶을 때
  - 시청자 참여와 유지율을 높이는 구조적인 대본이 필요할 때

## 추가 프롬프트 활용 팁

- **다국어 콘텐츠**: 생성된 대본을 번역하여 다국어 콘텐츠 제작의 기반으로 활용

## 자주 묻는 질문(FAQ)
### Q: 자막이 없는 영상은 분석할 수 없나요?
A: 네, 대본 생성기는 영상의 자막을 기반으로 작동하므로 자막이 없는 영상은 분석할 수 없습니다. 
자막이 제공되지 않는 영상의 경우 "자막 없음" 메시지가 표시됩니다.

### Q: 생성된 대본은 저작권 문제가 없나요?
A: AI 원본 재구성 모드는 어문저작권 침해 없이 원본 자막의 주제와 구성은 유지하면서 표현만 완전히 다르게 변경합니다. 그러나 법적 해석은 국가와 상황에 따라 다를 수 있으니, 최종적인 콘텐츠 사용 전에 저작권법을 확인하는 것이 좋습니다.

### Q: 대본 생성에 얼마나 시간이 걸리나요?
A: 자막의 길이와 복잡성, 선택한 대본 스타일, 서버 부하 상태에 따라 달라질 수 있지만, 일반적으로 10-30초 이내에 완료됩니다.

### Q: API 키 오류가 발생합니다.
A: Google AI Studio의 Gemini API 키가 필요합니다. 
설정 메뉴에서 유효한 API 키를 추가해주세요.

### Q: 생성된 대본이 너무 길거나 짧아요.
A: 대본은 원본 자막의 길이를 기준으로 생성됩니다. 
추가 프롬프트를 통해 "더 짧게" 또는 "더 자세히"와 같은 요청을 추가하여 길이를 조절할 수 있습니다.

### Q: 특정 용어나 브랜드를 포함해야 하는데 가능한가요?
A: 네, 추가 프롬프트 영역에 "다음 용어/브랜드를 반드시 포함: XXX"와 같이 입력하면 해당 내용이 대본에 포함됩니다.

## 문제 해결
### API 키 관련 문제
- **오류 메시지**: "API 키가 올바르지 않습니다"
  - 해결: 설정 메뉴에서 Google AI Studio의 Gemini API 키를 확인하고 업데이트하세요.
- **오류 메시지**: "API 서비스 요청 속도 제한에 도달했습니다"
  - 해결: 10초 정도 기다린 후 다시 시도하거나, 설정에서 다른 API 키를 사용하세요.

### 자막 수집 문제
- **오류 메시지**: "이 영상에는 자막이 없습니다"
  - 해결: 자막이 있는 다른 영상을 선택하세요.
- **오류 메시지**: "자막 수집 중 오류가 발생했습니다"
  - 해결: 다른 영상을 시도하거나, 네트워크 연결을 확인하세요.

### 대본 생성 문제
- **오류 메시지**: "분석 중 문제가 발생했습니다"
  - 해결: 인터넷 연결을 확인하고 잠시 후 다시 시도하세요.
- **생성된 대본이 원하는 내용과 다른 경우**:
  - 해결: 더 구체적인 추가 프롬프트를 입력하거나, 다른 대본 스타일로 다시 시도하세요.

### 실행 문제
- **대본 생성기가 응답하지 않는 경우**:
  - 해결: 대화상자를 닫고 다시 시도하거나, 프로그램을 재시작하세요.
- **탭 전환이 작동하지 않는 경우**:
  - 해결: 대화상자를 닫고 다시 실행하세요.

# 튜브렌즈 다운로드 기능 상세 가이드
## 개요
튜브렌즈의 다운로드 기능은 유튜브 영상을 다양한 형식과 품질로 쉽게 다운로드할 수 있게 해주는 편리한 도구입니다. 
MP4 비디오 형식과 MP3 오디오 형식 모두 지원하며, 다양한 해상도 옵션을 통해 사용자의 필요에 맞는 최적의 품질로 콘텐츠를 저장할 수 있습니다.
Youtube API와는 상관이 없기때문에, 특별한 횟수 제한없이 다운로드 할 수 있습니다.

## 주요 기능
### 1. 다양한 형식 지원
- **MP4 비디오 다운로드**: 영상을 비디오 형식으로 다운로드하여 오프라인에서 시청할 수 있습니다.
- **MP3 오디오 다운로드**: 영상의 오디오만 추출하여 음악이나 팟캐스트로 저장할 수 있습니다.

### 2. 품질 설정 옵션
- **최고화질**: 가능한 최고 품질의 영상을 다운로드합니다.
- **1080p(Full HD)**: 고화질 영상을 다운로드합니다.
- **720p(HD)**: 중간 수준의 화질로 다운로드합니다.
- **480p**: 데이터 사용량을 줄이면서 적절한 화질을 유지합니다.
- **360p**: 저용량, 저화질 옵션으로 저장 공간이 제한된 경우에 적합합니다.

### 3. 편집 도구 호환성
- 다운로드한 MP4 파일은 자동으로 널리 사용되는 편집 소프트웨어(프리미어 프로, Final Cut 등)와 호환되는 형식으로 변환됩니다.
- 비디오 코덱은 h.264로, 오디오 코덱은 AAC로 처리되어 최대한의 호환성을 보장합니다.
- Youtube API와는 상관이 없기때문에, 특별한 횟수 제한없이 다운로드 할 수 있습니다.

### 4. 일괄 다운로드
- 한 번에 여러 영상을 선택하여 일괄 다운로드할 수 있습니다.
- 개별 URL을 입력하여 특정 영상을 다운로드할 수도 있습니다.

## 사용 방법
### 검색 결과에서 영상 다운로드하기
1. 메인 화면에서 영상을 검색합니다.
2. 다운로드할 영상을 선택합니다(영상 왼쪽의 N열을 클릭하여 선택).
   - 영상을 선택하지 않으면 현재 표시된 모든 영상이 다운로드 대상이 됩니다.
3. 원하는 형식의 다운로드 버튼을 클릭합니다:
   - **📺 MP4 다운**: 비디오와 오디오를 포함한 영상 파일 다운로드
   - **🎵 MP3 다운**: 오디오만 추출하여 음악 파일로 다운로드
4. 파일을 저장할 위치를 선택합니다.
5. 다운로드가 시작되고 진행 상황이 프로그레스 바에 표시됩니다.
6. 다운로드가 완료되면 저장 위치가 자동으로 열립니다.

### URL로 직접 다운로드하기
1. 메인 화면 상단의 URL 입력 필드에 유튜브 영상 URL을 붙여넣습니다.
2. 품질 옵션을 드롭다운 메뉴에서 선택합니다(기본값: 최고화질).
3. 'URL 다운' 버튼을 클릭합니다.
4. 파일을 저장할 위치를 선택합니다.
5. 다운로드가 진행되고 완료되면 저장 위치가 자동으로 열립니다.

### 다운로드 중단하기
1. 다운로드가 진행 중일 때 다운로드 버튼이 '중지' 버튼으로 변경됩니다.
2. 다운로드를 중단하려면 '중지' 버튼을 클릭합니다.
3. 다운로드가 즉시 중단되고 프로그레스 바가 초기화됩니다.

## 품질 설정 가이드
### MP4 다운로드 품질 권장사항
- **⚡ 최고화질**: 고해상도 모니터나 TV에서 시청하거나 영상 편집할 때 권장
- **🎥 1080p**: 일반적인 컴퓨터나 TV 시청용으로 적합
- **📺 720p**: 노트북이나 태블릿에서 시청할 때 권장
- **📱 480p**: 모바일 기기에서 시청하거나 저장 공간이 제한된 경우 적합
- **💻 360p**: 데이터 사용량 최소화나 매우 제한된 저장 공간에 적합

### MP3 다운로드 특성
- MP3 다운로드는 항상 최고 품질의 오디오(320kbps)로 변환됩니다.
- 영상의 모든 오디오 트랙이 포함됩니다(배경 음악, 음성 등).

## 저장 파일 정보
### MP4 파일
- **파일명**: 원본 영상의 제목(특수문자는 제거됨)
- **확장자**: .mp4
- **비디오 코덱**: H.264 (AVC)
- **오디오 코덱**: AAC (192kbps)
- **호환성**: 대부분의 미디어 플레이어, 편집 소프트웨어와 호환

### MP3 파일
- **파일명**: 원본 영상의 제목(특수문자는 제거됨)
- **확장자**: .mp3
- **오디오 코덱**: MP3
- **비트레이트**: 320kbps (고품질)
- **호환성**: 모든 음악 플레이어와 호환

## 자주 묻는 질문(FAQ)
### Q: 다운로드 속도가 느립니다. 어떻게 해야 하나요?
A: 다운로드 속도는 인터넷 연결 상태, 서버 상태, 영상 품질 등 여러 요소에 영향을 받습니다. 
더 낮은 품질 옵션을 선택하거나, 네트워크 상태를 확인해보세요. 
또한 피크 시간대를 피해 다운로드하는 것도 도움이 됩니다.

### Q: '다운로드 중 오류가 발생했습니다' 메시지가 나타납니다.
A: 다음 사항을 확인해보세요:
   - 인터넷 연결 상태
   - 유튜브 영상이 비공개이거나 제한된 영상이 아닌지
   - FFmpeg가 올바르게 설치되어 있는지
   - 충분한 디스크 공간이 있는지
   - URL이 올바른지

### Q: 다운로드한 MP4 파일이 편집 프로그램에서 열리지 않습니다.
A: 튜브렌즈는 편집 프로그램과의 호환성을 위해 MP4 파일을 최적화합니다. 
그러나 간혹 특정 편집 프로그램과 호환되지 않을 수 있습니다. 
이 경우, HandBrake와 같은 추가 변환 소프트웨어를 사용해 다시 인코딩해보세요.

### Q: 'FFmpeg 설치가 필요합니다' 메시지가 나타납니다.
A: 튜브렌즈의 다운로드 기능은 FFmpeg를 사용합니다. 
메시지가 나타나면 '예'를 클릭하여 자동 설치를 진행하세요. 
자동 설치에 실패한 경우, [FFmpeg 공식 웹사이트](https://ffmpeg.org/download.html)에서 수동으로 다운로드하여 설치할 수 있습니다.

### Q: 여러 개의 영상을 한 번에 다운로드할 수 있나요?
A: 네, 검색 결과에서 여러 영상을 선택(N열 클릭)한 후 MP4 다운 또는 MP3 다운 버튼을 클릭하면 선택한 모든 영상을 순차적으로 다운로드합니다.
### Q: mp4 또는 mp3를 몇 개나 다운받을수 있나요?
A: Youtube API와는 상관이 없기때문에, 특별한 횟수 제한없이 다운로드 할 수 있습니다.

## 문제 해결
### 다운로드가 시작되지 않는 경우
- 인터넷 연결 상태를 확인하세요.
- URL이 올바른지 확인하세요.
- FFmpeg가 설치되어 있는지 확인하세요.
- 다른 다운로드가 진행 중이면 완료 후 다시 시도하세요.

### FFmpeg 설치 문제
- 자동 설치에 실패한 경우 수동으로 설치를 시도해보세요:
  1. [FFmpeg 공식 웹사이트](https://ffmpeg.org/download.html)에서 Windows 버전 다운로드
  2. 압축 해제 후 bin 폴더의 내용을 `%LOCALAPPDATA%\Programs\ffmpeg\bin\` 경로에 복사
  3. 튜브렌즈를 재시작하세요.

### 저장 경로 관련 문제
- 선택한 저장 경로에 쓰기 권한이 있는지 확인하세요.
- 경로에 특수 문자나 매우 긴 경로명이 포함되어 있지 않은지 확인하세요.
- 파일명에 윈도우에서 지원하지 않는 문자(`\ / : * ? " < > |`)가 포함된 경우 자동으로 제거됩니다.

### 다운로드된 파일 재생 문제
- VLC 미디어 플레이어와 같은 다양한 코덱을 지원하는 플레이어로 시도해보세요.
- 파일이 손상되었다면 다시 다운로드를 시도하세요.
- MP4 파일이 제대로 변환되지 않았다면 다른 품질 옵션으로 다시 시도해보세요.

# 튜브렌즈 업데이트 및 인증 시스템 상세 가이드
## 개요
튜브렌즈는 자동 업데이트 시스템과 인증 관리 기능을 갖추고 있습니다. 
이 시스템을 통해 사용자는 항상 최신 버전의 프로그램을 사용할 수 있으며, 라이센스 인증을 안전하게 유지할 수 있습니다. 이 가이드에서는 튜브렌즈의 업데이트 시스템과 인증 프로세스에 대해 상세히 설명합니다.

## 자동 업데이트 시스템
### 업데이트 확인 프로세스
1. **자동 버전 확인**: 프로그램 시작 시 데이터베이스에 저장된 최신 버전 정보를 자동으로 확인합니다.
2. **버전 비교**: 현재 설치된 버전과 최신 버전을 비교하여 업데이트가 필요한지 결정합니다.
3. **업데이트 알림**: 새 버전이 있는 경우 사용자에게 알림 창을 표시하여 업데이트 여부를 선택할 수 있게 합니다.

### 업데이트 수행 방법
1. **업데이트 대화상자 확인**:
   - 새 버전이 있을 때 표시되는 대화상자에서 현재 버전, 최신 버전, 업데이트 내용을 확인할 수 있습니다.
   - '업데이트' 또는 '다음에 하기' 버튼을 선택할 수 있습니다.

2. **업데이트 선택 시**:
   - '업데이트' 버튼을 클릭하면 자동 업데이트 프로세스가 시작됩니다.
   - 업데이트 준비 진행 상황이 진행 표시줄로 표시됩니다.
   - 업데이터 프로그램이 다운로드되고 실행되어 최신 버전의 프로그램을 설치합니다.
   - 업데이트가 완료되면 새 버전의 프로그램이 자동으로 시작됩니다.

3. **다음에 하기 선택 시**:
   - 현재 버전의 프로그램이 정상적으로 실행됩니다.
   - 다음 프로그램 실행 시 다시 업데이트 확인이 이루어집니다.

### 업데이트 문제 해결
- **업데이트 오류 발생 시**: 오류 메시지가 표시되며, 프로그램은 현재 버전으로 계속 실행됩니다.
- **인터넷 연결 문제**: 인터넷 연결이 없는 경우 버전 확인이 실패할 수 있으며, 이 경우 프로그램은 현재 버전으로 실행됩니다.
- **업데이터 다운로드 실패**: 업데이터를 다운로드할 수 없는 경우 오류 메시지가 표시되고, 현재 버전의 프로그램이 실행됩니다.

## 인증 시스템
### 인증 프로세스
1. **첫 실행 시 인증키 요청**:
   - 프로그램을 처음 실행하거나 인증 정보가 없는 경우, 인증키 입력 대화상자가 표시됩니다.
   - 인증키를 입력하면 데이터베이스에서 유효성이 검증됩니다.

2. **인증키 유효성 검사**:
   - 입력한 인증키가 라이센스 데이터베이스에 존재하는지 확인합니다.
   - 인증키가 이미 다른 컴퓨터에서 사용 중인지 확인합니다.
   - 인증키의 만료일을 확인하여 유효한 기간인지 검사합니다.

3. **인증 정보 저장**:
   - 유효한 인증키는 로컬 시스템(AppData 폴더)에 저장되어 다음 실행 시 자동으로 로드됩니다.
   - 컴퓨터 ID와 함께 저장되어 라이센스의 무단 복제를 방지합니다.

### 인증 정보 백업 및 복원
- **자동 백업**: 프로그램 업데이트 시 기존 인증 정보가 자동으로 백업됩니다.
- **자동 복원**: 업데이트 후 백업된 인증 정보가 자동으로 복원되어 다시 인증키를 입력할 필요가 없습니다.
- **백업 위치**: 인증 정보는 `%LOCALAPPDATA%\TubeLens` 폴더에 백업됩니다.

# 튜브렌즈 고급 검색 및 국가별 분석 가이드
## 국가별 검색 기능
튜브렌즈는 다양한 국가의 유튜브 트렌드와 인기 영상을 분석할 수 있습니다.

### 국가 선택 기능
- **지원 국가**: 한국(KR), 일본(JP), 미국(US), 대만(TW), 영국(GB), 캐나다(CA), 호주(AU), 독일(DE), 프랑스(FR), 스페인(ES), 브라질(BR), 인도(IN), 러시아(RU)
- **작동 방식**: 국가 선택 시 해당 국가의 YouTube 데이터를 기준으로 검색
- **설정 방법**: 왼쪽 사이드바의 '지역 선택' 드롭다운에서 원하는 국가 선택

### 자동 언어 적용
- **기능**: 국가 선택 시 해당 국가의 주요 언어가 자동으로 검색 설정에 적용됨
- **예시**: 일본(JP) 선택 시 일본어(ja) 콘텐츠 우선, 한국(KR) 선택 시 한국어(ko) 콘텐츠 우선

### 자동 번역 기능
- **개요**: 국가 선택 후 검색 시 검색어가 해당 국가 언어로 자동 번역됨
- **번역 과정**: 검색 버튼 클릭 시 자동으로 번역되며 상태바에 번역 결과 표시
- **지원 언어**: 모든 지원 국가의 언어(한국어, 일본어, 영어, 독일어, 프랑스어 등)

## 키워드 없는 검색 기능
### 인기 영상 검색
- **기능**: 검색어 없이 국가 선택만으로 해당 국가의 인기 영상 검색 가능
- **작동 방식**: 검색창을 비우고 검색 버튼 클릭
- **결과**: 선택한 국가에서 인기 있는 영상이 조회수순으로 표시

### 검색어 없이 조회수순 검색 관련 안내
튜브렌즈는 국가별 인기 영상을 검색할 수 있지만, 일부 제한사항이 있습니다:
**미국, 일본 등 큰 시장의 인기 영상 검색**
* 미국과 같은 대규모 시장에서는 매일 엄청난 양의 콘텐츠가 업로드됩니다
* YouTube API는 이렇게 방대한 데이터 중 일부만 샘플링하여 제공합니다
* 이로 인해 실제 가장 높은 조회수의 영상이 누락될 수 있습니다

**더 정확한 결과를 얻으려면:**
1. **기간 설정**: 3개월 이내, 1개월 이내, 7일 이내 등 최근 기간 선택
2. **키워드 사용**: 특정 분야나 주제에 관한 키워드 추가
3. **카테고리 세분화**: 더 구체적인 주제로 검색 범위 좁히기

### 기간 설정 가이드
- **최적 설정**: 1개월 이내, 7일 이내, 3개월 이내
- **주의사항**: 6개월 이상 또는 '모든 기간' 설정 시 정확도 떨어짐
- **직접 날짜 선택**: 90일 이내 기간 선택 권장

## 자막 수집 고급 기능
### 국가별 자막 우선순위
- **기능**: 선택한 국가에 맞는 언어의 자막을 우선적으로 수집
- **우선순위 예시**:
  - 한국(KR): 한국어 > 영어
  - 일본(JP): 일본어 > 영어
  - 미국(US): 영어
  - 독일(DE): 독일어 > 영어
  - 등등

### 자막 수집 방법
- **개별 수집**: 영상 행 클릭 후 자막 열 클릭 시 해당 영상의 자막만 수집
- **일괄 수집**: 여러 영상 선택 후 '자막 수집' 버튼 클릭
- **전체 수집**: 아무 영상도 선택하지 않고 '자막 수집' 버튼 클릭하면 모든 영상의 자막 수집

### 자막 형식
- **표시 방식**: 수집된 자막은 테이블의 자막 열에 표시되며, 클릭 시 전체 내용 확인 가능
- **저장 방법**: 엑셀로 저장 시 자막 전체 내용 포함

### 국가별 검색 정확도
- **검색어 중요성**: 키워드의 언어가 검색 결과에 가장 큰 영향을 미침
- **자동 번역 효과**: 국가 선택 후 검색어 자동 번역으로 해당 국가 결과의 정확도 향상
- **언어 설정 효과**: 언어 설정(`relevanceLanguage`)을 통해 해당 국가 콘텐츠에 우선순위 부여

## 자주 묻는 질문(FAQ)
### Q: 검색어 없이 검색했을 때 왜 일부 고조회수 영상이 누락되나요?
A: YouTube API의 한계로 인해 '모든 기간' 설정에서는 완벽한 조회수 정렬이 어렵습니다. 
3개월 이내 기간으로 설정하면 좀 더 정확한 결과를 얻을 수 있습니다.

### Q: 같은 검색어로 국가만 바꿔도 결과가 비슷합니다
A: 검색어를 해당 국가 언어로 직접 입력하면 더 정확한 국가별 결과를 얻을 수 있습니다. 
자동 번역 기능이 이를 도와줍니다.

### Q: 자막 수집이 안 되는 영상이 있습니다
A: 모든 영상에 자막이 있는 것은 아닙니다. 
일부 영상은 자막이 비활성화되어 있거나 자막이 없을 수 있습니다.

### Q: 국가 설정 후 검색 시 원래 검색어가 바뀌는 이유는 무엇인가요?
A: 자동 번역 기능이 작동하여 검색어를 해당 국가 언어로 변환한 것입니다. 
이는 더 정확한 국가별 검색 결과를 위한 것입니다.

### Q: 한국어를 입력해도 외국영상이 뜨는데요?
A: 튜브렌즈는 유튜브api를 통한 영상 수집을 하는데,
국가설정 '한국', 검색어 '한국어' 라고 해도,
유튜브api에서 제공하는 것은
'한국 영상' 이 아니라, '한국에서 시청 가능한' 영상을 검색하고 제공하기 때문입니다.

좀 더 엄밀히 말씀드리면,
국가설정 '한국' = 한국에서 볼 수 있는 영상 (한국 영상만이 아니라)
언어 '한국어' = 한국어와 관련성 높은 영상 (하지만 다른 언어도 포함)
즉, "한국 출신 영상만" 검색하는 게 아니라서 외국 영상도 섞여 나옵니다.

현재 완전한 해결방법은 없으나,
1) 구체적인 키워드를 입력해주세요. 단순히 '쇼핑', '버스' 보다는 '쇼핑쇼츠 하는 법'
2) '결과내 재검색' 기능 활용
: 예시로 '쇼핑' 검색시, 하단에 있는 '결과내 재검색'에 다시 한국어로 '쇼핑'을 입력하면, 수집된 영상내에서 '쇼핑'단어가 들어간 결과물만 표시됩니다.
위의 두가지 방법으로 원하는 결과를 얻으시기 바랍니다.

# 튜브렌즈 타임라인 요약 기능 가이드
## 개요
타임라인 요약 기능은 유튜브 영상의 주요 장면을 자동으로 추출하여 시간별로 정리해주는 기능입니다. 
이 기능을 통해 영상의 핵심 내용을 빠르게 파악하고, 중요 장면의 이미지와 설명을 한눈에 볼 수 있습니다.

## 타임라인 요약 기능의 특징
- **자동 장면 추출**: AI가 영상에서 중요한 장면을 자동으로 감지합니다
- **이미지 캡처**: 각 장면의 대표 이미지를 고품질로 제공합니다
- **텍스트 요약**: 각 장면에 대한 설명을 자동으로 생성합니다
- **시간 정보**: 각 장면의 정확한 시간대를 표시합니다
- **두 가지 모드**: 롱폼(긴 영상)과 쇼츠(짧은 영상) 모드를 선택할 수 있습니다

## 타임라인 요약 사용 방법
### 기본 사용법
1. 원하는 유튜브 영상을 검색하여 선택합니다
2. "분석 및 대본생성" 버튼을 클릭합니다
3. 열린 창에서 "타임라인 요약" 탭을 선택합니다
4. "타임라인 생성" 버튼을 클릭합니다
5. 모드 선택 창에서 원하는 모드를 선택합니다 (롱폼 또는 쇼츠)
6. 생성이 완료되면 타임라인 카드가 시간순으로 표시됩니다

### 모드 선택하기
두 가지 타임라인 생성 모드 중에서 선택할 수 있습니다:
- **🏆 롱폼 모드**: 3~10분 길이의 일반 영상에 적합합니다. 
큰 장면 변화를 포착하고 영상의 주요 구간을 추출합니다.
- **✍️ 쇼츠 모드**: 1분 이내의 짧은 영상에 적합합니다. 
짧은 영상에서 임팩트 있는 순간을 더 정밀하게 포착합니다.

### 타임라인 카드 사용하기
각 타임라인 카드는 다음 정보를 제공합니다:

- **타임스탬프**: 해당 장면의 시작 및 종료 시간 (00:00~00:00 형식)
- **제목**: 장면의 주요 내용을 요약한 제목
- **설명**: 장면에 대한 상세 설명
- **이미지**: 해당 장면의 대표 프레임 이미지

각 카드에서 사용할 수 있는 기능:
- **이미지 클릭**: 이미지를 클릭하면 큰 창에서 볼 수 있습니다
- **복사** 버튼: 해당 카드의 시간, 제목, 설명을 클립보드에 복사합니다
- **이미지 저장** 버튼: 해당 장면의 이미지를 저장합니다

### 일괄 작업 기능
타임라인 요약 화면 하단에 있는 버튼을 통해 일괄 작업을 수행할 수 있습니다:

- **타임라인 생성**: 새로운 타임라인을 생성합니다
- **모든 이미지 저장**: 모든 타임라인 이미지를 한 번에 저장합니다
- **모든 설명 복사**: 모든 타임라인 텍스트를 클립보드에 복사합니다

## 진행 상황 모니터링
타임라인 생성 중에는 진행 상황이 실시간으로 표시됩니다:

- **진행 바**: 전체 진행 상황을 시각적으로 표시합니다
- **상태 메시지**: 현재 진행 중인 작업에 대한 설명을 제공합니다
- **세부 진행 정보**: 세부적인 작업 단계를 텍스트로 표시합니다

타임라인 생성은 영상 길이와 복잡도에 따라 몇 분이 소요될 수 있습니다.

## 자주 묻는 질문 (FAQ)
### 타임라인 생성이 너무 오래 걸려요
타임라인 생성은 영상의 길이와 복잡도에 따라 시간이 걸릴 수 있습니다. 
특히 긴 영상이나 내용이 복잡한 영상은 처리 시간이 더 길어질 수 있습니다. 

### 타임라인이 생성되지 않아요
다음 사항을 확인해 보세요:
- 인터넷 연결이 안정적인지 확인하세요
- Google AI Studio의 Gemini API키가 설정에서 올바르게 입력되었는지 확인하세요
- 다른 영상으로 시도해 보세요
- 프로그램을 재시작한 후 다시 시도해 보세요

### 영상 길이가 매우 짧거나 긴 경우 어떤 모드를 선택해야 하나요?
- 30초 미만의 매우 짧은 영상: 쇼츠 모드
- 1~3분 영상: 쇼츠 모드 또는 롱폼 모드 모두 가능(세로는 쇼츠, 가로는 롱폼 모드 추천)
- 3분 이상의 영상: 롱폼 모드
- 10분 이상의 매우 긴 영상: 롱폼 모드 (처리 시간이 길어질 수 있음)

### 생성된 타임라인을 저장할 수 있나요?
다음 방법으로 타임라인 정보를 저장할 수 있습니다:
- "모든 이미지 저장" 버튼으로 모든 장면 이미지를 저장
- "모든 설명 복사" 버튼으로 모든 텍스트 정보를 복사하여 메모장 등에 붙여넣기

### 타임라인 생성 중 "API 키 오류" 메시지가 표시됩니다
Google AI Studio API 키 관련 문제입니다:
- 설정 메뉴에서 API 키가 올바르게 입력되었는지 확인하세요
- API 키가 활성 상태인지 확인하세요
- API 할당량이 초과되었을 수 있습니다. 
24시간 후에 다시 시도하거나 다른 API 키를 사용해 보세요

### 타임라인 이미지가 표시되지 않아요
다음 사항을 확인해 보세요:
- 인터넷 연결이 안정적인지 확인하세요
- 타림라인 생성 재시도를 해보세요.
- 영상이 비공개이거나 제한된 영상일 경우 이미지가 제대로 표시되지 않을 수 있습니다

# 튜브렌즈 채널 검색 기능 가이드
## 개요
튜브렌즈의 채널 검색 기능은 유튜브 채널을 분석하고 해당 채널의 영상을 효율적으로 수집할 수 있는 강력한 도구입니다. 이 문서는 새롭게 추가된 채널 검색 기능의 사용법과 활용 방법에 대한 상세 정보를 제공합니다.

## 채널 검색 기능 소개
채널 검색 기능을 통해 다음과 같은 작업을 수행할 수 있습니다:
1. **채널 이름으로 검색**: 특정 채널을 빠르게 찾아볼 수 있습니다.
2. **채널 분석**: 구독자 수, 총 영상 수, 평균 조회수 등 채널 통계를 확인할 수 있습니다.
3. **채널 영상 수집**: 선택한 채널의 영상을 한 번에 수집할 수 있습니다.

## 채널 검색 사용법
### 검색 모드 전환
1. 화면 왼쪽 검색 설정 패널에서 "영상 검색 🎬" 버튼을 클릭합니다.
2. 버튼이 "채널 검색 🌟"로 변경되며 모드가 전환됩니다.
3. 검색창의 테두리 색상이 파란색에서 빨간색으로 변경되어 채널 검색 모드임을 시각적으로 표시합니다.

### 채널 검색 수행
1. 검색창에 찾고자 하는 채널 이름을 입력합니다.
2. "검색" 버튼을 클릭하거나 Enter 키를 누릅니다.
3. 검색 결과로 채널 목록이 표시됩니다.

### 채널 영상 수집
1. 채널 정보 대화상자에서 "채널 영상 수집" 버튼을 클릭합니다.
2. 수집된 영상은 메인 테이블에 표시되며, 일반 검색 결과와 동일하게 필터링 및 분석이 가능합니다.

### 정확한 일치 표시
1. 검색어와 정확히 일치하는 채널은 "정확한 일치" 배지로 표시됩니다.
2. 이를 통해 동명이거나 유사한 이름의 여러 채널 중 원하는 채널을 빠르게 찾을 수 있습니다.

### 채널 영상 특성 분석
1. 채널 영상을 수집한 후, 메인 테이블에서 다양한 필터링을 적용할 수 있습니다.
2. "쇼츠" 또는 "롱폼" 필터를 통해 채널의 영상 포맷 전략을 분석할 수 있습니다.
3. CII(콘텐츠 영향력 지수) 필터를 통해 성공적인 영상 패턴을 파악할 수 있습니다.

## 채널 검색 활용 팁
### 경쟁 채널 분석
1. 경쟁 채널을 검색하여 핵심 지표를 분석합니다.
2. 인기 영상의 특성을 파악하여 자신의 콘텐츠 전략에 반영합니다.
3. 기여도와 성과도를 비교하여 경쟁력 있는 콘텐츠 방향을 설정합니다.

### 협업 채널 발굴
1. 특정 키워드나 니치 분야의 채널을 검색합니다.
2. 구독자 수와 참여율을 확인하여 협업 가능성이 높은 채널을 식별합니다.
3. 채널의 인기 영상 스타일을 참고하여 협업 콘텐츠 아이디어를 개발합니다.

### 콘텐츠 트렌드 분석
1. 인기 채널의 최근 영상을 수집합니다.
2. 조회수나 좋아요 수 기준으로 정렬하여 트렌드를 파악합니다.
3. 영상 길이, 업로드 시간 등의 패턴을 분석하여 최적의 콘텐츠 전략을 수립합니다.

### 시간 경과에 따른 채널 성장 분석
1. 동일 채널을 주기적으로 검색하여 데이터를 수집합니다.
2. Excel로 저장 기능을 활용하여 시계열 데이터를 확보합니다.
3. 채널의 성장 패턴과 성공 요인을 분석합니다.

## 자주 묻는 질문 (FAQ)
### 채널 검색 관련 질문
#### Q: 채널 검색과 영상 검색의 차이점은 무엇인가요?
A: 영상 검색은 키워드에 관련된
모든 영상을 검색하는 반면, 채널 검색은 채널 이름으로 특정 채널을 찾아 해당 채널의 영상만 수집합니다.

#### Q: 최대 몇 개의 영상을 수집할 수 있나요?
A: 채널 검색을 통해 설정된 영상 수집 수(50개, 100개, 200개, 500개)만큼 영상을 수집할 수 있습니다.

#### Q: 채널 통계는 어떻게 계산되나요?
A: 채널 통계는 YouTube API를 통해 실시간으로 가져온 데이터를 기반으로 계산됩니다. 총 조회수, 평균 조회수, 평균 좋아요 수 등의 지표가 포함됩니다.

#### Q: 다른 사람의 채널을 검색해도 되나요?
A: 네, 공개된 모든 YouTube 채널을 검색할 수 있습니다. 이는 YouTube의 공개 API를 통해 제공되는 데이터만을 활용합니다.

### 기술적 질문
Q: 채널 검색 시 API 할당량은 어떻게 소비되나요?
A: 채널 검색은 영상 수에 따라 효율성이 달라집니다. 
50개 이하의 영상을 가져올 때는 일반 검색보다 약간 더 많은 할당량(117유닛 vs 100유닛)이 소모됩니다. 
하지만 100개 이상의 영상을 수집할 때는 오히려 더 효율적입니다(127유닛 vs 200유닛). 
채널 검색은 처음에 채널을 찾는 비용(100유닛)이 들지만, 
그 후 영상을 가져오는 비용이 매우 저렴하기 때문에 대량의 영상을 수집할수록 할당량 절약 효과가 커집니다.

#### Q: 채널 영상 수집이 중단되었을 때 어떻게 해야 하나요?
A: API 할당량 초과 등의 이유로 수집이 중단된 경우, 다른 API 키를 사용해 보세요. 
또는 시간이 지난 후 다시 시도할 수 있습니다.

## 문제 해결
### 채널 검색 문제
#### "채널을 찾을 수 없습니다" 오류
- 채널 이름을 정확히 입력했는지 확인하세요.
- 다른 검색어로 시도해 보세요.
- 인터넷 연결을 확인하세요.

#### "채널 영상을 가져올 수 없습니다" 오류
- API 할당량이 초과되었을 수 있습니다. 다른 API 키를 시도하세요.
- 채널의 영상이 비공개로 설정되어 있을 수 있습니다.
- 인터넷 연결을 확인하세요.

#### 느린 채널 영상 수집
- 채널에 영상이 많을 경우 수집 시간이 오래 걸릴 수 있습니다.
- 인터넷 연결 속도를 확인하세요.
- 수집 영상 수를 줄여보세요.

## 채널 검색과 다른 기능의 연동
### AI 추천 아이디어와 연동
1. 채널 검색으로 특정 채널의 영상을 수집합니다.
2. 인기 영상을 선택한 후 "🤖 AI 추천 아이디어" 기능을 활용하면 해당 영상 스타일을 참고한 아이디어를 얻을 수 있습니다.

### 타이틀 메이커와 연동
1. 채널 검색으로 인기 영상을 파악합니다.
2. "✏️ 타이틀 메이커" 기능을 활용하여 인기 영상의 제목 스타일을 분석하고 새로운 타이틀을 생성할 수 있습니다.

### 대본 생성기와 연동
1. 채널 검색으로 특정 채널의 영상을 수집합니다.
2. "📝 분석 및 대본생성" 기능을 활용하여 해당 채널 스타일의 대본을 생성할 수 있습니다.

### 자막 수집과 연동
1. 채널 검색으로 영상을 수집합니다.
2. "자막 수집" 기능을 통해 해당 채널 영상의 자막을 수집하여 콘텐츠 분석에 활용할 수 있습니다.

## 개인정보 및 데이터 보안
- 튜브렌즈의 채널 검색 기능은 YouTube의 공개 API를 통해 제공되는 공개 정보만을 활용합니다.
- 검색된 채널 정보는 사용자의 컴퓨터에만 저장되며, 외부 서버로 전송되지 않습니다.
- 채널 분석 결과는 개인적인 연구 및 콘텐츠 전략 수립 목적으로만 사용하시기 바랍니다.

# 튜브렌즈 채널 즐겨찾기 기능 가이드

## 개요
튜브렌즈의 채널 즐겨찾기 기능은 관심 있는 유튜브 채널들을 체계적으로 관리하고, 필요할 때마다 해당 채널들의 최신 영상을 빠르게 수집할 수 있는 강력한 도구입니다. 이 기능을 통해 정기적으로 모니터링하고 싶은 채널들을 그룹별로 정리하고, 효율적으로 콘텐츠 분석을 수행할 수 있습니다.

## 채널 즐겨찾기 기능 소개

채널 즐겨찾기 기능을 통해 다음과 같은 작업을 수행할 수 있습니다:

1. **채널 그룹 관리**: 채널을 용도별로 그룹화하여 체계적으로 관리할 수 있습니다.
2. **즐겨찾기 채널 등록**: 관심 채널을 검색하여 즐겨찾기에 추가할 수 있습니다.
3. **일괄 영상 수집**: 선택한 채널들의 최신 영상을 한 번에 수집할 수 있습니다.
4. **URL 기반 추가**: YouTube 링크를 통해 채널을 자동으로 추가할 수 있습니다.
5. **그룹별 필터링**: 특정 그룹의 채널만 선별하여 관리할 수 있습니다.

## 채널 즐겨찾기 접근 방법

### 즐겨찾기 다이얼로그 열기
1. 메인 화면 상단의 "📥 보관함" 버튼을 클릭합니다.
2. 다이얼로그가 열리면 "⭐ 채널 즐겨찾기" 탭을 선택합니다.

### 화면 구성 이해
- **상단**: 그룹 선택 및 관리 버튼들
- **왼쪽 영역**: 즐겨찾기된 채널 목록과 관리 도구
- **오른쪽 영역**: 새 채널 검색 및 추가 기능

## 그룹 관리 기능

### 기본 그룹 구조
- **그룹 미지정**: 삭제할 수 없는 기본 그룹
- **그룹1~그룹5**: 기본 제공 그룹들
- **사용자 정의 그룹**: 필요에 따라 추가 생성 가능

### 새 그룹 추가
1. "➕" 버튼을 클릭합니다.
2. 그룹 이름을 입력하고 "추가" 버튼을 클릭합니다.
3. 새 그룹이 생성되어 콤보박스에 추가됩니다.

### 그룹 이름 변경
1. "✏️" 버튼을 클릭합니다.
2. 새로운 그룹 이름을 입력합니다.
3. 변경사항이 즉시 적용됩니다.

### 그룹 삭제
1. "🗑️" 버튼을 클릭합니다.
2. 확인 메시지에서 "예"를 선택합니다.
3. ⚠️ **주의**: 그룹 삭제 시 그룹 내 모든 채널도 함께 삭제됩니다.

## 채널 추가 방법

### 1. 채널 검색을 통한 추가
1. 오른쪽 영역의 검색창에 채널명을 입력합니다.
2. "검색" 버튼을 클릭하거나 Enter 키를 누릅니다.
3. 검색 결과에서 원하는 채널을 선택합니다.
4. "➕ 선택한 채널 즐겨찾기에 추가" 버튼을 클릭합니다.

### 2. YouTube 링크를 통한 추가
검색창에 다음과 같은 YouTube 링크를 입력할 수 있습니다:

#### 지원하는 링크 형식
- **채널 링크**: `https://www.youtube.com/channel/UC...`
- **사용자 링크**: `https://www.youtube.com/user/username`
- **커스텀 링크**: `https://www.youtube.com/c/channelname`
- **핸들 링크**: `https://www.youtube.com/@channelhandle`
- **영상 링크**: `https://www.youtube.com/watch?v=...` (영상의 채널이 추가됨)

#### 여러 링크 동시 추가
1. 검색창에 여러 YouTube 링크를 줄바꿈으로 구분하여 입력합니다.
2. "검색" 버튼을 클릭합니다.
3. 모든 링크가 자동으로 처리되어 채널이 추가됩니다.

## 채널 관리 기능

### 채널 선택 및 체크
- 각 채널 항목을 클릭하면 체크박스가 토글됩니다.
- "전체 선택/해제" 버튼으로 모든 채널을 한 번에 선택하거나 해제할 수 있습니다.
- 선택된 채널 수가 상단에 실시간으로 표시됩니다.

### 채널 검색 및 필터링
1. 왼쪽 영역 상단의 "🔍 내 채널 검색..." 입력창을 사용합니다.
2. 채널명을 입력하면 즉시 필터링되어 표시됩니다.
3. 검색은 모든 그룹의 채널을 대상으로 수행됩니다.

### 채널 그룹 이동
#### 우클릭 메뉴를 통한 이동
1. 이동하려는 채널에서 우클릭합니다.
2. "📁 이 채널 이동" 또는 "📁 선택된 N개 채널 이동" 메뉴를 선택합니다.
3. 원하는 대상 그룹을 선택합니다.

#### 여러 채널 동시 이동
1. 이동하려는 채널들을 체크합니다.
2. 아무 채널에서 우클릭하여 일괄 이동 메뉴를 선택합니다.
3. 대상 그룹을 선택하면 모든 선택된 채널이 이동됩니다.

### 채널 삭제
1. 삭제하려는 채널을 선택합니다.
2. "선택 삭제" 버튼을 클릭하거나 우클릭 메뉴에서 "🗑️ 즐겨찾기에서 삭제"를 선택합니다.
3. 확인 메시지에서 "예"를 선택합니다.

## 채널 영상 수집 기능

### 영상 수집 설정
1. 왼쪽 영역 하단의 "각 채널당 가져올 영상 수" 입력창에서 수집할 영상 개수를 설정합니다.
2. 기본값은 50개이며, 1~500개 범위에서 설정 가능합니다.

### 선택 채널 영상 수집
1. 영상을 수집하려는 채널들을 체크합니다.
2. "불러오기" 버튼을 클릭합니다.
3. 각 채널에서 설정된 개수만큼의 최신 영상이 수집됩니다.
4. 수집된 영상은 메인 테이블에 표시되며, 일반 검색 결과와 동일하게 분석 가능합니다.

### 영상 수집 프로세스
1. **진행 상황 표시**: 수집 중 진행률이 실시간으로 표시됩니다.
2. **자동 다이얼로그 닫기**: 수집이 시작되면 즐겨찾기 다이얼로그가 자동으로 닫힙니다.
3. **메인 테이블 업데이트**: 수집된 영상이 메인 화면에 즉시 반영됩니다.

## 고급 활용 팁

### 채널 모니터링 전략
1. **경쟁사 분석 그룹**: 경쟁 채널들을 별도 그룹으로 관리
2. **벤치마킹 그룹**: 참고하고 싶은 성공 채널들을 그룹화
3. **협업 후보 그룹**: 잠재적 협업 파트너 채널들을 관리
4. **트렌드 모니터링 그룹**: 최신 트렌드를 반영하는 채널들을 추적

### 정기적 분석 워크플로우
1. **주간 모니터링**: 관심 채널들의 최신 영상을 정기적으로 수집
2. **성과 분석**: 수집된 데이터를 통해 트렌드와 성과 패턴 파악
3. **전략 수정**: 분석 결과를 바탕으로 콘텐츠 전략 조정

### 효율적 채널 관리
1. **그룹명 체계화**: 목적별로 명확한 그룹명 사용 (예: "게임_경쟁사", "요리_벤치마킹")
2. **정기적 정리**: 더 이상 관련성이 없는 채널은 주기적으로 제거
3. **중복 방지**: 같은 채널이 여러 그룹에 추가되지 않도록 주의

## 문제 해결
### 채널 추가 관련 문제
#### "채널을 찾을 수 없습니다" 오류
- YouTube 링크가 올바른지 확인하세요.
- 채널이 비공개로 설정되어 있을 수 있습니다.
- 인터넷 연결 상태를 확인하세요.

#### "이미 존재하는 채널" 메시지
- 해당 채널이 이미 다른 그룹에 추가되어 있습니다.
- 채널 이동 기능을 사용하여 원하는 그룹으로 이동시키세요.

#### 여러 링크 추가 시 일부 실패
- API 할당량이 부족할 수 있습니다.
- 잘못된 링크가 포함되어 있는지 확인하세요.
- 진행 상황 다이얼로그에서 실패한 링크를 확인할 수 있습니다.

### 영상 수집 관련 문제
#### "API 할당량 초과" 오류
- 다른 API 키를 추가하거나 시간이 지난 후 다시 시도하세요.
- 영상 수집 개수를 줄여서 시도해 보세요.

#### 영상 수집이 중단됨
- 인터넷 연결을 확인하세요.
- 채널에 영상이 부족할 수 있습니다.
- 채널이 비공개로 설정되었을 수 있습니다.

#### 느린 영상 수집 속도
- 채널 수와 각 채널당 영상 수를 줄여보세요.
- 인터넷 연결 속도를 확인하세요.
- 백그라운드에서 다른 네트워크 작업이 진행 중인지 확인하세요.

### 그룹 관리 관련 문제
#### 그룹 삭제가 안됨
- "그룹 미지정"은 삭제할 수 없는 보호된 그룹입니다.
- 다른 그룹을 선택한 후 시도하세요.

#### 그룹명 변경 실패
- 이미 존재하는 그룹명은 사용할 수 없습니다.
- "전체"는 시스템 예약어로 사용할 수 없습니다.

## 개인정보 및 데이터 보안

- 모든 즐겨찾기 데이터는 사용자의 로컬 컴퓨터에만 저장됩니다.
- YouTube의 공개 API만을 사용하여 공개된 정보만 수집합니다.
- 채널 분석 결과는 개인적 연구 및 콘텐츠 전략 수립 목적으로만 사용하시기 바랍니다.
- 타인의 채널 정보를 상업적 목적으로 무단 사용하지 마시기 바랍니다.

## 화면 최적화 설정 (작은 모니터 사용자 필독)

### 프로그램 창 크기 조절 방법

노트북이나 작은 모니터를 사용하시는 경우, 튜브렌즈 프로그램 창이 화면에 비해 너무 클 수 있습니다. 이때 다음 방법으로 화면을 최적화할 수 있습니다:

#### Windows 디스플레이 설정 조정
1. **바탕화면에서 마우스 우클릭**을 합니다.
2. **"디스플레이 설정"**을 선택합니다.
3. **"배율 및 레이아웃"** 섹션을 찾습니다.
4. **배율을 현재보다 낮게 설정**합니다. (예시 : 현재값이 200%면 175%)
5. **디스플레이 해상도를 현재보다 높게 설정**합니다.
6. 설정 적용 후 튜브렌즈를 다시 실행합니다.

#### 설정 효과
- 프로그램 창과 텍스트가 더 작아져서 더 많은 정보를 화면에서 볼 수 있습니다.
- 채널 즐겨찾기 다이얼로그가 화면에 잘 맞도록 조정됩니다.
- 전체적인 인터페이스가 더 컴팩트하게 표시됩니다.

#### 주의사항
- 텍스트와 버튼이 작아질 수 있으므로, 시각적 불편함이 있다면 적절한 균형점을 찾아 조정하시기 바랍니다.
- 이 설정은 다른 프로그램들에도 영향을 줄 수 있습니다.

### 창이 너무 클 때 닫기 방법

창이 화면을 벗어나서 우측 상단의 ❌ 닫기 버튼이 보이지 않을 때:

#### 키보드로 닫기
- **Alt + F4** 키를 동시에 누르면 즉시 프로그램이 종료됩니다.

#### 작업 표시줄에서 닫기
- 화면 하단 작업 표시줄의 튜브렌즈 아이콘에서 우클릭 → "창 닫기" 선택

#### 튜브렌즈 가격
- 튜브렌즈의 월간 구독제 (정기결제)
: 월 29,900원

- 365일 1년 이용권
: 현재 특별할인 31% 적용으로, 249,900원에 단일권 판매중

#### 인증키 찾는 방법
- 홈페이지 상단의 '내 인증키 찾기'에서 유료/무료 인증키 찾기가 가능합니다.

#### 구독해지 방법
- 홈페이지 제일 하단에 '구독해지' 버튼이 있습니다.
휴대폰 번호와 이메일 주소를 입력하여 구독정보를 찾고, 직접 구독해지를 할 수 있습니다.
구독 해지 후, 마지막 결제일로 부터 30일간 튜브렌즈를 사용하실 수 있습니다.

#### 인증키 교체
- 컴퓨터 업데이트 / 포맷 / 새 컴퓨터로 변경 등의 사유가 생길 시,
메일로 인증키 교체를 문의주시면,

## 게시일 기간선택 필터링 기능

### 기능 개요
테이블에 표시된 검색 결과에서 특정 날짜 범위의 영상만 필터링해서 볼 수 있는 기능입니다. 
게시일 열의 정렬 메뉴에 새로 추가된 "기간선택" 옵션을 통해 사용할 수 있습니다.

### 기능 접근 방법
1. 테이블 헤더의 "게시일" 열에서 마우스 우클릭합니다.
2. 나타나는 메뉴에서 "기간선택"을 클릭합니다.
3. 기간선택 다이얼로그가 열립니다.

### 사용 방법
#### 날짜 선택하기
1. 시작일 입력: 직접 입력 (YYYY-MM-DD 형식) 또는 📅 버튼 클릭하여 캘린더에서 선택
2. 종료일 입력: 직접 입력 (YYYY-MM-DD 형식) 또는 📅 버튼 클릭하여 캘린더에서 선택
3. "적용" 버튼 클릭하여 필터 적용
4. "닫기" 버튼으로 취소 및 다이얼로그 종료

#### 캘린더 사용법
- 📅 버튼을 클릭하면 캘린더가 팝업으로 표시됩니다.
- 원하는 날짜를 클릭하면 자동으로 입력창에 날짜가 입력됩니다.

### 필터링 범위 및 특징
#### 날짜 범위 처리
- 시작일과 종료일 모두 포함: 2024-08-16부터 2024-08-16까지 선택하면 8월 16일의 모든 영상이 표시됩니다.
- 날짜 형식 자동 인식: YYYY-MM-DD 형식으로 입력해야 하며, 잘못된 형식 입력 시 경고 메시지가 표시됩니다.

### 활용 사례
#### 트렌드 분석
- 특정 기간 성과 분석: "2024-07-01"부터 "2024-07-31"까지 선택하여 7월 한 달간의 영상 성과만 분석
- 이벤트 기간 분석: 특별한 이벤트나 시즌 기간의 영상들만 필터링하여 패턴 파악

### 문제 해결
#### "날짜 형식이 올바르지 않습니다" 오류
- 날짜를 YYYY-MM-DD 형식으로 입력했는지 확인하세요.
- 존재하지 않는 날짜 (예: 2024-02-30)를 입력하지 않았는지 확인하세요.

#### "시작일이 종료일보다 늦을 수 없습니다" 경고
- 시작일과 종료일을 올바른 순서로 입력했는지 확인하세요.
- 같은 날짜를 보려면 시작일과 종료일을 동일하게 설정하세요.

## 결과내 재검색 기능

### 기능 개요
이미 수집된 검색 결과에서 추가적인 키워드 검색을 통해 원하는 영상을 빠르게 찾을 수 있는 기능입니다. 
네이버 메일의 "결과내 재검색"과 같은 개념으로, 전체 데이터는 그대로 유지하면서 테이블 표시만 필터링합니다.

### 기능 위치 및 접근
- 위치: 화면 하단 버튼 영역의 중앙
- 표시: 입력창에 "🔍 결과내 재검색..." 플레이스홀더 텍스트가 희미하게 표시됩니다.

### 사용 방법
#### 기본 검색
1. 입력창에 검색어 입력: 원하는 키워드나 숫자를 입력합니다.
2. 실시간 필터링: 타이핑하는 즉시 결과가 실시간으로 필터링됩니다.
3. Enter 키 불필요: 별도의 검색 버튼이나 Enter 키를 누를 필요가 없습니다.

#### 검색 초기화
- 입력창 지우기: 입력창을 비우면 모든 검색 결과가 다시 표시됩니다.
- ESC 키 사용: 입력창에 커서가 있을 때 ESC 키를 누르면 입력된 텍스트가 모두 삭제됩니다.

### 검색 범위 및 대상
결과내 재검색은 7개 항목에서 동시에 검색됩니다:

#### 1. 영상 제목
- 검색 예시: "요리" 입력 → 제목에 "요리"가 포함된 영상들만 표시
- 대소문자 무관: 영어 대소문자를 구분하지 않습니다.

#### 2. 채널명
- 검색 예시: "백종원" 입력 → 채널명에 "백종원"이 포함된 영상들만 표시
- 부분 검색 가능: "백종" 입력해도 "백종원" 채널을 찾을 수 있습니다.

#### 3. 영상 설명
- 검색 예시: "레시피" 입력 → 영상 설명에 "레시피"가 포함된 영상들만 표시
- 긴 텍스트에서 검색: 영상 설명의 전체 내용에서 검색됩니다.

#### 4. 게시일
- 검색 예시: 
  - "2024-08" 입력 → 2024년 8월에 게시된 영상들만 표시
  - "2024" 입력 → 2024년에 게시된 모든 영상들 표시
- 날짜 형식 유연: 부분적인 날짜 입력도 가능합니다.

#### 5. 조회수
#### 6. 구독자수
#### 7. 자막 내용
- 전제조건: 자막이 미리 수집된 영상에서만 검색 가능합니다.
- 검색 예시: "안녕하세요" 입력 → 자막에 "안녕하세요"가 포함된 영상들만 표시
- 자막 수집 상태: "자막수집" 또는 "자막없음/재시도" 상태인 영상은 자막 검색 대상에서 제외됩니다.

### 주의사항 및 제한사항
#### 검색 대상 제한
- 수집된 데이터만: 현재 테이블에 표시된 검색 결과에서만 재검색이 가능합니다.
- 자막 검색 조건: 자막 검색은 미리 자막이 수집된 영상에서만 가능합니다.

### 문제 해결
#### 검색이 작동하지 않음
- 데이터 확인: 테이블에 검색 결과가 로드되어 있는지 확인하세요.
- 검색어 확인: 검색어가 올바르게 입력되었는지, 오타가 없는지 확인하세요.

#### 자막 검색이 안됨
- 자막 수집 상태 확인: 해당 영상의 자막이 "자막수집" 버튼으로 미리 수집되어 있는지 확인하세요.
- 자막 내용 확인: 자막이 수집되었더라도 실제 내용이 있는지 자막 보기로 확인하세요.

## 외국어 자막 기능
- 한국어 이외에, 영어, 일본어, 대만, 독일어, 스페인어 등
각 국가의 영상 기본자막을 인식하여 자막수집을 할 수 있습니다.
- 자막이 없거나, 인식불가한 경우 수집이 안될 수 있습니다.

# 튜브렌즈 유튜브 멤버십 서비스 가이드

## 개요
튜브렌즈 유튜브 멤버십은 기존 튜브렌즈 프로그램과 별개로 운영되는 프리미엄 서비스입니다.

# 튜브레터(TubeLetter) 상세 가이드

## 개요
튜브레터(TubeLetter)는 유튜브 크리에이터를 위한 자동 트렌드 분석 메일 서비스입니다.
매일 어제 떡상한 영상들을 AI 분석과 함께 이메일로 받아볼 수 있습니다.
`튜브레터`는 튜브렌즈 유료 사용자에게만 제공되는 프리미엄 서비스입니다.

## 주요 기능

### 1. 자동 떡상 영상 모니터링
매일 원하는 시간에 어제의 인기 영상을 자동으로 수집하여 메일로 제공합니다:
- **등록 채널 떡상 영상**: 본인이 등록한 채널들의 '어제'의 인기 영상을 확인할 수 있습니다.
- **유사 채널 발견**: 등록한 채널과 결이 비슷한 경쟁 채널의 떡상 영상도 자동으로 발견합니다.

### 2. AI 기반 영상 분석
수집된 떡상 영상에 대한 심층 분석을 제공합니다:
- **제목 패턴 분석**: 인기 영상들의 제목 패턴을 분석하여 트렌드를 파악합니다.
- **조회수 비결**: 높은 조회수를 기록한 이유를 분석합니다.
- **차별화 요소**: 다른 영상과 차별화된 성공 요인을 분석합니다.

### 3. 추천 제목 생성
AI가 분석 결과를 바탕으로 실제 사용 가능한 제목을 추천합니다:
- **맞춤형 제목**: 분석한 트렌드를 반영한 제목을 3~6개 제안합니다.
- **호기심 유발**: 클릭을 유도하는 효과적인 제목 구조를 제공합니다.
- **키워드 최적화**: 검색에 유리한 키워드가 포함된 제목을 추천합니다.

### 4. 키워드 검색 인기 영상 탑5
고급 키워드 분석을 통해 더 넓은 범위의 인기 영상을 발견합니다:
- **자동 키워드 추출**: 등록 채널 영상 1000개(채널당 50개 × 20채널) 중 조회수 높은 Top20을 분석하여 메인/서브/롱테일 키워드를 추출합니다.
- **경쟁 채널 발견**: 추출된 3가지 키워드로 검색하여 내가 몰랐던 경쟁 채널의 떡상 영상을 발견합니다.
- **조회수 기반 정렬**: 최종 상위 5개 인기 영상을 조회수 순으로 제공합니다.

### 5. 영상 정보 제공
각 떡상 영상에 대한 상세 정보를 확인할 수 있습니다:
- **영상 제목**: 떡상한 영상의 제목을 확인할 수 있습니다.
- **채널명**: 영상을 게시한 채널을 확인할 수 있습니다.
- **조회수**: 영상의 조회수를 확인할 수 있습니다.
- **썸네일**: 클릭 가능한 썸네일을 통해 해당 영상으로 바로 이동할 수 있습니다.

### 6. 맞춤 시간 설정
원하는 시간에 메일을 받을 수 있습니다:
- **시간 선택**: 오전 1시부터 밤 11시까지 1시간 단위로 선택 가능합니다.
- **자동 발송**: 선택한 시간 전후로 분석 완료 후 자동 발송됩니다.

## 사용 방법

### 초기 설정하기

1. **튜브레터 페이지 접속**
   - 튜브렌즈 홈페이지(https://tubelens.kr/)에 접속합니다.
   - 상단 메뉴에서 '튜브레터'를 클릭합니다.

2. **API 키 등록**
   - **YouTube API 키**: 유튜브 영상 수집을 위한 API 키를 입력합니다.
     - '유튜브 키 받으러 가기' 버튼을 클릭하여 키를 발급받을 수 있습니다.
   - **Gemini API 키**: AI 분석을 위한 Gemini API 키를 입력합니다.
     - ⚠️ 주의: YouTube API 키와 Gemini API 키를 혼동하지 마세요!
     - 키가 없거나 기억나지 않는 경우 'API 키 받는 법 영상'을 참고하세요. 
     - API 키 받는 법 영상 링크 : https://youtu.be/ivjPaNpGaoc?si=1X49qMMxPA9Ca8i9


3. **메일 수신 시간 설정**
   - 오전 1시~밤 11시 중 원하는 시간을 선택합니다.
   - 선택한 시간 전후로 메일이 발송됩니다(분석 시간 소요).

4. **이메일 주소 인증**
   - 튜브렌즈 결제 시 등록한 이메일 주소를 입력합니다.
   - '인증' 버튼을 클릭하여 이메일 인증을 완료합니다.
   - 인증이 완료되어야 알람 등록이 가능합니다.

### 채널 등록하기

1. **채널 추가 방법 1: 검색으로 추가**
   - 채널 검색창에 키워드를 입력합니다.
   - 검색 결과에서 원하는 채널을 선택합니다.
   - 여러 채널을 한 번에 선택할 수 있습니다.
   - '채널 등록' 버튼을 클릭하여 추가합니다.

2. **채널 추가 방법 2: 링크로 추가**
   - 유튜브 영상 링크 또는 채널 링크를 입력창에 붙여넣습니다.
   - 여러 링크를 한 번에 입력할 수 있습니다.
   - '채널 추가' 버튼을 클릭하거나 Enter 키를 누릅니다.

3. **채널 등록 시 주의사항**
   - ✅ 최대 20개까지 채널을 등록할 수 있습니다.
   - ✅ 반드시 결이 비슷한 채널만 등록하세요.
   - ❌ 쇼핑, 국뽕, 시니어 등 서로 다른 주제를 섞으면 분석이 부정확해집니다.
   - ❌ 잘못 추가한 채널은 삭제 버튼으로 제거할 수 있습니다.

### 서비스 등록 완료하기

1. **약관 동의**
   - 서비스 이용 약관 및 개인정보 처리방침을 확인합니다.
   - 동의 체크박스를 선택합니다.

2. **알람 등록**
   - '알람 등록' 버튼을 클릭합니다.
   - 등록 완료 메시지가 나타나면 설정이 완료된 것입니다.

### 메일 확인하기

1. **메일 수신**
   - 설정한 시간에 등록한 이메일로 튜브레터가 발송됩니다.

2. **메일 내용**
   - **등록 채널 어제의 인기 영상 탑5**
   - **AI 분석**: 제목 패턴, 조회수 비결, 차별화 요소
   - **추천 제목**: 바로 사용 가능한 제목 3~6개
   - **키워드 검색 인기 영상 탑5**: 유사 채널의 떡상 영상


3. **영상 확인**
   - 썸네일을 클릭하면 해당 유튜브 영상으로 바로 이동합니다.

### 정보 수정하기

1. **등록 정보 변경**
   - 튜브레터 페이지에 다시 접속합니다.
   - 변경할 정보를 새로 입력합니다.
   - '알람 등록' 버튼을 다시 클릭합니다.
   - 기존 정보가 자동으로 삭제되고 새 정보로 업데이트됩니다.

### 서비스 취소하기

1. **알람 취소**
   - 튜브레터 페이지에서 '알람 취소' 버튼을 클릭합니다.
   - 팝업창에 등록한 이메일 주소를 입력합니다.
   - 확인 버튼을 누르면 모든 정보가 구글 시트에서 삭제됩니다.

## 자주 묻는 질문(FAQ)

### Q: 튜브레터는 누가 사용할 수 있나요?
A: 튜브렌즈 유료 사용자만 사용할 수 있는 프리미엄 서비스입니다.
https://tubelens.kr/

### Q: YouTube API 키와 Gemini API 키는 어디서 받나요?
A: 
- YouTube API 키: 튜브레터 페이지의 '유튜브 키 받으러 가기' 버튼 또는 'API 키 받는 법 영상'을 참고하세요.
- Gemini API 키: 튜브레터 페이지의 'Gemini 키 받으러 가기' 버튼 또는 'API 키 받는 법 영상'을 참고하세요.

### Q: 채널을 몇 개까지 등록할 수 있나요?
A: 최대 20개까지 등록 가능합니다.

### Q: 서로 다른 주제의 채널을 섞어서 등록해도 되나요?
A: 아니요. 반드시 결이 비슷한 채널만 등록해야 정확한 분석 결과를 받을 수 있습니다. 예를 들어 쇼핑, 국뽕, 시니어 채널을 섞으면 분석이 부정확해지고 엉뚱한 영상이 추천됩니다.

### Q: 메일이 정확히 설정한 시간에 오나요?
A: 선택한 시간 전후로 발송됩니다. 분석 작업에 시간이 소요되므로 약간의 시차가 있을 수 있습니다.

### Q: 등록한 정보를 수정하고 싶어요.
A: 튜브레터 페이지에서 새로운 정보를 입력하고 '알람 등록' 버튼을 다시 클릭하면 자동으로 업데이트됩니다.

### Q: 서비스를 중단하고 싶어요.
A: '알람 취소' 버튼을 클릭하고 이메일 주소를 입력하면 모든 정보가 삭제되고 메일 발송이 중단됩니다.

### Q: 503 에러 또는 500 에러 메시지가 나타납니다.
A: 503 에러는 "모델이 과부하 상태(The model is overloaded)"를 의미하며, 500 에러는 "내부 서버 오류(Internal Server Error)"를 의미합니다. 두 에러 모두 Google Gemini API 서버 측의 일시적인 문제로 튜브레터 서비스와는 무관합니다. 대부분 자동으로 재시도되며, 다음 레터부터는 정상적으로 발송됩니다.

### Q: 쇼츠와 롱폼 중 어느 것이 더 유용한가요?
A: 쇼츠도 유용하지만, 튜브레터는 제목을 기반으로 분석하기 때문에 롱폼 크리에이터에게 특히 유용합니다. 롱폼은 썸네일과 제목이 매우 중요하기 때문입니다.

### Q: 키워드 검색 인기 영상 탑5는 어떻게 선정되나요?
A: 등록한 채널들의 최근 영상 1000개(채널당 50개)를 수집하여, 일주일 이내의 조회수 상위 20개 영상의 제목을 분석합니다. 여기서 메인/서브/롱테일 키워드를 추출하여 검색하고, 조회수가 높은 순서대로 5개를 선정합니다.

### Q: 내가 등록하지 않은 채널의 영상도 나오나요?
A: 네! 키워드 분석을 통해 등록한 채널과 결이 비슷한 다른 채널들의 떡상 영상도 자동으로 발견하여 제공합니다. 이를 통해 내가 몰랐던 경쟁 채널을 발견할 수 있습니다.

### Q: 어제 올라온 영상만 분석하나요?
A: 네, 튜브레터는 어제(전날) 업로드된 영상 중 떡상한 영상들을 분석합니다. 가장 최신 트렌드를 파악할 수 있습니다.

## 문제 해결

### API 키 인증 실패
- YouTube API 키와 Gemini API 키를 올바른 위치에 입력했는지 확인하세요.
- 두 키를 혼동하여 입력하는 경우가 많습니다.
- API 키가 유효한지 확인하세요.
- 특히 유튜브 API키 발급시 '사용'을 눌렀는지 꼭 확인하세요. '사용'을 누르지 않고도 튜브레터에 등록은 되지만, 권한이 없어 사용이 불가합니다.

### 이메일 인증 실패
- 튜브렌즈 결제 시 사용한 이메일 주소가 맞는지 확인하세요.
- 이메일 주소에 오타가 없는지 확인하세요.
- 무료 사용자는 사용할 수 없습니다. 유료 사용자인지 확인하세요.

### 메일이 오지 않을 경우
- 스팸 메일함을 확인하세요.
- 이메일 주소가 정확하게 입력되었는지 확인하세요.
- 다른곳에서 해당 유튜브API를 많이 사용했다면, 유튜브 API 할당량이 충분하지 않아, 메일 발송이 안될수 있습니다.
- 알람이 제대로 등록되었는지 다시 확인하세요.

### 분석 결과가 이상할 경우
- 서로 다른 주제의 채널을 섞어 등록하지 않았는지 확인하세요.
- 결이 비슷한 채널들만 등록해야 정확한 분석이 가능합니다.

### YouTube API 할당량 사용량
- 하루 유튜브 api 할당량은 1만 units
- 10개 등록시, 약 326 units 사용
- 20개 채널 등록시, 최대 347 units 사용


### Gemini API 오류
- Gemini API 키가 유효한지 확인하세요.
- 필요 시 새 API 키를 발급받아 등록하세요.

# 튜브렌즈 100일 챌린지 가이드

## 개요
튜브렌즈 100일 챌린지는 튜브렌즈 사용자들이 꾸준한 콘텐츠 제작 습관을 만들고, 채널 성장을 이루기 위한 도전 이벤트입니다. 
100일 동안 매일 영상을 업로드하며, 튜브렌즈를 활용한 분석과 최적화를 통해 실질적인 채널 성장을 경험할 수 있습니다. 완주자에게는 현금 100만원 추첨과 15개월 50% 할인권 등 푸짐한 보상이 제공됩니다.
상세내용 영상 : https://youtu.be/KRE9Lh1Y6vA

## 챌린지 핵심 규칙

### 1. 기본 규칙
**업로드 조건**
- 100일 동안 매일 유튜브 영상 1개씩 업로드
- 쇼츠든 롱폼이든 영상 형식은 자유
- 본인이 편한 형식으로 제작 가능
- 주제도 자유

**완주 기준**
- 100일 중 80일만 채우면 완주 인정
- 한 달에 약 6~7일 정도는 휴식 가능
- 아프거나 바쁜 날은 부담없이 쉴 수 있음

**업로드 시간**
- 한국시간 기준 매일 00:00 ~ 23:59 사이 게시만 인정
- 예약 업로드 가능하지만, 실제 공개된 당일만 인정 ('게시됨'만 인정)
- 자정(밤 12시) 넘기면 다음날로 인정

### 2. 보상 체계

**현금 100만원 추첨 (3.3% 세금 제외)**
- 라이브 추첨으로 1명 선정
- 완주 일수에 따른 차등 추첨권 부여:
  - 80일 완주: 추첨권 1장
  - 90일 완주: 추첨권 3장
  - 100일 완주: 추첨권 5장
- 100일 완주자는 80일 완주자보다 5배 높은 당첨 확률

**15개월 50% 할인권**
- 80일 이상 완주자 전원 지급
- 20만원 이상의 할인 혜택
- 챌린지 참여 비용 이상의 혜택

**챌린지 전용 단톡방**

### 상품 및 이용 관련

**Q: 455일은 어떻게 계산되나요?**

A: 365일(1년) + 90일(3개월) = 455일, 약 15개월 이용 가능합니다.

**Q: 튜브렌즈 구매 후 바로 사용 안 하면 사용일이 그만큼 뒤로 밀리나요?**

A: 아니요. 인증키는 결제일로부터 즉시 생성되며 사용일이 시작됩니다. 
실제로 프로그램을 사용하지 않아도 455일 기간은 흘러갑니다.

**Q: 월 구독 중인데 블프 이벤트 상품을 구매하면 기간이 합산되나요?**

A: 아니요, 합산되지 않습니다.

1) 월 구독과 블랙프라이데이 단일권(455일)은 완전히 별개 상품입니다
2) 두 상품의 라이센스 시스템이 서로 달라서 기간 합산이 불가능합니다

즉, 각 인증키별 결제 시스템이 서로 연동되어 있지 않고, 인증키 생성도 결제 시점에 발동되기 때문에 기존 키 완료 후 이어서 사용하는 방법은 어렵습니다. (중복 기간 발생)

**Q: 이미 1년 단일권을 사용 중인데 블프 이벤트 상품을 구매하면 어떻게 되나요?**

A: 기존 인증키와 새 인증키는 별개입니다. 
기간 합산이나 연장이 되지 않으며, 두 인증키 중 하나를 선택해서 사용해야 합니다. 

**Q: 인증키를 여러 개 구매할 수 있나요?**

A: 네, 가능합니다. 하지만 각 인증키는 독립적으로 작동하며 기간이 합산되지 않습니다.

**Q: 블프 이벤트로 구매한 후 나중에 월 구독으로 전환할 수 있나요?**

A: 월 구독이나 다른 상품을 새로 구매하실 수 있습니다만, 이미 구매한 키를 중간에 전환하거나 변경은 불가능합니다.
